# コンテキスト剥離

ある情報・知識・手続きが、制度を通過する過程で「点」だけに切り詰められ、由来・目的・帰結・代替などのコンテキストが構造的に削ぎ落とされる動作。情報の事実部分は保存されるが、それが**なぜ・何のために・何を犠牲にして**現在の形になったのかが伝達経路から消える。

## 起源（Why this exists）

制度（学校・行政・メディア・テスト・カタログ・規格）は、不特定多数を同一水準で扱うことを設計上の前提とする。同一水準化は、伝達コストを最小化する圧力を生む。コンテキストは個体差・文脈差・解釈差を含むため、最初に削られる。

3つの構造的要因：

1. **時間制約**：単位時間あたりの伝達量を最大化するには、削ぎ落としが最も安価な圧縮手段。
2. **評価可能性の要請**：制度はアウトプットを測定可能でなければならない。「点」は採点可能、コンテキストは採点困難。
3. **代替可能性の確保**：教師・担当者・端末を交換可能にするには、伝達物が脱文脈化していなければならない。

結果として、制度から出力される知識は「形式は完全、由来は不在」という構造をとる。

## 観察可能なproxy

- 子が「読める」が「わからない」と言う現象（例：時計の長針短針は読めるが「午前/午後」が宙に浮く）
- ニュース見出しは記憶されるが背景の年表・力学が共有されない
- 単語テストの語彙は答えられるが、その語が言語史で何を置き換えてきたか語れない
- 行政掲示の手続きは実行できるが、その手続きが何を防ぐために生まれたか説明できない
- カタログのスペック比較はできるが、その製品カテゴリの歴史的選択肢が見えない
- SNS の「事実」断片が起源・反証条件を伴わずに流通する
- 数式・公式は適用できるが、誰がどんな問題に対してそれを発明したかが空白

## 介入点（Disable conditions）

- 「なぜそうなっているのか」を発する場の再設計（家庭・読書会・徒弟関係）
- 由来語りができる人を制度の外側に確保（親・祖父母・市井の専門家）
- 点とコンテキストの分業を明示化（「学校は点、家庭はコンテキスト」という役割の言語化）
- 反例・代替案の併記を伝達様式に組み込む（「12進ではなく10進だったら」を問う）
- 制度の出力を「未完成品」として扱う規範（受け取り側で補完が必要であることの常態化）

## 補足

context-stripping は「悪意の隠蔽」ではなく**構造の自己運動**である。誰かがコンテキストを隠したのではなく、制度がコンテキストを保持できない設計になっているために、結果として剥がれる。これが介入の難しさの源泉となる。教師・担当者の善意では解決できない。

並走的処方が要請される：制度の外側に「コンテキストを引き受ける層」を別途置く。学校教育に対する家庭教育、ニュースに対する古典、カタログに対する徒弟、検索結果に対する読書会。点を打つ層とコンテキストを引く層は、止揚されず**同じ時間を並走する**。

事例：
- **時計・午前午後の理解（2026, 日本）**：小学校2年生が時計の長針短針は読めるが「午前/午後/正午」が結びつかない。学校は時刻読み取りという「点」を伝達できているが、なぜ世界の時間が2分割されているか・なぜ12進かというコンテキストを伝達できない。`200_Output/substack-daily/2026-05-12-clock-and-parental-education.md` で扱う。
- **ニュース消費**：見出しと事実関係は流通するが、その問題が現在の形になった歴史的経路（規制の経緯、過去の類似事例、代替案の系譜）が伝達経路から脱落する。
- **検索ファースト学習**：個別の知識項目は即座にアクセスできるが、その知識が誰のどんな問いに対する回答だったかが分からない。
