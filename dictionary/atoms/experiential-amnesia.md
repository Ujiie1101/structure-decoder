# 経験的記憶の劣化

ある脅威・教訓が、それを経験した世代がまだ生きているにもかかわらず、集団的記憶として機能しなくなる現象。記憶の物理的消失ではなく、**記憶の質的劣化**（身体化された記憶 → 制度化された記憶 → 他人事化された記憶）が原因。

## 起源（Why this exists）

集団記憶には少なくとも3つの状態がある：

1. **身体化された記憶**：体験者本人の身体・感情に紐付く。語り口に重みがあり、聞き手の警戒を起動できる。
2. **制度化された記憶**：教科書・統計・記念碑として保存される。情報は残るが、現在の感覚に届かない。
3. **他人事化された記憶**：「昔そういうことがあったらしい」として伝達される。情報の存在は確認できるが、行動を起こす根拠にならない。

[generational-amnesia](generational-amnesia.md) は1の体験者世代の物理的退場による現象。本atomは、**体験者世代がまだ生きているのに、1から3への質的劣化が起きる**現象を扱う。

劣化の駆動因子：
- 情報メディアの変化（口承・家庭内伝達 → 書籍・メディア → デジタル断片）
- 核家族化による世代間接触の減少
- 専門分業による経験の専門家への外部委託
- 平和・安全の継続による警戒の「奇妙さ」（警戒する側が浮く）

## 観察可能なproxy

- 「昔はそうだったらしい」という伝聞形での記憶の語られ方
- 専門家の警告と当事者の感覚の乖離（医師の警告が「過剰反応」に聞こえる）
- 祖父母世代の警戒行動が「神経質」「世代差」として処理される
- Wikipedia 記述・統計数値はあるが、それを参照しても行動が変わらない
- 経験者の証言が「個別エピソード」として一般性を剥奪される
- 同時代に同じ脅威の証拠があっても「自分の周りでは起きていない」で却下される

## 介入点（Disable conditions）

- 当事者の語りの一人称化（インタビュー・ドキュメンタリーで身体性を回復）
- 体験との接触機会の再設計（病院見学、追悼施設、被害者家族との対話）
- メタファーの身体化（数字を「自分の子の年齢」「自分の街の人口」に置換）
- 警戒する人が浮かない社会設計（小規模再発の許容、儀礼化）
- 経験記憶を持つ移民・難民との接触

## 補足

[generational-amnesia](generational-amnesia.md) との違い：
- **generational-amnesia**: 体験者世代の物理的退場（生物学的時間軸、60-80年）
- **experiential-amnesia**: 体験者世代が生存中の記憶の質的劣化（社会的時間軸、20-40年で起きうる）

両者は連動するが独立に観察可能。介入点も異なる：
- generational-amnesia → 世代交代前の制度化（オーラルヒストリー、教育、記念碑）
- experiential-amnesia → 世代内での身体性回復（接触機会の再設計）

experiential-amnesia の方が「犯人不在」性が強い。記憶を持つ世代が生きているのに、彼らの記憶が機能していない状態は、誰が責任を負うべきか特定が困難。

事例：
- **ビタミンK拒否（2026, 米）**：祖父母世代は1960年代の新生児頭蓋内出血を辛うじて記憶しているが、孫世代の親には「Wikipedia の歴史記述」として届く。記憶の質が身体化から他人事化へ劣化したことで、ワクチン拒否が「合理的選択」として現れる。`200_Output/substack-daily/2026-05-11-prevention-self-erasure.md` で扱う。
- **ワクチン懐疑全般**：感染症パンデミックを直接体験した世代がまだ多数生存中だが、その記憶が「個別エピソード」として一般性を失い、若い親世代に届かない。
- **金融危機の経験者**：2008年危機を実務で経験した銀行員がまだ現役だが、業界文化として「もう繰り返さない」が継承されず、リスクテイクが累積する構造。
