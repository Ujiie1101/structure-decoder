# 二次効果盲目

行為の**直接効果（一次）**は予測できても、**間接効果（二次・三次）**を予測できない認知傾向。

## 起源（Why this exists）

人間の予測能力は線形思考に最適化されている。「Aすれば B になる」は捕捉できるが、「Aすれば B になり、B が C を引き起こし、C が D を…」のような連鎖は計算量的にも認知的にも追跡困難。

加えて、二次効果は通常、**意図しない領域**で発生する。経済政策が文化に影響、技術導入が社会関係に影響、規制強化が地下経済に影響。介入者は自分の専門領域（一次効果）に注意を集中しているため、隣接領域（二次効果）の変化を見落とす。

これは個人の能力不足ではなく、**注意の有限性**と**領域の専門化**から自動的に発生する。

## 観察可能なproxy

- 政策評価レポートの分析範囲（一次のみ vs 二次まで含む）
- 「想定外でした」という事後発話の頻度
- 介入の副作用が「別の専門家」によって発見される
- システム思考・複雑系の教育レベル

## 介入点（Disable conditions）

- 二次効果の事前ブレインストーミング（pre-mortem の拡張）
- 領域横断レビュー（経済政策に文化人類学者を入れる等）
- システム思考訓練
- 過去の二次効果事例の組織記憶化
