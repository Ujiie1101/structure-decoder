## Dimensional Analysis

| Dimension | substitution-category | substitution-functional | substitution-scarcity |
|-----------|----------------------|------------------------|----------------------|
| **Object** | Organizing framework | Operational authority | Value concentration |
| **Speed** | Variable (can be slow or fast) | Usually gradual | Technology-driven |
| **Visibility** | High (contested) | Low (gradual) | Medium (market signals) |
| **Reversibility** | Low (once established) | Moderate | Low (path dependent) |
| **Trigger** | Legitimacy crisis | Efficiency differential | Cost collapse |
| **Mechanism** | Paradigm replacement | Function migration | Scarcity point shift |

## Selection Guide

```
Is the replacement about:

├── HOW things are perceived/legitimized?
│   └── substitution-category
│       (paradigm, doctrine, aesthetic canon)
│
├── WHO performs functions?
│   └── substitution-functional
│       (authority, operations, control)
│
└── WHERE value concentrates?
    └── substitution-scarcity
        (resource locus, attention, access)
```

## Interaction Patterns

Subtypes often co-occur:
- **category → functional**: New paradigm enables new authority structure
- **scarcity → category**: New value locus requires new legitimacy framework
- **functional → scarcity**: Authority migration shifts resource concentration

## Usage

- Use `substitution-category` when: framework, paradigm, or legitimacy basis changes
- Use `substitution-functional` when: operational control or authority transfers
- Use `substitution-scarcity` when: value concentration point shifts
- Use generic `substitution` when: classification unclear or multiple types present
