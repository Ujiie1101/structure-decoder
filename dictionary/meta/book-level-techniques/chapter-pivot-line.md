# 章を裏返す一行

章末（まれに章頭）に置かれた**それ単体で章全体の意味を後付けで反転させる短文**。読者は章を最後まで読んで初めて、章全体の出来事が「実は別の文脈に属していた」ことを知る。

## 起源（Why this exists）

書物は時間連続的に読まれる。読者は前から後ろへ意味を組み立てるが、最後の一行で組み立てが**事後的に解体・再構築**されうる。

この再構築は、書き手の編集権力の源泉である。長い章全体を、最後の一行で**異なる意味の前置き**に変換できる。読者は「読み直し」を強制されるが、書字の不可逆性により**書かれた章は変えられない**ため、解体は読者の認知の中で完結する。

進化的には、危険信号の事後想起（あの音は捕食者だった）の物語化された応用。

## 観察可能なproxy

- 短編小説のラスト一行による全体の意味反転
- 推理小説の犯人開示が前章までの描写を再解釈させる
- 訃報・破綻発表が組織の過去の言動を全て遺言化する
- 創業者の遺書の最後の一文が経営判断の意味を変える
- ニュース記事の見出しと最終段落の対比による意味の逆転

## 介入点／応用可能性

- 章末の一行を**意図的に置かない**（フラットな終わり方）
- 章末の一行を**対比的に並べる**（複数の解釈を残す）
- 章末の一行を**冒頭で予告する**（裏返しの効果を中和する）
- 組織報告書・遺書・公式声明の「最終文の権力」への自覚

## 太宰治『斜陽』における運用

| 章 | 章末の一行 | 反転効果 |
|---|---|---|
| 1 | （胸の中の蛇）| 後続全章への徴候投下 |
| 2 | 「私たちの本当の地獄がはじまった」| 火事の章を地獄論の前置きに |
| 3 | 「不良とは、優しさの事ではないかしら」| 過去の侵入の章を倫理転倒の前置きに |
| 4 | 「マイ、チャイルド」| 押しかけ愛人の手紙を母性宣言に裏返す |
| 5 | 「ピエタのマリヤ」| 母の死の章を宗教画化 |
| 6 | 「弟の直治は、その朝に自殺していた」| 上京と幻滅の章を遺書の前置きに |
| 7 | 「僕は、貴族です」| 自殺の説明を出自の最終引き受けに |
| 8 | 「マイ、コメデアン。昭和二十二年二月七日」| 書物全体を喜劇として送信 |

全八章で運用された結果、書物は「章単位の徴候投下→章末で意味確定→次章の前置きへ」という連鎖構造を持つ。各章末は次章の重力場を決定する。
