# 書名の自己反転

書名・タイトル・組織名が**書物・物語・運動の終盤で逆の意味に書き換えられる、または同じ語が逆の事象を指す**装置。書名は冒頭で固定されるが、内容の終端で**事後的に再解釈**される。

## 起源（Why this exists）

書名は書字の最初の確定要素で、書物全体に対するメタタグとして機能する。読者は書名から内容を予測し、内容は書名に従って読まれる。

しかし書名は**一語または短句**で書物全体を圧縮するため、解釈の余地が広い。書物の終端で、書き手が**書名と内容の関係を逆転させる**操作を行えば、書名そのものが新しい意味を獲得する。これは書字の不可逆性を逆手に取った技法で、書名は変更できないからこそ、内容によって**書名の意味が後付けで定義し直される**。

進化的には、警告信号の自己反転（あの音は危険ではなかった）の言語化された応用。

## 観察可能なproxy

- 『斜陽』の終端で「太陽」が出現し、書名が逆転する
- 『罪と罰』のラスコーリニコフが「罰」を「贖い」に置き換える
- 企業の社名と実態の事後乖離（Theranos の "truth" が嘘の代名詞に）
- 運動の名称の事後反転（"進歩主義"が停滞の代名詞に、"自由主義"が抑圧の代名詞に）
- 製品名の意味の歴史的反転（"オーガニック"の商業化など）

- 書名と内容の整合性監査（後付け反転を見抜く）
- 「○○協会」「○○の会」が時間と共に逆の意味を帯びる現象の予測

## 太宰治『斜陽』における運用

```
書名「斜陽」（傾いていく太陽、没落の象徴）
  ↓ 全八章を通じて没落の徴候が累積される
  ↓ 第八章末「ピエタのマリヤ」「マイ・コメデアン」で
  ↓ 書名と内容の関係が反転する
  ↓
最終手紙の語彙：「太陽」「革命」「コメデアン」
  → 没落していたのは「斜陽」族であり、書物の主体（かず子）は「太陽」へ向かう
  → 書名「斜陽」は、書物の主体ではなく、過去の階級の名前として確定する
```

書名そのものが書物の主役だったが、終端で**反転して主役の対極**になる。読者は書物を読み終えた後、書名を**逆の意味で**思い出す。これは書名による自己消去の操作と言える。
