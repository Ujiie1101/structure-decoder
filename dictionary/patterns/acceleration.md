# acceleration（加速）

変化速度の増大から、質的に異なる状態・経験・社会構造が創発する構造。

## 定義

量的な速度増加が閾値を超えることで、単なる「速い」ではなく、根本的に異なる存在様式、経験の質、社会的関係が出現する。

## 構造

```
baseline_velocity
  → speed_increase
    → threshold_crossing
      → qualitative_shift
        → emergent_modality (new mode of being)
```

## 発動条件

1. 加速を可能にする技術・制度
2. 加速への動機・圧力
3. 適応メカニズムの存在
4. 資源の持続的供給

## 失敗条件

- **摩擦**: 加速を阻む抵抗
- **疲弊**: 持続不可能な速度
- **事故**: 制御不能による破綻
- **虚無**: 意味を置き去りにする速度

## 関連パターン

- **emergence**: 加速から新モードが創発
- **transformation**: 加速による質的変革
- **cascade**: 加速が連鎖を誘発
