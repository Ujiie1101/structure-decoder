# accumulation（蓄積）

時間をかけて資源・知識・力が増大していく構造。

## 定義

継続的な追加・成長により、ストックが増大する。量的蓄積が質的変化をもたらす臨界点、複利効果、path dependency を含む。

## 構造

```
initial_stock
  → continuous_addition
    → compound_growth
      → critical_mass
        → qualitative_transformation
```

## 発動条件

1. 蓄積可能な資源/知識
2. 保存・維持メカニズム
3. 継続的インプット
4. 複利効果/正のフィードバック

## 失敗条件

- **散逸**: 蓄積の漏出・消失
- **停滞**: インプットの枯渇
- **収穫逓減**: 追加の価値低下
- **崩壊**: 蓄積物の急激な喪失

## 適用事例

| Layer | Entities |
|-------|----------|
| technology | knowledge-accumulation, data-accumulation |
| economics | capital-accumulation, wealth-concentration |
| science | scientific-progress, paradigm-building |
| history | civilizational-development |

## 関連パターン

- **emergence**: 蓄積からの質的変化
- **transformation**: 臨界点での相転移
- **erosion**: 蓄積の漸進的喪失
