# alternating-current（交流）

方向を周期的に反転させることで、直線的アプローチでは不可能な効率・到達を実現する構造。

## 定義

一方向の連続ではなく、振動・反転・周期を利用する。電気の交流から、異文化交流、思考の往復まで、反対方向の動きが全体の効率を高める。Tesla vs Edison の「電流戦争」の教訓。

## 構造

```
unidirectional_limit
  → direction_reversal
    → periodic_oscillation
      → efficiency_gain
        → extended_reach
```

## 発動条件

1. 直線的アプローチの限界
2. 反転可能な構造
3. 周期の最適化
4. 変換機構の存在

## 失敗条件

- **同期失敗**: 周期のずれ
- **変換損失**: 反転コストの過大
- **共振破壊**: 過度の振幅
- **単一方向固執**: 反転拒否
