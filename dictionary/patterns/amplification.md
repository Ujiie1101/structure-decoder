# amplification（増幅）

小さな信号・影響・動きが拡大し、システム全体に大きな効果を及ぼす構造。

## 定義

初期の微小な変化や刺激が、正のフィードバックループやネットワーク効果を通じて指数関数的に拡大し、全体の状態を変える。

## 構造

```
initial_signal
  → positive_feedback
    → network_propagation
      → exponential_growth
        → systemic_effect (scale beyond origin)
```

## 発動条件

1. 増幅可能な媒体・ネットワーク
2. 正のフィードバック機構
3. 抑制力の不在または弱体
4. 受容する環境条件

## 失敗条件

- **減衰**: 負のフィードバックによる抑制
- **飽和**: 増幅の上限到達
- **分散**: 信号の拡散・希釈
- **遮断**: 伝播経路の断絶

## 適用事例

| Layer | Entities |
|-------|----------|
| religion | evangelism, reformation, charismatic-movement |
| technology | viral-media, network-effect, social-media |
| economics | bubble, bank-run, speculation |
| politics | propaganda, mass-movement |

## 関連パターン

- **cascade**: 増幅が連鎖的波及を引き起こす
- **feedback**: 増幅の基盤メカニズム
- **emergence**: 増幅から新秩序が創発

## メカニズムID

**[E-01] Invisible Hand Coordination** — 分散的増幅が全体効果を創発
