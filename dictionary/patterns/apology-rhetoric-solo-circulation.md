# apology-rhetoric-solo-circulation（謝罪レトリックの単独流通）

謝罪の言葉と、それに伴う**処分・行動変化**が別々に流通する装置。謝罪は宣言される、しかし処分は別人に向けられ、行動は変わらない。

## 定義

謝罪は本来三層から成る:

| 層 | 内容 |
|---|---|
| 第一層 | 内面の悔恨 |
| 第二層 | 外面の所作（頭を下げる、言葉を発する） |
| 第三層 | 以後の行動の変化 |

通常は第一層と第三層が連動する。本パターンは**第二層だけを切り出して流通させる**運用。

組織は儀礼の完了を確認する装置であって、行動の変化を追跡する装置ではない。だから第二層の完了だけで「儀礼は終わった」と判定される。

## 構造

```
incident（事件発生）
  → public_apology（公式謝罪）
    → layer_separation（謝罪三層の分離）
      ├─ layer_1（内面）: not required
      ├─ layer_2（所作）: completed
      └─ layer_3（行動）: not tracked
        → ritual_closure（儀礼の完了判定）
          → unchanged_behavior（行動は不変）
            → repeated_incident（同種事件の再発）
```

## 観察可能な現代具体例

- 記者会見で深々と頭を下げた経営者の組織で、翌年同じ不祥事が繰り返される
- 「お騒がせして申し訳ありません」と投稿したインフルエンサーが、翌週同じパターンで炎上
- SDG'sや多様性の宣言を毎年更新する企業の現場で、何も変わらない
- 学校で生徒が「公開謝罪」した翌日、教師への呼びかけが継続する
- 政治家の「真摯に受け止める」発話の翌週、同じ問題が再発する

## 対抗装置

- 謝罪後の行動を**期限付きで追跡**する仕組み
- 第三層（行動変化）の指標を事前に合意する
- 「儀礼の完了」と「問題の解決」を別途判定する責任者の分離
