# appropriation（流用）

既存の素材・形式・文脈を取り込み、新たな意味が出現する構造。

## 定義

他者の作品、大衆文化、歴史的素材、異文化の要素などを意図的に借用・転用し、新しい文脈に置くことで、オリジナルとは異なる意味や批評的視点を生成する。

## 構造

```
source_material (original_context)
  → extraction
    → recontextualization
      → meaning_displacement
        → new_significance (critical/creative)
```

## 発動条件

1. 引用可能な既存素材の存在
2. 意図的な文脈の転換
3. オリジナルとの差異の認識
4. 批評的または創造的意図

## 失敗条件

- **単なる模倣**: 文脈転換なしのコピー
- **文化収奪**: 権力関係の無視
- **著作権問題**: 法的リスク
- **意味の消失**: 参照元が忘却される

## 適用事例

| Layer | Entities |
|-------|----------|
| art | pop-art, found-object, remix-culture, sherrie-levine |
| music | sampling, cover-version, mashup |
| literature | pastiche, collage, intertextuality |
| fashion | vintage, street-style |
| technology | open-source, api |

## 関連パターン

- **emergence**: 新意味の創発
- **transformation**: 素材の変容
- **dematerialization**: 概念としての流用
