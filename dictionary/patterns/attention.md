# attention（注意）

限られた認知資源を特定の対象に選択的に配分する構造。

## 定義

無数の刺激・情報の中から、特定のものを選択的に処理し、他を無視することで、限られた認知資源を効率的に活用するメカニズム。デジタル時代においては希少資源・通貨として機能する。

## 構造

```
information_overload
  → selective_filtering
    → focused_processing
      → attention_allocation
        → action_guidance (behavior shaped by focus)
```

## 発動条件

1. 情報・刺激の過剰
2. 処理能力の有限性
3. 選択のための基準・優先順位
4. 目標・関心の存在

## 失敗条件

- **分散**: 注意の断片化（マルチタスク）
- **捕獲**: 無関係な刺激への固着
- **枯渇**: 注意疲労・燃え尽き
- **操作**: 注意の外部的誘導（広告、通知）

- **amplification**: 注意の集中による増幅
- **feedback**: 注意パターンの強化
