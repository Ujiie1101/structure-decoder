# Authority-Gradient Evidence Pattern（権威勾配事例パターン）

## 定義
同一構造の事例を「権威レベルの勾配」に沿って並べることで、「誰でも該当する」普遍性を示す事例提示手法。

## 公式
```
事例1: [一般層/若年]
事例2: [中堅]
事例3: [著名層]
事例4: [巨匠/世界的権威]
↓
「権威の強弱に関わらず、同じ構造が発動する」
→ 読者は「自分も例外ではない」と気づく
```

## メカニズム

### 1. 勾配による普遍性証明
同じカテゴリ内で並べると「業界特有の問題」に見える。
異なる権威レベルで並べると「構造の問題」に見える。

### 2. 最後に巨匠を配置する効果
「坂本龍一ですら潰された」のように、最高権威者を最後に置くと、「この構造から逃れる方法はない」と読者が悟る。

### 3. 数値エビデンスの積層
各事例に数値（金額/人数/期間/回数）を入れることで、印象論ではなくデータに見せる。

## 成功条件
- 事例が**異なるカテゴリ**から選ばれている（業界・国・時代）
- 各事例に**数字**が入っている
- 時系列は必ずしも年順である必要はない（**権威勾配優先**）

## 失敗パターン
- 同業界の事例ばかり → 「特殊問題」に見える
- 数字がない → 印象論になる
- 権威の強弱が曖昧 → 勾配効果が出ない

## 事例

### Day 25「99万人の沈黙」
```
1. ローラ（2018, タレント, 署名4万件）
2. 村本大輔（2017〜, 芸人, 出演100本→1本）
3. きゃりーぱみゅぱみゅ（2020, アーティスト, 分断→削除）
4. 坂本龍一（2023, 世界的音楽家, 小池都知事に一蹴）
```
→ タレント→芸人→アーティスト→巨匠 の勾配

## 適用条件
- 構造的問題の普遍性を示したい記事
- 「一部の過激事例」という反論を封じたい場合
- 5件以内に絞れる（多すぎると散漫）

## 推奨事例数
- **3件**: 最低ライン
- **4〜5件**: 最適
- **6件以上**: 削る

## 注意点
- 各事例の説明は「行動→結果」で統一する
- 事例間の分量を均等にする（特定事例だけ長いと偏って見える）
