# authority-inversion（権威逆転）

従属者が支配者を凌駕し、権力関係が逆転する構造。

## 定義

権力・権威・正統性の非対称的関係が、時間経過と状況変化により逆転する。かつての従属者が新たな権威となり、かつての支配者が従属的地位に転落する。

## 構造

```
established_hierarchy
  → subordinate_accumulation (power, legitimacy, resources)
    → superior_decline (rigidity, corruption, obsolescence)
      → legitimacy_transfer
        → role_reversal
          → authority_inversion (emergent new order)
```

## 発動条件

1. 従属者の能力・資源蓄積
2. 支配者の硬直化・腐敗
3. 正統性の源泉の変化
4. 外部環境の変動

## 失敗条件

- **抑圧**: 従属者の成長阻止
- **適応**: 支配者の自己革新
- **共生**: 相互依存関係の維持
- **正統性維持**: 権威の基盤保持
