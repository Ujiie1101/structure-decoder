# balance（均衡）

対立する力・利害・要素が相互に牽制し合い、安定状態が出現する構造。

## 定義

複数の競合する力や主体が、相互作用を通じて一定の安定状態に達する。いずれの主体も一方的優位を獲得できず、緊張関係の中で秩序が維持される。均衡は静的なものと動的なものがある。

## 構造

```
competing_forces (A, B, C...)
  → mutual_checking
    → counterbalancing
      → stabilization
        → equilibrium (no unilateral dominance)
```

## 発動条件

1. 複数の対等な力の存在
2. 相互牽制のメカニズム
3. 一方的勝利の困難さ
4. 均衡維持のインセンティブ

## 失敗条件

- **ヘゲモニー**: 一方の力の圧倒的優位
- **外乱**: 外部ショックによる均衡崩壊
- **arms race**: 軍拡競争による不安定化
- **膠着**: 発展の停止
