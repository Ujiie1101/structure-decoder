# bankruptcy（破産）

既存の美的価値・様式・制度の崩壊から、新たな芸術的可能性が生まれる構造。

## 定義

従来の芸術規範・価値体系が行き詰まり、もはや機能しなくなる状態。この「破産」は終わりではなく、古い枠組みからの解放であり、ラディカルな再出発を可能にする。アヴァンギャルドの前提条件。

## 構造

```
established_order (conventions, institutions)
  → exhaustion_and_crisis
    → loss_of_credibility
      → collapse
        → void (creative_vacuum)
          → radical_renewal
```

## 発動条件

1. 既存様式の疲弊・形骸化
2. 社会的・文化的危機
3. 破壊を恐れない先駆者
4. 新しい可能性への予感

## 失敗条件

- **虚無**: 再建なき破壊
- **ノスタルジア**: 過去への退行
- **空転**: 破壊の自己目的化
- **商品化**: 破産の様式化
