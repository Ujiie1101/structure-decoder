# beet-sugar-law（ビート糖の法則）

封鎖が、封鎖されたものより強い代替を創発させる構造。

## 定義

チョークポイントや供給源の封鎖・遮断が、一時的な欠乏を経て、封鎖前には「非効率」「ニッチ」とされていた代替品・代替技術・代替ルートを創発させ、最終的に封鎖前より強い構造を生み出す。封鎖した側は長期的に敗北し、代替を提供した側が勝利する。

## 構造

```
blockade (封鎖)
  → scarcity (欠乏)
    → search_for_alternatives (代替の探索)
      → "inefficient" solution resurfaces (非効率とされた解の復活)
        → refinement_under_pressure (圧力下での技術磨き)
          → stronger_structure_than_before (封鎖前より強い構造)
```

## 発動条件

1. **チョークポイントの存在**: 供給が単一点に依存
2. **封鎖の発生**: 意図的（経済制裁）または非意図的（災害）
3. **代替候補の存在**: 封鎖前から「非効率」とされていた代替品
4. **封鎖の持続**: 代替が成熟するための時間的猶予

## 失敗条件

- **封鎖の短期終了**: 代替が成熟する前に元の供給が回復
- **代替候補の不在**: 技術的に代替が存在しない
- **技術磨きの失敗**: 封鎖解除後の競争に敗れる（ドイツ・ビート糖）

## 勝者の3類型

| 類型 | 説明 | 歴史的例 |
|------|------|----------|
| **在庫保有者** | 平時に「非効率」と笑われたバッファを持つ者 | トヨタ（半導体在庫） |
| **代替提供者** | 封鎖前は「ニッチ」だった代替を提供できる者 | フランス・ビート糖産業 |
| **学習者** | 今回負けても、次の危機で勝つ者 | トヨタ（2011→2021） |

## 適用事例

| Layer | Event | 封鎖されたもの | 創発した勝者 |
|-------|-------|--------------|------------|
| economics | 大陸封鎖令（1806-1814） | サトウキビ糖 | フランス・ビート糖産業 |
| economics | スエズ危機（1956/1967） | 運河ルート | 日本造船業（VLCC/ULCC） |
| economics | 東日本大震災（2011） | ルネサス半導体 | トヨタ（学習→2021年勝利） |
| economics | ホルムズ封鎖（2026） | ナフサ→溶剤 | （進行中） |

## 裏面パターン: 効率の呪い

「ビート糖の法則」の裏面として、**効率化した者が有事に負ける**構造がある。

|  | 平時 | 有事 |
|--|------|------|
| **効率化（JIT）** | 称賛される | 崩壊する |
| **冗長性保持** | 笑われる | 生き残る |

効率は平時の最適化であり、有事の脆弱性である。

## 関連パターン

- **substitution**: 代替による置換（ビート糖の法則は「封鎖が引き金」という特殊ケース）
- **fragility**: 効率化による脆弱性の蓄積
- **cascade**: 封鎖から連鎖崩壊への遷移
- **convergence**: 封鎖前は分散していた代替が、封鎖後に収束

## 診断的応用

封鎖・危機のニュースを見たとき、以下の問いで勝者を予測できる：

1. **何が封鎖されたか？**（チョークポイントの特定）
2. **封鎖前に「非効率」とされていた代替は何か？**（勝者候補の特定）
3. **代替が成熟する時間はあるか？**（封鎖の持続期間の見積もり）

## 核心洞察

> **「封鎖した側は長期的に負け、代替を提供した側が勝つ」**

ナポレオンは封鎖を仕掛け、敗北した。封鎖は意図しない創発を生む。
