# blindspot（盲点）

認識枠組み・専門性・立場に起因する、特定の現実を見落とす構造的傾向。

## 定義

ある視点・専門性・成功体験が、それとは異なる角度からのみ見える現実を体系的に見落とさせる。盲点は自覚が困難であり、しばしば致命的な判断ミスの原因となる。

## 構造

```
perspective_formation
  → selective_attention
    → systematic_neglect
      → invisible_risk_accumulation
        → surprise_failure (from unseen factors)
```

## 発動条件

1. 特定の視点・専門性への深い習熟
2. 成功体験による確信の強化
3. 同質的な環境（エコーチェンバー）
4. フィードバック機構の欠如

## 失敗条件

- **多様性**: 異なる視点の積極的取り込み
- **外部監査**: 第三者による検証
- **失敗分析**: 過去の見落としからの学習
- **謙虚さ**: 自己の限界の認識
