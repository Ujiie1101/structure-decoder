# breakthrough（突破）

限界への持続的挑戦から、従来の制約を超えた新次元・新領域が創発する構造。

## 定義

既存の限界・障壁への集中的取り組みが臨界点を超えることで、漸進的改善ではなく、不可能と思われていた領域への飛躍が出現する。

## 構造

```
existing_limit
  → sustained_pressure
    → critical_insight
      → barrier_transcendence
        → emergent_possibility (new domain)
```

## 発動条件

1. 明確な限界の認識
2. 突破への執念
3. 失敗からの学習
4. 予期せぬ視点転換

## 失敗条件

- **諦念**: 限界の受容
- **分散**: 集中力の欠如
- **固執**: 同一アプローチの反復
- **孤立**: 必要な協力の欠如

- **emergence**: 突破から新可能性が創発
- **transformation**: 突破による質的変革
- **paradigm-shift**: 突破の認識論的形態
