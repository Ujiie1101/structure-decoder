# bystander-economics（傍観の経済学）

傍観は安く、行動は高い。この非対称性が、組織的な傍観を生む構造。

## 定義

警告や問題を認識していても行動しない選択が、経済的に合理的になる状況。

> 傍観のコスト ≈ 0
> 行動のコスト > 0
> → 傍観が選ばれる

## 構造

```
警告・問題の認識
  → 行動するかどうかの判断
    → コスト計算
      → 傍観: コストゼロ、リスク不明確
      → 行動: 調査コスト、法的リスク、売上減少
        → 傍観が選ばれる
          → 被害発生
            → 「知らなかった」「対応できなかった」
```

## 発動条件

1. **コストの非対称性**: 傍観は無料、行動は有料
2. **ベネフィットの不可視性**: 「防いだ被害」は測定できない
3. **責任の分散**: 誰か一人の責任にならない
4. **大量の警告**: 個別対応が「不可能」という言い訳

## 2×2マトリックス: 警告と行動

|  | **行動した** | **行動しなかった** |
|---|---|---|
| **警告あり** | 被害防止（理想） | **傍観者の責任** |
| **警告なし** | 過剰反応 | 知らなかった |

右上の象限（警告あり＋行動しなかった）が問題。

## 適用事例

| 領域 | 警告 | 傍観 | 結果 |
|------|------|------|------|
| OpenAI/ストーカー事件 | 3回の警告 | 何もしなかった | 訴訟 |
| ボーイング737 MAX | エンジニアの警告 | 無視 | 2件の墜落事故 |
| Theranos | 内部告発 | 無視 | 90億ドル詐欺 |
| セクハラ事件 | 被害者の報告 | 無視 | 組織的隠蔽 |
| SNSの有害コンテンツ | 大量の報告 | 対応不能 | 被害の拡大 |

## グッドハートの法則との関係

傍観の経済学はグッドハートの法則と結びつく。

- **測定できるもの**: 売上、ユーザー数、対応件数
- **測定できないもの**: 防いだ被害、避けた訴訟、守った信頼

測定できるものだけが最適化される。「今四半期、何件の被害を防ぎましたか？」には答えられない。

## ミルグラムの35%との関係

ミルグラム実験で35%は拒否した。

彼らは「これでいいのか」と問い、行動した。

傍観の経済学を打破するのは、**個人の倫理的判断**。

- 経済的にはコストがかかる
- しかし、「これでいいのか」と問う
- そして、35%の側に立つ

## 失敗条件（傍観を防ぐ条件）

1. **行動のコストを下げる**: 報告→自動対応のシステム化
2. **傍観のコストを上げる**: 法的責任の明確化
3. **ベネフィットの可視化**: 「防いだ被害」の推計・報告
4. **責任の個人化**: 「誰が判断したか」の記録
5. **倫理的文化**: 「これでいいのか」と問う習慣

## 関連パターン

- **悪の凡庸さ（thoughtlessness）**: 思考の欠如が道徳的破局を可能にする
- **責任の分散**: 集団では誰も責任を取らない
- **Goodhart's Law**: 測定できるものだけが最適化される
- **傍観者効果（Bystander Effect）**: 周りに人がいるほど、誰も助けない
