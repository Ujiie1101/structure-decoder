# capture（捕獲）

瞬間・現象・リアリティを記録・固定することで、新たな表象が出現する構造。

## 定義

流れ去る時間、一回性の出来事、生のリアリティを、技術的・芸術的手段によって固定・保存する。捕獲された瞬間は、現実とは異なる独自の存在論的地位を獲得する。

## 構造

```
fleeting_reality
  → observation/framing
    → technical_mediation (camera, recording)
      → fixation
        → captured_representation (new ontological status)
```

## 発動条件

1. 捕獲技術・媒体の存在
2. 観察者の視点・選択
3. フレーミング（何を含め、何を除外するか）
4. 保存・再生可能性

## 失敗条件

- **取りこぼし**: 本質的要素の欠落
- **過剰介入**: 捕獲が現実を変容させる
- **脱文脈化**: 元の文脈が失われる（atom化: [context-stripping](../atoms/context-stripping.md)）
- **劣化**: 記録媒体の消失

## 適用事例

| Layer | Entities |
|-------|----------|
| art | photography, documentary, street-photography |
| technology | recording, archive, database |
| science | observation, data-collection |
| journalism | reportage, witness |
| anthropology | ethnography, fieldwork |

## 関連パターン

- **emergence**: 捕獲による新表象の出現
- **dematerialization**: 捕獲と概念化
- **appropriation**: 捕獲された素材の転用
