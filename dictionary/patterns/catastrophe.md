# catastrophe（破局）

蓄積した歪みが臨界点を超え、システムが突然・不可逆的に崩壊する構造。

## 定義

連続的な変化が非連続的な結果を生む。カタストロフ理論が示すように、緩やかな圧力が蓄積し閾値を超えた瞬間、システムが別の状態に急激にジャンプする。事前予測は困難で、事後には「必然」に見える。

## 構造

```
gradual_stress_accumulation
  → hidden_threshold_approach
    → trigger_event (often minor)
      → sudden_state_transition
        → irreversible_new_state
```

## 発動条件

1. システム内の歪み蓄積
2. 隠れた閾値の存在
3. 正のフィードバック（崩壊加速）
4. 復元力の枯渇

## 失敗条件（破局回避）

- **早期介入**: 歪み蓄積前の是正
- **安全弁**: 圧力逃がしメカニズム
- **冗長性**: バックアップシステム
- **監視**: 早期警戒システム
