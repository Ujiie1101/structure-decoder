# causal-misattribution（因果誤帰属）

結果の真の原因を誤認し、誤った因果関係を信じる構造。

## 定義

複雑な現象の原因を、認知バイアス・イデオロギー・利害関係により、実際とは異なる要因に帰属させる。誤帰属は個々の認知傾向の集積から社会的に創発し、政策・行動・歴史解釈を歪める。

## 構造

```
complex_phenomenon
  → cognitive_simplification
    → available_explanation (salient, simple, convenient)
      → confirmation_bias
        → social_reinforcement
          → causal_misattribution (emergent false consensus)
```

## 発動条件

1. 複雑な因果関係
2. 認知的省力化の欲求
3. イデオロギー的誘導
4. 利害関係者による叙述操作

## 失敗条件

- **検証**: 厳密な因果分析
- **多元的視点**: 複数の説明の併存
- **時間経過**: 真の因果の顕在化
- **利害の変化**: 誤帰属の利益消滅
