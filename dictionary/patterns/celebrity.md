# celebrity（名声）

注目の集積から、実体を超えた社会的存在が出現する構造。

## 定義

個人や作品への注目が臨界量を超えて集積することで、本来の実体や業績を超えた独立した社会的存在が創発する。名声は実績の反映ではなく、注目の自己増殖的システムである。

## 構造

```
individual_achievement
  → public_attention
    → attention_accumulation
      → self-reinforcing_cycle
        → celebrity_emergence (autonomous from achievement)
```

## 発動条件

1. 初期の注目喚起
2. 注目の可視化と共有
3. 自己増殖的フィードバック
4. メディアの増幅機能

## 失敗条件

- **忘却**: 注目の持続失敗
- **スキャンダル**: 負の注目への転化
- **燃え尽き**: 注目維持の疲弊
- **実体乖離**: 名声と実力の致命的乖離
