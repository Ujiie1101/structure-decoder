# chilling-effect（萎縮効果）

見せしめによって大多数が自己検閲し、沈黙が拡大する構造。

## 定義

特定の個人への制裁・攻撃が可視化されることで、それを観察した多数が「同じ目に遭いたくない」とリスクを認知し、自発的に沈黙を選択する現象。見せしめは「対象」を潰すためではなく、「観客」を黙らせるために機能する。

## 構造

```
exemplary_punishment（見せしめ）
  → high_visibility（高い可視性）
    → risk_perception_by_observers（観察者のリスク認知）
      → self_censorship（自己検閲）
        → mass_silence（大衆の沈黙）
          → normalization_of_silence（沈黙の正規化）
```

## 核心構造

**1人を潰すことで、99万人を黙らせる。**

- 見せしめのコスト：1人分
- 見せしめの効果：99万人分
- ROI：99万倍

## 発動条件

1. 制裁の可視性（SNS、メディア報道）
2. 制裁対象と観察者の同一視可能性（「自分もああなるかも」）
3. 制裁の不可逆性・深刻さ（謝罪、失職、社会的排除）
4. 制裁基準の曖昧さ（「何がアウトかわからない」）

## 失敗条件

- **連帯**: 多数が同時に発言し、見せしめが機能しない
- **匿名性**: 制裁対象を特定できない
- **制裁コストの顕在化**: 見せしめ側が批判される
- **プラットフォーム移動**: 制裁が及ばない空間への退避

## 数値構造

| 要素 | 数値例 |
|------|--------|
| 見せしめ対象 | 1人 |
| 直接観察者 | 100万人 |
| 沈黙選択者 | 99万人（99%） |
| 発言継続者 | 1万人（1%） |

## 関連パターン

- **normalization**: 沈黙が正規化される
- **groupthink**: 沈黙が「普通」になると異論が出なくなる
- **output-less-critique**: 批判者は見せしめのコストを負わない
- **purity**: 見せしめは純粋性維持の手段
