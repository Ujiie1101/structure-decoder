# codification（成文化）

口承・慣習・暗黙知が文字・法典・教義として固定され、制度が出現する構造。

## 定義

流動的で多様な伝承・実践・知識が、権威ある文書として書き記され、正典化される。成文化により、解釈の基準が確立し、制度的伝達が可能になるが、同時に固定化・硬直化のリスクも生じる。

## 構造

```
oral_tradition / tacit_practice
  → collection / compilation
    → selection / editing
      → authorization
        → canonical_text (scripture, law code)
          → institutional_transmission
```

## 発動条件

1. 伝統の危機意識（消失の恐れ）
2. 文字文化・記録技術の存在
3. 編纂権威の確立
4. 統一・標準化の必要性

## 失敗条件

- **分裂**: 複数の正典の並立
- **硬直化**: 文字への過度の固執
- **形骸化**: 精神の喪失
- **排除**: 成文化されなかった伝統の消失

## 適用事例

| Layer | Entities |
|-------|----------|
| religion | bible, quran, talmud, tripitaka |
| law | code-of-hammurabi, justinian-code, constitution |

| language | grammar, dictionary, standardization |
| technology | protocol, specification, api-documentation |
