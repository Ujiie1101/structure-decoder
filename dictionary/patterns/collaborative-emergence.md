# collaborative-emergence（協働的創発）

複数主体の相互作用から、個人では到達不能な創造が出現する構造。

## 定義

複数の創作者が相互作用することで、個々の能力や意図の総和を超えた作品、アイデア、または価値が創発する。協働は分業ではなく、創造的相乗効果の発生装置である。

## 構造

```
individual_creators
  → interaction_space
    → mutual_stimulation
      → synergistic_feedback
        → collective_emergence (irreducible to individuals)
```

## 発動条件

1. 多様な参加者
2. 相互作用の場
3. 共通言語または媒介
4. 個人の主体性と集団への開放性の両立

## 失敗条件

- **支配**: 一者による他者の抑圧
- **分裂**: 協働の崩壊
- **妥協**: 創造性なき折衷
- **責任拡散**: 当事者意識の希薄化

## 適用事例

| Layer | Entities |
|-------|----------|
