# collapse（崩壊）

システムが維持不可能となり、急速に機能を失う構造。

## 定義

複雑なシステムが、内部矛盾・外部圧力・資源枯渇などにより、その構造と機能を維持できなくなり、より単純な状態へと急速に移行する。

## 構造

```
complex_system (stable equilibrium)
  → stress_accumulation
    → threshold_approach
      → trigger_event
        → cascade_failure
          → systemic_breakdown
            → simplified_state
```

## 発動条件

1. 累積的ストレス（過負荷、過剰複雑性）
2. 適応能力の枯渇
3. トリガーイベント
4. 連鎖的失敗メカニズム

## 失敗条件（崩壊回避）

- **適応**: システムが変化に対応
- **外部介入**: 救済・支援による安定化
- **ストレス緩和**: 負荷の分散・除去
- **冗長性**: バックアップシステムの機能

## 適用事例

| Layer | Entities |
|-------|----------|
| history | fall-of-rome, soviet-collapse |
| economics | financial-crisis, hyperinflation |
| technology | system-failure, infrastructure-collapse |
| politics | regime-collapse, state-failure |
| religion | schism, institutional-decline |

## 関連パターン

- **erosion**: 崩壊に先立つ漸進的弱体化
- **cascade**: 崩壊の連鎖的拡大
- **substitution**: 崩壊後の新システム出現
