# competition（競争）

複数の主体が限られた資源・地位・市場を争うことで、効率・革新・淘汰が創発する構造。

## 定義

希少な資源（顧客・資金・地位・生存）をめぐる複数主体間の争い。競争圧力が各主体の行動を規律し、全体として効率性向上・イノベーション・適者生存を生み出す。過当競争は破壊的消耗を、競争不足は停滞を招く。

## 構造

```
multiple_actors
  → scarce_resources
    → rivalry
      → selection_pressure
        → emergent_outcomes (efficiency, innovation, elimination)
```

## 発動条件

1. 複数の競争主体
2. 希少性のある目標・資源
3. 参入・退出の可能性
4. 勝敗を決する評価基準

## 失敗条件

- **独占**: 競争不在による停滞
- **共謀**: カルテル・談合
- **過当競争**: ゼロサム消耗戦
- **基準操作**: 評価基準の歪曲
