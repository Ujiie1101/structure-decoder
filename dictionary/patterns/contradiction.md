# contradiction（矛盾）

同一システム内に両立不能な要素が共存することで、緊張・運動・変容が創発する構造。

## 定義

論理的または実践的に共存できない二つの要素が同一の枠組み内に存在し続けることで、システムに内在的な不安定性と変化への圧力が生じる。矛盾は解消を求めて運動を生む。

## 構造

```
incompatible_elements
  → coexistence_within_system
    → internal_tension
      → instability
        → movement_toward_resolution (emergent dynamic)
```

## 発動条件

1. 両立不能な要素の共存
2. 統一的枠組みによる強制的共存
3. 矛盾の認識
4. 解消への内在的圧力

## 失敗条件

- **分離**: 要素が別システムに分離
- **抑圧**: 一方の要素の無視・隠蔽
- **形骸化**: 矛盾の認識の喪失
- **止揚**: 高次の統合による解消

## 適用事例

| Layer | Entities |
|-------|----------|
| religion | theodicy, free-will-omniscience, incarnation-paradox |
| philosophy | antinomy, liar-paradox, being-becoming |
| economics | growth-sustainability, efficiency-equity |
| politics | freedom-security, individual-collective |
| history | slavery-liberty-declaration, imperial-democracy |

## 関連パターン

- **dialectic**: 矛盾からの止揚
- **fragmentation**: 矛盾による分裂
- **transformation**: 矛盾解消による変容
