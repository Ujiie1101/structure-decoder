# convergence（収束）

異なる軌道・技術・システムが、共通の方向や形態に向かって近づく構造。

## 定義

当初は独立して発展していた複数の要素が、進化の過程で類似の形態・機能・目的地に到達する。技術収束（メディア融合）、生物収束（収斂進化）、制度収束（制度的同型化）などの形態がある。

## 構造

```
divergent_paths (A₁, A₂, A₃)
  → environmental_pressure / optimization
    → similar_solutions
      → convergence_point
        → unified_form or standard
```

## 発動条件

1. 共通の選択圧・最適化目標
2. 有限な解空間
3. 情報の相互参照・模倣
4. グローバル化・標準化圧力

## 失敗条件

- **経路依存性**: 過去の選択による分岐固定
- **多様な環境**: 異なる最適解の併存
- **意図的差別化**: 競争戦略としての多様性維持
- **文化的抵抗**: 同質化への反発

## 診断的応用：非収束シグナル

> **「効くものは収斂する。効かないものは増殖する」**

収束の**不在**は、市場・技術・制度の非効率を示すシグナルとして読める。

```
市場に類似製品が多数存在する
  → 収束が起きていない
    → どれも決定的に優位でないシグナル
      → 効果・価値の不在を示唆
```

### 適用例

- **脳トレアプリ**: 数百種類が乱立 → どれも認知症予防に決定的効果なし
- **ダイエットサプリ**: 棚に数十種類 → 本当に効くなら一種類に収束するはず
- **自己啓発書**: 毎年数千冊 → 決定的解法の不在

### 診断の限界

収束しない理由は「効かない」以外にもありうる：
- 好みの多様性（ファッション、音楽）
- ニッチ市場の併存
- 意図的差別化戦略

**健康・効果を謳う市場**では、非収束は特に強い警告シグナルとなる。
