# coordination（協調）

分散した主体が共通の行動パターンに収束する構造。

## 定義

中央の指令なしに、または限定的な調整のもとで、多数の独立した行為者が相互に整合的な行動をとり、協力の利益を実現する。

## 構造

```
independent_agents
  → shared_signals / focal_points
    → expectation_alignment
      → coordinated_action
        → mutual_benefit
```

## 発動条件

1. 共通利益の存在
2. コミュニケーション手段
3. 信頼または評判メカニズム
4. 焦点（シェリング・ポイント）

## 失敗条件（協調失敗）

- **利害対立**: ゼロサム状況
- **コミュニケーション断絶**: 期待の不一致
- **信頼欠如**: 裏切りの予期
- **複数均衡**: 収束点の不明確さ
