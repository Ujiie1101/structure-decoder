# cyclical（循環）

特定のパターンや状態が周期的に繰り返される構造。

## 定義

システムがある状態から別の状態へ移行し、やがて元の状態に回帰する。歴史、経済、政治において反復する力学を説明する。

## 構造

```
state_A
  → transition
    → state_B
      → further_transition
        → return_to_state_A (cycle completes)
```

## 発動条件

1. 内在的な反転力学
2. 極端への到達と揺り戻し
3. 記憶の忘却または世代交代
4. 構造的変化の欠如

## 失敗条件（循環脱出）

- **変革**: 根本的構造変化
- **学習**: 過去の教訓の記憶
- **外部介入**: サイクルを断ち切る力
- **消滅**: システム自体の終了

## 適用事例

| Layer | Entities |
|-------|----------|
| politics | regime-cycle, revolution-restoration, election-cycle |
| economics | business-cycle, boom-bust, kondratiev-wave |
| history | rise-and-fall, dynastic-cycle |
| philosophy | eternal-recurrence |

## 関連パターン

- **feedback**: 循環を駆動するメカニズム
- **transformation**: 循環からの脱出
- **erosion**: 循環内での漸進的変化
