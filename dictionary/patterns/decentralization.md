# decentralization（分散化）

権力・機能・資源が中央から周辺へ分散し、多中心的な構造が創発する。

## 定義

集中していた意思決定権、データ、処理能力などが複数のノードに分散される。単一障害点の除去、検閲耐性、参加者自律性の向上をもたらす一方、調整コストが増大する。

## 構造

```
centralized_system
  → distribution_pressure (inefficiency/risk/demand)
    → node_multiplication
      → coordination_protocol
        → emergent_resilience + coordination_cost
```

## 発動条件

1. 中央集権の非効率・リスク
2. 分散技術の利用可能性
3. 自律性・参加への需要
4. 調整プロトコルの確立

## 失敗条件

- **調整不全**: 分散ノード間の合意困難
- **再集中**: スケールメリットによる中央回帰
- **フリーライダー**: 貢献なき利用者の増加
- **複雑性**: 参加者の理解・運用困難

## 適用事例

| Layer | Entities |
|-------|----------|
| technology | blockchain, peer-to-peer |
| economics | cryptocurrency, cooperative |
| politics | federalism, local-autonomy |
| religion | congregationalism, lay-movement |
| art | independent-publishing, creator-economy |

## 関連パターン

- **network-effects**: 分散ネットワークの価値増大
- **coordination**: 分散システムの調整
- **emergence**: 分散から創発する秩序
