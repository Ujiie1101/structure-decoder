# 決定委託欲求 (Decision Delegation Desire)

## 構造

```
選択肢過多 → 決断疲れ → 「誰かに決めてほしい」欲求
                              ↓
              ┌─────────────────┼─────────────────┐
              ↓                 ↓                 ↓
         アルゴリズム        専門家/組織       強いリーダー
         （便利）            （責任回避）       （自由の放棄）
```

**同じ心理構造が、異なる領域で異なる結果を生む。**

## 定義

自由の重荷（決断疲れ）から逃れるため、決定権を外部に委ねようとする心理構造。日常の無害な委託（おまかせコース）から、民主主義の崩壊（権威主義への服従）まで、同一の心理メカニズムが異なるスケールで作動する。

## 発生条件

1. **選択肢の過多**: 決断すべき事項が多すぎる
2. **判断基準の不在**: 何を基準に選べばいいかわからない
3. **失敗への恐怖**: 自分で決めて失敗したくない
4. **疲労の蓄積**: 決断のエネルギーが枯渇している

## 4領域での発現

| 領域 | 委託先 | 表面的効果 | 潜在的リスク |
|------|--------|------------|--------------|
| **テック** | アルゴリズム | 選択の効率化 | 自己選択能力の減退 |
| **金融** | ロボアドバイザー | 専門知識の代替 | 同質化→システミックリスク |
| **経営** | コンサル | 意思決定の高速化 | 責任の外部化、思考停止 |
| **政治** | 強いリーダー | 混乱の収束 | 民主主義の崩壊 |

## フロムとの接続

エーリッヒ・フロム『自由からの逃走』の現代的適用：

- **近代以前**: 不自由だが安心（村、家業、宗教が決めてくれた）
- **近代**: 自由だが不安（自分で決めなければならない）
- **現代**: 自由の外注（アルゴリズム、専門家、リーダーに委ねる）

## Observable Proxies（観察可能な兆候）

- 「なんでもいい」の増加
- 「おすすめ」への依存
- 「専門家が言うなら」という思考停止
- 「強いリーダーが必要」という言説の増加
- 意思決定への忌避感

## 対抗戦略（逆・決定委託）

1. **小さな決断を自分でする**: 夕飯、映画、本を自分で選ぶ
2. **「おすすめ」をスキップする**: アルゴリズムを一度無視する
3. **「任せたい」衝動を観察する**: なぜそう感じるか内省する

## 関連パターン

- **MP-09 (Vacuum-Authoritarian Cycle)**: 混乱→秩序への渇望→権威主義
- **MP-02 (Threshold Transition)**: 蓄積→閾値突破→急激な変化
- **MP-17 (Accountability Fatigue Loop)**: 疲労による監視機能の低下
- **parental-deposit**: 親が子供への権威を国家に外注する家族領域の特殊形（Day62）
- **child-shield**: 委託された権威が「保護」の名で発動する形態（Day62）

## メタ観察

「決定委託欲求」は、MP-09の**心理的基盤**を説明する。

MP-09は「真空→権威主義」という社会的ダイナミクスを記述するが、**なぜ人々がそれを受け入れるのか**を説明しない。DDD-01はその心理的メカニズムを補完する。

```
DDD-01（心理） → MP-09（社会） → 体制変化（結果）
```
