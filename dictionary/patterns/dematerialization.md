# dematerialization（非物質化）

物質的形態から離れ、概念・プロセス・関係性そのものが作品として出現する構造。

## 定義

芸術作品が物理的オブジェクトとしての存在から離脱し、アイデア、行為、経験、関係性へと重心を移す。物質への依存が減少し、概念や過程が芸術の本体となる。

## 構造

```
material_artwork
  → conceptual_questioning (what is art?)
    → process_over_object
      → idea_as_medium
        → dematerialized_art (concept = work)
```

## 発動条件

1. 物質的形態の限界への自覚
2. 概念の優位性の主張
3. 観客の参与による意味生成
4. 市場・制度への批判的距離

## 失敗条件

- **再物質化**: 概念が再び商品として固定化
- **空虚化**: 概念だけが残り体験が失われる
- **エリート化**: 文脈理解を前提とした閉鎖性
