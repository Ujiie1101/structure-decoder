# democratization（民主化）

特権的なものへのアクセスが広く一般に開放される構造。

## 定義

かつては一部の専門家・富裕層・権力者のみが持っていた知識・技術・資源・権利へのアクセスが、技術革新・制度変革により広範な人々に開放されるプロセス。

## 構造

```
exclusive_access
  → barrier_reduction (cost, skill, permission)
    → mass_adoption
      → power_redistribution
        → new_equilibrium (capability widely distributed)
```

## 発動条件

1. アクセス障壁の技術的低下
2. コスト構造の変革
3. 既存権力の弱体化または合意
4. 新規参入者の存在

## 失敗条件

- **デジタル格差**: 新たな排除の創出
- **品質低下**: アクセス拡大による希釈化
- **再集中**: プラットフォームによる権力再集中
- **過負荷**: 参加能力を超える情報・選択肢

## 適用事例

| Layer | Entities |
|-------|----------|
| technology | smartphone, open-source, 3d-printing, ai-tools |
| economics | microfinance, crowdfunding, etf |
| history | printing-press, universal-suffrage |
| philosophy | enlightenment, public-sphere |

## 関連パターン

- **decentralization**: 民主化の一形態
- **disruption**: 民主化による既存秩序の破壊
- **emergence**: 民主化から新秩序が創発
