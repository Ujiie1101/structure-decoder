# destruction（破壊）

既存の形式・秩序の解体から、新たな創造の可能性が出現する構造。

## 定義

意図的または必然的な破壊行為を通じて、固定化した形式や制度が解体され、その空白から予期せぬ創造的可能性が生まれる。破壊は終点ではなく、新たな創発の起点となる。

## 構造

```
existing_form (ossified)
  → destructive_intervention
    → dissolution_of_structure
      → creative_void
        → new_emergence (unpredictable)
```

## 発動条件

1. 既存形式の硬直化・限界
2. 破壊の意志または外的圧力
3. 破壊後の空白を受容する余地
4. 新たな創造への潜在的エネルギー

## 失敗条件

- **虚無**: 破壊のみで創造が続かない
- **復古**: 破壊後に旧形式が復活
- **過剰破壊**: 再構築不能な損壊
- **表面的破壊**: 根本構造が温存される
