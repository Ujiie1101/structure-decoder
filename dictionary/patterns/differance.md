# differance（差延）

意味が永遠に遅延し、差異の連鎖の中でのみ仮構される構造。

## 定義

Jacques Derridaの造語。「差異」(difference)と「遅延」(deferral)を同時に示す。意味は直接的に現前せず、他の記号との差異の戯れの中で無限に延期され続ける。

## 構造

```
signifier_A
  → differs_from_signifier_B
    → meaning_deferred_to_B
      → B_differs_from_C
        → infinite_chain_of_references
          → meaning_never_fully_present
```

## 二重の運動

### 差異（Differ）

- 記号は他の記号との差異によってのみ意味を持つ
- Saussureの差異体系の徹底化
- A は「非B、非C、非D...」として規定

### 遅延（Defer）

- 意味は常に「次の記号」に先送りされる
- 完全な意味の到来は永遠に延期
- 最終的な所記（transcendental signified）は存在しない

## 発動条件

1. 言語的意味作用
2. 記号の体系性
3. 現前の形而上学への批判的視座
4. テクスト性への注目

## 関連パターン

- **trace**: 差延の痕跡としての意味
- **supplement**: 外部が内部を補完し脅かす

## 注意

「差延」は概念ではなく「概念」の可能性の条件。それ自体を定義しようとすると、定義もまた差延の運動に巻き込まれる。形而上学を内部から脱構築する戦略。
