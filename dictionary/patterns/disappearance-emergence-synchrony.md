# disappearance-emergence-synchrony（消失と誕生の同時性）

旧世代・旧体制・旧アイデンティティの**消失**と、新世代・新体制・新アイデンティティの**誕生**が、別々のイベントではなく**同一の時点で同時に起こる**構造。

## 定義

以下の三条件を満たす:

1. **同時性の必然**: 旧の消失が新の誕生の条件である（順序ではなく同時）
2. **系内の総量保存**: 消える存在の生気・主導権・物語性が新たに立つ存在に乗り移る
3. **徴候の長距離回収**: 冒頭で投下された徴候が同時点で完成する

## 構造

```
旧の消失イベント（母の死／創業者の引退／旧体制の崩壊）
  ↓（同時）
新の誕生イベント（娘の革命家宣言／後継者の覚醒／新体制の樹立）
  ↓
冒頭徴候の完成（蛇／予兆／伏線が意味を確定）
```

## 観察可能な現代具体例

- 創業者の死と後継者の宣言が同日に重なる事業承継
- 旧体制の崩壊と新政権樹立が同じ広場で起こる革命
- 親世代の終末ケアと子世代の妊娠が同時期に集中する家族
- 雇用主の倒産と元従業員の起業が同月に発表される労働市場
- 紙メディアの廃刊号とデジタル創刊号の同時発行

## 対抗装置

- 旧の消失と新の誕生を**意図的にずらす**（半年〜数年の空白を置く）
- 過渡期の制度化（暫定政府・暫定経営者の置き換え）
- 同時性の儀式化（公式な交代式典による意味の管理）
