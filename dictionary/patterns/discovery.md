# discovery（発見）

探求・観察・実験を通じて、既存の認識枠組みでは捉えられなかった真理・法則・対象が明らかになる構造。

## 定義

体系的または偶発的な探求の過程で、これまで認識されていなかった現実の側面が露わになり、知識体系の拡張・修正・革新をもたらす。

## 構造

```
inquiry_or_exploration
  → anomaly_encounter
    → recognition_struggle
      → conceptual_breakthrough
        → new_knowledge (expands understanding)
```

## 発動条件

1. 探求への動機と継続
2. 異常・矛盾への感受性
3. 既存枠組みへの批判的距離
4. 発見を受容する知的土壌

## 失敗条件

- **見過ごし**: 異常を無視・正常化
- **概念固着**: 既存枠組みへの固執
- **抑圧**: 発見の社会的・政治的封殺
- **孤立**: 発見を共有・検証する場の欠如

## 適用事例

| Layer | Entities |
|-------|----------|

## 関連パターン

- **serendipity**: 偶然からの発見
- **emergence**: 発見から新秩序が創発
- **paradigm-shift**: 発見がパラダイム転換を駆動
