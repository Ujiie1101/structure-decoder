# dispersion（分散）

集中した権力・資源・機能が複数の拠点に分散し、新たな秩序が出現する構造。

## 定義

中央に集中していた権力、資源、意思決定機能が、意図的または自然発生的に複数の主体・場所に分散する。分散により、単一障害点が解消され、レジリエンスが向上するが、調整コストも増大する。

## 構造

```
centralized_system
  → pressure (overload, risk, inefficiency)
    → distribution_mechanism
      → multiple_nodes
        → distributed_network (emergent resilience)
```

## 発動条件

1. 中央集中の非効率または脆弱性
2. 分散を可能にする技術・制度
3. 複数主体の自律性
4. 調整メカニズムの存在

## 失敗条件

- **断片化**: 調整不全による機能不全
- **再集中**: 分散の逆行
- **不均衡**: 分散の偏り
- **責任の拡散**: アカウンタビリティの喪失

## 適用事例
