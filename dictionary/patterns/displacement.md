# displacement（置換）

既存の要素・システム・秩序が、新たなものによって押しのけられる構造。

## 定義

技術・制度・慣行などが、より優れた代替物によって徐々にまたは急速に置き換えられる。完全置換（旧の消滅）と部分置換（共存・ニッチ化）がある。置換は多くの場合、非連続的イノベーションによって引き起こされる。

## 構造

```
incumbent (A)
  → challenger emerges (B)
    → performance crossover
      → adoption shift
        → A displaced (extinct / niche)
```

## 発動条件

1. 既存システムの限界・非効率
2. より優れた代替物の出現
3. スイッチングコストの許容範囲
4. 制度的障壁の不在

## 失敗条件

- **ロックイン**: 既存システムへの過剰投資
- **規制保護**: 既得権益による参入障壁
- **不完全代替**: 新旧の機能差
- **慣性**: 習慣・文化による抵抗

## 適用事例

## 関連パターン

- **substitution**: ほぼ同義だが置換は過程を強調
- **cascade**: 置換の連鎖的波及
- **collapse**: 被置換システムの崩壊
- **transformation**: 置換による変容
