# disruption（破壊的変化）

既存の秩序・システム・市場を根本から覆し、新たな均衡を生み出す構造。

## 定義

従来の延長線上にない革新が、既存のプレイヤー・構造・価値基準を無効化し、全く異なるルールに基づく新秩序を創出する。

## 構造

```
existing_order
  → peripheral_innovation
    → value_network_shift
      → incumbent_blindness
        → displacement (old order becomes obsolete)
```

## 発動条件

1. 既存秩序の硬直化
2. 辺境からの革新
3. 新しい価値基準の出現
4. 既存プレイヤーの適応遅延

## 失敗条件

- **吸収**: 既存勢力による革新の取り込み
- **過早**: 市場・社会の受容準備不足
- **孤立**: 新価値ネットワーク形成の失敗
- **模倣**: 差別化要因の消失

## 適用事例

| Layer | Entities |
|-------|----------|
| art | avant-garde, impressionism, punk-rock |
| technology | internet, smartphone, digital-photography |
| economics | sharing-economy, cryptocurrency |
| history | printing-revolution, industrial-revolution |

## 関連パターン

- **transformation**: 破壊的変化が変革を駆動
- **substitution**: 破壊後の置換
- **emergence**: 破壊から新秩序が創発
