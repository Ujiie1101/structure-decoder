# dissolution（溶解）

境界の融解から、区別不能な新たな統一が出現する構造。

## 定義

明確に分離されていた要素間の境界が徐々に溶解し、個別性を保ちながらも区別不能な状態へと移行する。溶解は消失ではなく、より高次の統合形態の創発である。

## 構造

```
distinct_boundaries
  → gradual_interpenetration
    → boundary_blur
      → mutual_absorption
        → unified_emergence (elements preserved yet indistinguishable)
```

## 発動条件

1. 接触と相互浸透の機会
2. 境界維持の緩和
3. 共通基盤の発見
4. 時間的持続

## 失敗条件

- **固着**: 境界の再強化
- **消滅**: 一方が完全に吸収される
- **表層融合**: 深層での分離が残存
- **急激溶解**: 統合なき混沌
