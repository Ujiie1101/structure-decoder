# diversification（多様化）

統一された様式から多元的な表現へと分岐し、豊かな表現生態系が出現する構造。

## 定義

支配的な単一様式・規範・センターが弱まり、多数の異なるスタイル・声・視点が併存・競合・交流する状態へ移行する。多様性そのものが創造性の源泉となる。

## 構造

```
unified_norm (dominant_style)
  → centrifugal_forces
    → multiple_trajectories
      → coexistence
        → emergent_ecosystem (plurality)
```

## 発動条件

1. 支配的規範の弱体化
2. 周縁からの発信
3. 多元的価値観の受容
4. 交流・混交の機会

## 失敗条件

- **断片化**: 対話なき孤立
- **相対主義**: 基準の消失
- **混乱**: 過剰な選択肢
- **階層復活**: 新たな中心の形成

## 適用事例

| Layer | Entities |
|-------|----------|
| art | postmodernism, pluralism, global-art |
| music | genre-proliferation, world-music |
| media | fragmentation, niche-market |
| society | multiculturalism, identity-politics |
| biology | speciation, biodiversity |

## 関連パターン

- **emergence**: 多様性から創造性が創発
- **fragmentation**: 分散化との関連
- **decentralization**: 中心の解体
