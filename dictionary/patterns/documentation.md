# documentation（ドキュメンテーション）

現実の記録・証言の蓄積から、意図を超えた歴史的・社会的意味が創発する構造。

## 定義

個々の記録行為の集積が、記録者の意図を超えた集合的証言・歴史的文書として新たな意味を獲得する。断片の総体が、断片の総和以上の真実を語り出す。

## 構造

```
individual_records
  → accumulation
    → cross_reference
      → pattern_revelation
        → emergent_testimony (exceeds original intent)
```

## 発動条件

1. 記録行為の持続性
2. 記録間の相互参照可能性
3. 批判的検証のプロセス
4. アクセス可能なアーカイブ

## 失敗条件

- **断片化**: 記録が散逸し全体像が構成不能
- **改竄**: 意図的な記録の歪曲
- **忘却**: 記録へのアクセス喪失
- **過剰**: 情報過多による意味の埋没

## 適用事例

| Layer | Entities |
|-------|----------|
| art | documentary-photography, oral-history, archive-art |
| history | holocaust-testimony, colonial-records |
| technology | blockchain, version-control |
| politics | truth-commission, whistleblowing |
| philosophy | phenomenology, hermeneutics |
| religion | scripture, hagiography |

## 関連パターン

- **emergence**: 記録の集積から意味が創発
- **preservation**: 記録の保存が前提
- **trace**: 記録が痕跡として機能
