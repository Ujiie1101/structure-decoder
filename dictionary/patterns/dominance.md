# dominance（支配）

一方が他方に対して圧倒的優位を獲得し、その状態を維持・強化する構造。

## 定義

市場支配、技術支配、文化支配など、特定の領域において競合を排除または従属させる状態。ネットワーク効果、規模の経済、標準化などが支配を強化。支配は反発と代替の誘因にもなる。

## 構造

```
competitive_advantage
  → market_capture
    → reinforcement_mechanism
      → competitor_exclusion
        → sustained_dominance / disruption
```

## 発動条件

1. 初期の競争優位
2. 正のフィードバック機構
3. 参入障壁の構築
4. 代替品の不在

## 失敗条件

- **破壊的イノベーション**: 新パラダイムの出現
- **規制介入**: 独占禁止・分割
- **過信**: イノベーション停滞
- **反発連合**: 対抗勢力の結集
