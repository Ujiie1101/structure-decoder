# east-west-division（東西分裂）

統一体が地理的・文化的・政治的断層線に沿って二極化する構造。

## 定義

単一の文明・帝国・システムが、内部の差異の蓄積と外部圧力により、東西（または南北）の二つの極に分裂する。分裂は偶発ではなく、構造的差異から創発する。

## 構造

```
unified_system
  → regional_differentiation
    → administrative_divergence
      → cultural_drift
        → mutual_incomprehension
          → formal_split (emergent bipolarity)
```

## 発動条件

1. 地理的距離による統治困難
2. 文化・言語・宗教の差異蓄積
3. 外部圧力の非対称性
4. エリート層の利害分化

## 失敗条件

- **強力な中央権力**: 分裂圧力の抑制
- **共通の脅威**: 統合を維持する外敵
- **インフラ**: 物理的・情報的結合
- **統合イデオロギー**: 共有される帰属意識
