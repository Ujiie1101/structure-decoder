# emergence（創発）

部分の相互作用から、部分の総和では説明できない新しい性質・秩序が生まれる構造。

## 定義

個々の要素が相互作用することで、要素単体の性質からは予測できない新しいパターン、構造、機能が自発的に出現する。

## 構造

```
individual_elements
  → local_interactions
    → feedback_loops
      → pattern_formation
        → emergent_property (irreducible to parts)
```

## 発動条件

1. 多数の相互作用する要素
2. 非線形フィードバック
3. 十分な複雑性
4. 自己組織化のメカニズム

## 失敗条件

- **孤立**: 要素間の相互作用が不足
- **過剰制御**: トップダウン介入が創発を阻害
- **均一性**: 多様性不足でパターンが形成されない

## 適用事例

| Layer | Entities |
|-------|----------|
| philosophy | consciousness, emergent-properties |
| economics | market-order, price-system |
| technology | internet, open-source |
| art | jazz-improvisation, collective-creativity |
| history | civilization, culture |
| religion | religious-community, ritual-practice |

## 関連パターン

- **coordination**: 創発の一形態
- **cascade**: 創発が連鎖する
- **substitution**: 創発した新秩序が旧秩序を置換

**[E-01] Invisible Hand Coordination** — 自己利益追求から社会的秩序が創発
