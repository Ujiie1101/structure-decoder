# epidemiological-amnesia（疫学的忘却）

公衆衛生の成功が、その成功を可能にした集団的恐怖記憶・専門知の権威・防御行動の正統性を侵食し、結果として撲滅したはずの脅威が制度の隙間から戻ってくる構造。

## 定義

ある脅威に対する公衆衛生介入が成功すると、(1) 体験者世代が脅威の深刻さを継承する前に退場し、(2) 個人は「もう必要ない」と判断し、(3) 専門知の権威が政治・SNS的正統性に置き換えられ、(4) 「自然」「身体の声」型の物語が統計的説明を駆逐する。4つが揃った瞬間、脅威は**誰の意図でもなく**戻ってくる。

memory-erasure（意図的削除）でも self-undermining-success（戦略の自己破壊）でもない第三領域：**集団記憶の自然減衰と権威源泉移動の交差点**。

## 公式

```
epidemiological-amnesia
  = success-blindness        （成功体験が脅威認知を曇らせる）
  × generational-amnesia     （体験者世代の退場による警戒コスト再払い不能）
  × legitimacy-source-shift  （専門知から SNS/政治への正統性移動）
  × narrative-pressure       （「自然免疫」物語が統計を駆逐）
```

4つすべて揃ったとき発動。1つでも欠ければ脅威は戻らない。

## 発動条件

1. **撲滅レベルの成功**: 体感的に「もう脅威ではない」状態が60-80年継続
2. **体験者世代の退場**: 病気の身体的恐怖を直接知る世代が物理的に減少
3. **専門知の権威源泉移動**: 制度的医学から「自分で調べた人」「自然派」「政治家」へ
4. **物語化可能な代替説明**: 「自然」「身体の知恵」「製薬陰謀」が登場

4つ揃わない場合の停止形態：
- success-blindness のみ欠如 → 親世代の警戒で個人レベル防御維持
- generational-amnesia のみ欠如 → 専門家コミュニティが補修
- legitimacy-source-shift のみ欠如 → 制度が独走で政治的圧力を吸収
- narrative-pressure のみ欠如 → 不安はあるが対抗物語がないため接種継続

## 失敗条件（amnesia が回避される条件）

### 個別原子への介入

- **success-blindness**: 親世代の体験を curriculum に組み込む（病院訪問、博物館展示）
- **generational-amnesia**: 周期的小規模再発の可視化（10-20年サイクルで記憶を更新）
- **legitimacy-source-shift**: 専門知の正統性を新源泉（SNS）の文法で再翻訳する
- **narrative-pressure**: 統計を体験談化する hybrid narrative の発明

### 組み合わせ介入（最強）

**generational-amnesia × narrative-pressure への同時介入** = 体験者の証言を物語形式で制度化。4 atom のうち2 atom を同時に弱める。「体験者がいなくなる前にその物語を捕まえる」。

## 判別法

1. 体験者世代の発言が「過剰反応」「不安症」と再フレーミングされている
2. 撲滅成功の数字（撲滅前の死者数、入院数）が日常から消えている
3. 「自然」「身体」「個人の選択」が議論の正統な軸として登場する
4. 専門機関の発表より個人インフルエンサーの動画の方が拡散する

4つ揃うと epidemiological-amnesia が作動している。

| 時代 | 事例 | 撲滅期 | 再発 | 再発時の amnesia 比 |
|------|------|--------|------|------|
| 麻疹 | 米国 SC州（2025-2026） | 2000年撲滅宣言 | 1000人感染、35年で最悪 | 撲滅から26年で再発 |
| 麻疹 | 欧州（2017-2019, WHO） | 2000年代に低水準 | 8万人超感染 | WHO「ワクチン忌避が要因」と公式声明 |
| 百日咳 | 米国カリフォルニア（2010, 2014） | 1990年代撲滅近接 | 9000人超感染、乳児10人死亡 | カリフォルニア州が宗教免除規定を強化 |
| はしか | 英国（1998-2010s, Wakefield効果） | 1980年代に高接種率 | MMR接種率80%まで低下 | Wakefield 1998 Lancet論文（2010撤回） |
| ジフテリア | 旧ソ連崩壊後（1990s） | 撲滅近接 | 15万人感染、5000人死亡 | 制度崩壊との結合 |
| インフル | 1957/1968記憶忘却 → 2020年COVID初動 | 1957/1968で約300万人死亡 | COVID時点で警戒記憶ゼロ | 60年経過で完全脱落 |

| 金融 | 1929大恐慌記憶 → 2008サブプライム | 80年経過 | 同型レバレッジ問題 | 規制緩和の正統化 |

## 核心洞察

> **撲滅とは、忘却の許可証である。**
>
> 公衆衛生の成功は、その成功を覚えていることを困難にする。なぜなら、覚えるべき脅威が日常から消えているからである。記憶は脅威の不在によって干上がる。
>
> success-blindness は認知科学的（個人の脳の問題）、generational-amnesia は社会学的（世代交代の問題）、legitimacy-source-shift は政治哲学的（権威の問題）、narrative-pressure は言語学的（伝達の問題）。4つの異なる学問領域の時間軸が、撲滅から60-80年経過した社会で**収束**する。

## メタ観察

本パターンは **Knowledge OS の atoms層**から派生した第二の登録パターン（algorithmic-abdication に続く）。

事象観察起点ではなく、(1) memory-erasure と self-undermining-success の隙間に存在する未命名構造への気づき、(2) 既存3 atom + 新規1 atom の論理積による演繹、(3) 既存事象（米国麻疹、欧州麻疹、百日咳、Wakefield効果）への適合確認、という順序で登録された。

## 反証条件

以下が観察された場合、本パターンの定義は再検討が必要：

1. 4つの atom のうち1つでも欠けている社会で、撲滅された脅威が大規模再発する
2. 全 atom が揃っているが脅威が戻らない（=他の防御要因が強い）
3. 「忘却」を経由せずに、純粋に意図的破壊（memory-erasure 単独）で脅威が戻る

観察期間：2026年Q3〜2027年Q4。検証対象：米国麻疹流行の地理的拡散、欧州各国の MMR 接種率動向、日本での風疹/麻疹の動向、COVID-19 警戒記憶の経過観察。
