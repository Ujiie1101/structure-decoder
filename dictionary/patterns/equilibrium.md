# equilibrium（均衡）

対立する力が相互に打ち消し合い、安定状態が自発的に形成される構造。

## 定義

複数の競合する力・利害・アクターが相互作用の結果として、いずれも一方的優位を達成できない安定点に収束する。動的均衡（絶えず調整される）と静的均衡（固定化）の二類型がある。

## 構造

```
competing_forces (A ↔ B)
  → mutual_adjustment
    → balance_point
      → stable_state (temporary or persistent)
        → [perturbation → re-equilibration]
```

## 発動条件

1. 複数の対等な力の存在
2. 相互牽制メカニズム
3. 情報共有または反応速度の均等性
4. コミットメント装置（離脱コスト）

## 失敗条件

- **非対称性**: 一方の力が圧倒的に優勢
- **外部衝撃**: 均衡を破壊する外生的ショック
- **情報非対称**: 一方のみが状況を把握
- **コミットメント崩壊**: 約束の実行可能性喪失
