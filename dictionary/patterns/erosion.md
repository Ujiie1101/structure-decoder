# erosion（侵食）

システムの基盤が徐々に弱体化する構造。

## 定義

システムを支える制度・規範・資源が、目立たない形で漸進的に削られ、表面上は安定を維持しながら内部では脆弱化が進行する。

## 構造

```
stable_system
  → incremental_pressures (often invisible)
    → gradual_degradation
      → capacity_decline
        → vulnerability_increase
          → eventual_threshold_breach
```

## 発動条件

1. 持続的な負荷または圧力
2. 修復・補充の不足
3. 劣化の不可視性（認知されにくい）
4. 時間の経過

## 失敗条件（侵食停止）

- **早期発見**: 劣化の認識と対処
- **強化**: 基盤の補強
- **圧力除去**: 侵食原因の解消
- **適応**: システムの進化

## 適用事例

| Layer | Entities |
|-------|----------|
| politics | institutional-decay, norm-erosion |
| economics | inflation, capital-depletion |
| history | imperial-overextension |

## 関連パターン

- **collapse**: 侵食の終点としての崩壊
- **substitution**: 侵食されたシステムの置換
- **fragmentation**: 侵食による分断化
