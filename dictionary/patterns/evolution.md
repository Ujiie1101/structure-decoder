# evolution（進化）

変異・選択・継承のサイクルを通じて、システムが漸進的に適応・変化する構造。

## 定義

ランダムな変異（イノベーション）が生じ、環境による選択を経て、有利な特性が次世代に継承される。ダーウィン的進化を超えて、技術・制度・文化・思想の変化にも適用される普遍的パターン。

## 構造

```
variation (random/intentional)
  → selection (environmental fit)
    → retention (inheritance)
      → [iteration]
        → cumulative_adaptation
```

## 発動条件

1. 変異を生む多様性源
2. 選択圧（環境・競争）
3. 継承・複製メカニズム
4. 十分な時間・世代数

## 失敗条件

- **変異不足**: イノベーション枯渇
- **選択弱化**: 淘汰圧の不在（保護・補助）
- **継承断絶**: 知識・形質の伝達失敗
- **環境急変**: 適応速度を超える変化

## 関連パターン

- **emergence**: 進化から複雑性が創発
- **transformation**: 進化の結果としての変容
- **substitution**: 進化的置換
- **cascade**: 進化的変化の波及
