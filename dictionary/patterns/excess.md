# excess（過剰）

規範・節度を超えた過剰な表現から、新しい美的可能性が開かれる構造。

## 定義

量・強度・装飾・感情において既存の基準を逸脱し、過剰さそのものを美的価値とする。過剰は禁忌を破り、新たな感覚・経験の領域を開拓する。崇高・グロテスク・バロック的美学の源泉。

## 構造

```
established_norms (moderation, decorum)
  → transgression
    → accumulation_and_intensification
      → overflow
        → new_aesthetic_territory
          → excess (sublime_or_grotesque)
```

## 発動条件

1. 既存規範への反発
2. 過剰への衝動
3. 限界突破の意志
4. 過剰を受容する文化的文脈

## 失敗条件

- **混乱**: 意味の崩壊
- **悪趣味**: 過剰と下品の混同
- **疲弊**: 感覚の摩耗
- **自己模倣**: 過剰の定型化

## 適用事例

| Layer | Entities |

## 関連パターン

- **emergence**: 過剰から新しい美学が創発
- **transformation**: 規範からの逸脱による変容
- **escalation**: 過剰のエスカレーション
