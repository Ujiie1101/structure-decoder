# exchange（交換）

異なる主体間での価値の相互授受から、関係・秩序・意味が創発する構造。

## 定義

二者以上の主体が、物質的または象徴的な価値を相互に授受することで、単なる取引を超えた関係性、社会的紐帯、共有された意味世界が生成される。

## 構造

```
distinct_parties
  → mutual_offering
    → reciprocal_receiving
      → obligation_creation
        → relationship_emergence (emergent bond)
```

## 発動条件

1. 交換可能な価値の所有
2. 相互性の期待
3. 信頼または強制のメカニズム
4. 継続的関係への動機

## 失敗条件

- **一方的収奪**: 相互性の欠如
- **自給自足**: 交換の必要性消失
- **信頼崩壊**: 履行への期待消失
- **等価破綻**: 交換価値の不均衡

- **reciprocity**: 交換の規範的側面
- **trust**: 交換を可能にする前提
- **coordination**: 交換による秩序形成
