# exit-blockade（逃走阻害）

離脱コストが高すぎて「逃げる」選択肢が実質的に封じられている構造。

## 定義

組織・関係・状況からの離脱が、経済的・社会的・心理的コストにより事実上不可能となっている状態。「逃げられない」環境では、本来ならば回避可能な問題が深刻化し、代替的な防衛手段（告発、ハラスメント申告、内部抗争）が増殖する。

## 構造

```
high_exit_cost
  → exit_option_blocked
    → conflict_internalization
      → escalation (no escape valve)
        → surrogate_defense_mechanisms
          → systemic_dysfunction
```

## 発動条件

1. **経済的ロックイン** — 退職金後払い、住宅ローン、年功序列
2. **社会的ロックイン** — 「辞める＝落伍者」という偏見、レールからの逸脱恐怖
3. **心理的ロックイン** — サンクコスト、アイデンティティの依存
4. **情報的ロックイン** — 外部選択肢の不可視化

## 失敗条件（ロック解除）

- **流動性の向上**: 転職市場の活性化、中途採用の一般化
- **コスト低減**: 退職金ポータビリティ、リスキリング支援
- **偏見の解消**: 「転職は正常」という社会規範
- **外部選択肢の可視化**: 他の働き方の具体的事例

## 適用事例

| Layer | Entities |
|-------|----------|
| labor | 日本型雇用（終身雇用・年功序列・新卒一括採用）、ブラック企業 |
| family | DV関係、経済的依存、「家を出られない」 |
| religion | カルト組織、脱会コストの意図的設計 |
| economics | vendor-lock-in、switching-cost戦略 |
| politics | 独裁体制、亡命の困難さ |

## 代替的防衛手段の発生

逃げられない環境では、以下の代替手段が増殖する：

| 防衛手段 | 説明 |
|----------|------|
| ハラスメント申告 | 攻撃者を「加害者」認定して排除 |
| 内部告発 | 組織の問題を外部に暴露 |
| 消極的抵抗 | サボタージュ、quiet quitting |
| 心身症状 | 適応障害、うつ病（身体が「逃走」する） |

## 日本的特殊性

日本の「逃げられない」構造は複合的：

1. **告解システムの不在** — 罪を処理する公式回路がない
2. **穢れの論理** — 「排除」以外の解決法を持たない文化
3. **法的解決の回避** — 訴訟を避ける傾向
4. **集団主義** — 「和を乱す」ことへの恐怖

これらが重なり、「〇〇ハラ」が代替的裁判システムとして機能する。

- **chilling-effect**: 逃げられない環境での萎縮

## 核心洞察

> **「〇〇ハラ」は病気ではなく症状。病気は「逃げられない社会構造」と「罪を処理できない文化構造」**
