# expertise-blindness（経験盲点）

経験が増えるほど、特定のものが見えなくなる構造。

## 定義

専門性や経験の蓄積が、かえって新しい可能性や異なる視点への認知を妨げる現象。「20年の経験は、20年分の盲点でもある」。

## 構造

```
experience_accumulation
  → pattern_recognition_efficiency
    → automatic_judgment (bypassing reflection)
      → blind_spots (invisible assumptions)
        → inability_to_see_alternatives
```

## 発動条件

1. 長期間の同一領域での経験
2. 成功体験の蓄積
3. 「自分は客観的だ」という自己認識
4. 反省・検証プロセスの欠如

## 失敗条件（盲点を防ぐ）

- **エポケー（判断停止）**: フッサール的な前提の括弧入れ
- **初心者の質問**: 「なぜ？」を問い続ける習慣
- **異分野との接触**: 前提を共有しない人との対話
- **定期的な前提監査**: 自分の前提を書き出して検証
