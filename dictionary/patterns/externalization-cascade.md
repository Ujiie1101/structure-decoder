# Externalization Cascade Pattern（外部化の連鎖）

## 定義
社会・産業・組織・個人の各階層が、**上層がコストを下層に外部化する連鎖構造**を成している現象。各層では「当然の手段」として実行されるが、**最下層の当事者が可視化の力を持たないため、全体としては構造の歪みが隠蔽される**。

## 公式
```
経営層 → 従業員へ外部化（時間）
  ↓
会社 → NPO/関連団体へ外部化（善意・感情労働）
  ↓
業界 → 国の制度・補助金へ外部化（公益性の担保）
  ↓
社会 → 弱者（高齢者・相続人・子ども）へ外部化（不安・コスト）
  ↓
最終的なコストは、可視化する力を持たない者に堆積する
```

## メカニズム

### 1. 各層が独立に「合理的」に見える
経営層から見れば「従業員の工夫」は当然。会社から見ればNPO連携は「社会貢献」。業界から見れば国制度の活用は「公益」。社会から見れば高齢者への相談対応は「サービス」。**各層だけを見ると問題は見えない**。

### 2. 可視化の非対称
最上層ほどデータ・会計・発言力・メディア接続を持ち、コストと歪みを言語化できる。最下層ほど可視化手段を持たず、「違和感」までは感じても構造として言語化できない。**認識の勾配がコストの勾配と逆になる**。

### 3. 連鎖の自己強化
外部化が一つの層で機能すると、その層は「解決できた」と認識し、同じ仕組みを次の課題にも適用する。**外部化が組織文化として定着**し、代替手段が検討されなくなる。

### 4. 閾値崩壊時の不可逆性
最下層が崩れると（退職者殺到、NPO瓦解、高齢者の離反）、上位層は一気にリードを失う。**延命装置が最下層にあるため、そこが崩れると全体が崩壊する**。

## 成功条件（議論として提示する上での）
- 各層の具体事例を**最低4層**揃える（個人→組織→業界→社会）
- 各層で「誰が何を外部化しているか」を1行で言語化できる
- 最下層の可視化困難性を示すデータ（時間統計・感情労働研究等）

## 失敗パターン
- 「悪いのは誰か」を特定しようとする → 外部化の連鎖はシステム的現象、犯人はいない
- 1層だけで論じる → 連鎖の全体像が見えない
- 全ての組織に当てはまると過剰一般化する → 具体性が失われる

## 判別法
1. 「合理的なコスト削減」が、常に同じ方向（下・外）に流れているか？
2. 最下層の当事者が「違和感はあるが言語化できない」状態にあるか？
3. 各層の担当者が「自分は正しいことをしている」と信じているか？

3つYesなら外部化の連鎖が作動している。

## 事例

### Day 55「外部化の連鎖」
```
個人層: 会社員Kの時間（延命費）
  ↓
会社層: 不動産会社（IT不在、業界慣習維持）
  ↓
業界層: 不動産業（DX平均2.3、業界最下位）
  ↓
社会層: 高齢者・相続人（相続・空家不安の市場化）
```

### 他分野への応用
- 教育層: 教員の時間 → 学校のICT形式化 → 教育産業の資格化 → 保護者の不安の市場化
- 医療層: 看護師の時間 → 病院の電子化コスト転嫁 → 医療業界のアウトカム外注 → 患者の自己責任化
- 環境層: 現場作業員の労働 → 下請け企業の利益圧縮 → 業界のグリーンウォッシュ → 消費者の罪悪感商品化

## 関連概念
- **下位層パターン**: time-sacrifice-life-extension（個人層）
- **中位層パターン**: goodwill-infrastructuralization（組織・業界層）
- **上位層の駆動因**: life-extending-investment（延命投資）
- **最終帰結**: delayed-exponential-collapse（時間差崩壊）
- **哲学的親族**:
  - Marx「労働の疎外」
  - Baudrillard「象徴的交換」
  - Zygmunt Bauman「液状化する社会」
  - Ulrich Beck「リスク社会」

## 配置場所（記事での使い方）
- 記事の**終盤（80%地点以降）**で全体像を束ねる命名として使う
- 4階層ズームアウト図を図解すると読者の理解が一気に進む
- 各層の「外部化の通貨」（時間／善意／公益性／不安）を明示する
- Bauman等の外部権威引用と組み合わせる

## 注意点
- **個別企業・団体の告発にしない**。構造として論じること
- 「全員が犠牲者」で終わらせない。**誰が最も戻り道を失うか**まで指摘する（Day 55では大手企業が「最も脆弱」と示した）
- 対策案を過度に示さない。意図的不完全を維持
- 連鎖を固定視しない。各層に小さな選択肢がある（個人の構造投資、業界の自己革新等）
