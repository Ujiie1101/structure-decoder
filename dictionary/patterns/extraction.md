# extraction（抽出）

資源・価値・労働力が特定の主体により一方的に収奪される構造。

## 定義

ある集団・地域・システムから、別の主体が資源や価値を移転させる。抽出される側には十分な対価が支払われず、長期的な枯渇や従属関係が生じる。

## 構造

```
resource_holder
  → power_asymmetry
    → extraction_mechanism (force/contract/dependency)
      → value_transfer
        → depletion + accumulation
```

## 発動条件

1. 価値ある資源の存在
2. 抽出者と被抽出者の力の非対称
3. 抽出を正当化する論理・制度
4. 抵抗の困難さ

## 失敗条件

- **枯渇**: 抽出対象の消尽
- **抵抗**: 被抽出者の反乱・離脱
- **代替**: 抽出対象の価値喪失
- **規制**: 抽出を制限する法・規範
