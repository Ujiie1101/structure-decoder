# failure（失敗）

意図からの逸脱から、予期せぬ創造的可能性が出現する構造。

## 定義

計画や意図が達成されない状況において、その逸脱自体が新たな発見、美学、または価値を創発する。失敗は欠如ではなく、意図を超えた創造の契機となる。

## 構造

```
intention
  → execution_attempt
    → deviation_from_plan
      → unexpected_result
        → creative_emergence (value from failure)
```

## 発動条件

1. 明確な意図の存在
2. 実行と結果の不一致
3. 逸脱を受容する姿勢
4. 意味付けの再解釈

## 失敗条件

- **否認**: 失敗の拒絶と隠蔽
- **反復**: 同じ失敗の繰り返し
- **破滅**: 回復不能な損害
- **諦念**: 創造への転化なき受容

## 適用事例

| Layer | Entities |
|-------|----------|
| art | glitch-art, wabi-sabi, happy-accident |
| science | serendipity, penicillin, post-it |
| technology | bug-as-feature, pivot |
| philosophy | negative-capability, apophatic |
| business | fail-fast, iteration |

## 関連パターン

- **emergence**: 失敗から予期せぬ価値が創発
- **transformation**: 失敗を通じた変容
- **inversion**: 失敗と成功の反転
