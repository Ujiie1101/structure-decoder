# foundation（基盤）

上位構造を支え、その存在を可能にする下部構造。

## 定義

目に見える成果・システム・活動の背後にある、それを支える基礎的条件。インフラ、前提、土台として機能し、その上に多様な構造が構築される。

## 構造

```
base_infrastructure
  → enabling_conditions
    → supported_activities
      → higher_structures
        → visible_outcomes (dependent on foundation)
```

## 発動条件

1. 基盤の安定性
2. 上位構造との接続
3. 維持・更新のメカニズム
4. 容量・拡張性

## 失敗条件

- **崩壊**: 基盤自体の破壊
- **劣化**: 漸進的弱体化
- **不適合**: 上位構造との不整合
- **過負荷**: 容量超過

## 適用事例

| Layer | Entities |
|-------|----------|
| technology | infrastructure, protocol, operating-system, internet |
| economics | financial-system, legal-framework, property-rights |
| philosophy | axiom, presupposition, ontology |
| politics | constitution, rule-of-law |

## 関連パターン

- **emergence**: 基盤上に新秩序が創発
- **coordination**: 基盤が協調を可能にする
- **collapse**: 基盤崩壊による上部構造の崩壊

## メカニズムID

**[E-01] Invisible Hand Coordination** — 基盤上の分散的活動から秩序が創発
