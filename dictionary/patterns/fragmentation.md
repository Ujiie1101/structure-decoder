# fragmentation（分断）

統一されたシステムが複数の部分に分裂する構造。

## 定義

かつて統合されていたシステム、コミュニティ、領域が、内部対立・外部圧力・求心力の喪失により、複数の独立した単位に分裂する。

## 構造

```
unified_system
  → centrifugal_pressures
    → internal_divisions
      → coordination_failure
        → separation
          → multiple_fragments
```

## 発動条件

1. 求心力の弱体化
2. サブユニット間の利害対立
3. 遠心力の増大
4. 分離コストの低下

## 失敗条件（統合維持）

- **強い求心力**: 共通目標・脅威・利益
- **統合メカニズム**: 制度的結合
- **アイデンティティ**: 共有される帰属意識
- **高い分離コスト**: 独立の不利益

## 適用事例

| Layer | Entities |
|-------|----------|
| politics | balkanization, secession |
| religion | schism, sectarianism |
| history | empire-dissolution |
| economics | market-fragmentation |
| technology | platform-fragmentation |
| art | movement-splintering |

## 関連パターン

- **erosion**: 分断に先立つ結合力の弱体化
- **collapse**: 分断の極端な形態
- **emergence**: 断片からの新秩序創発
