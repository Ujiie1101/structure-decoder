# Goodhart's Law（グッドハートの法則）

指標が目標になった瞬間、その指標は指標としての機能を失う。

## 定義

> "When a measure becomes a target, it ceases to be a good measure."
> — Charles Goodhart, 1975

もともとは中央銀行の金融政策について述べたもの。「マネーサプライを政策目標にすると、マネーサプライ自体が歪む」。

しかし、この法則は**あらゆる測定**に当てはまる。

## 構造

```
指標Xを設定
  → Xを達成すると報酬
    → Xを最大化する最も簡単な方法を探す
      → 最も簡単な方法 ≠ 本来の目的
        → Xは達成されるが、本来の目的は達成されない
          → Xは指標として機能しなくなる
```

## 発動条件

1. **測定可能な指標**: 数値化できる代理変数が存在
2. **インセンティブ構造**: 指標達成で報酬が得られる
3. **抜け道の存在**: 本来の目的を達成せずに指標を操作できる
4. **監視の不在**: 「どうやって」が問われない

## 適用事例

| 領域 | 指標 | 歪み |
|------|------|------|
| 教育 | テストの点数 | 「テスト対策」に変質、本質的学力は不問 |
| 経済政策 | GDP | 環境破壊が「成長」に計上される |
| 学術 | 論文引用数 | 自己引用、引用カルテルが増加 |
| AI評価 | ベンチマークスコア | タスクを解かずにスコアを上げる |
| 企業KPI | 売上目標 | 値引き販売で数字は達成、利益は減少 |
| 警察 | 検挙件数 | 軽微な犯罪を重点的に取り締まり |
| 医療 | 手術成功率 | 難しい手術を避ける |

## 問い置換トリックとの関係

グッドハートの法則は「**問い置換トリック**」の帰結として発生する。

1. **問い置換**: 難しい問い→測定可能な問いに置換
2. **目標化**: 置換後の問いが「目標」になる
3. **グッドハート発動**: 目標を達成する最短経路が選ばれる
4. **乖離**: 元の問いと置換後の問いの乖離が拡大

| 元の問い | 置換後（指標） | グッドハート的帰結 |
|---------|---------------|-------------------|
| 機械は思考できるか？ | 30%を騙せるか？ | 騙すことに最適化、思考は不問 |
| 学生は賢いか？ | テスト点数 | テスト対策に最適化、知性は不問 |
| AIは有能か？ | ベンチマーク% | スコア操作に最適化、能力は不問 |

## 2×2マトリックス: 指標と目的

|  | **目的達成** | **目的未達成** |
|---|---|---|
| **指標達成** | 理想状態 | **グッドハート発動** |
| **指標未達成** | 測定ミス | 失敗 |

右上の象限が問題。指標は達成されているが、本来の目的は達成されていない。

## 防御策

1. **複数指標**: 単一指標に依存しない
2. **プロセス監視**: 「何を」だけでなく「どうやって」も問う
3. **定期的見直し**: 指標の有効性を定期的に検証
4. **質的評価**: 数値化できないものも評価に含める

## 関連パターン

- **問い置換トリック**: 難しい問いを測定可能な問いに置換
- **代理指標の独り歩き**: 代理変数が本質と乖離
- **最適化の罠**: 局所最適化が全体最適を阻害
- **評価者操作**: 能力向上より評価操作が効率的になる

## 歴史的文脈

1975年、イギリスの経済学者チャールズ・グッドハートが中央銀行の金融政策について指摘。

その後、社会科学全般に拡張され、「キャンベルの法則」（Donald T. Campbell, 1979）として社会政策にも適用。

> "The more any quantitative social indicator is used for social decision-making, the more subject it will be to corruption pressures and the more apt it will be to distort and corrupt the social processes it is intended to monitor."
