# governance（統治）

集団の意思決定・資源配分・紛争解決のための秩序が形成・維持される構造。

## 定義

複数の主体が共存する場で、調整・決定・執行のメカニズムが発展し、集合的な行動を可能にする秩序が構築される。統治は強制だけでなく、合意・慣習・インセンティブを通じても実現される。

## 構造

```
collective_action_problem
  → coordination_mechanism_emergence
    → rule_formation
      → enforcement_development
        → stable_order (enables collective action)
```

## 発動条件

1. 集合的行動の必要性
2. 繰り返しの相互作用
3. 調整の便益の認識
4. 執行メカニズムの確立

## 失敗条件

- **正統性欠如**: 統治への同意の崩壊
- **執行不能**: ルールの実効性喪失
- **分裂**: 統治対象の断片化
- **腐敗**: 統治者の私的利益追求
