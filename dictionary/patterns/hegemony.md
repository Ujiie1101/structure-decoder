# hegemony（覇権）

特定の様式・潮流・制度が支配的地位を獲得し、芸術界の規範を形成する構造。

## 定義

特定の芸術様式・理論・制度が文化的覇権を確立し、「正統」と「逸脱」を定義する権力を持つ。覇権は強制ではなく同意によって維持され、代替的な美学を周縁化しながら芸術界の構造を規定する。

## 構造

```
competing_styles (plurality)
  → legitimation_struggle
    → dominant_position
      → norm_setting
        → consent_production
          → hegemony (cultural_dominance)
```

## 発動条件

1. 卓越した芸術的成果
2. 制度的支援（アカデミー、美術館、批評）
3. 経済的・政治的後ろ盾
4. 正統性を付与する言説

## 失敗条件

- **硬直**: 革新力の喪失
- **抵抗**: 対抗文化の台頭
- **空洞化**: 形式のみの支配
- **分散**: 多元化による覇権喪失
