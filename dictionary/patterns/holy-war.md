# holy-war（聖戦）

世俗的対立が宗教的・イデオロギー的正当化を獲得し、妥協不能な絶対的闘争へ転化する構造。

## 定義

利害対立や権力闘争が、超越的価値（神・正義・歴史の必然）による正当化を得ることで、交渉・妥協・共存の余地が消滅する。敵は「悪」となり、殲滅が「義務」へと創発する。

## 構造

```
secular_conflict
  → ideological_framing
    → transcendent_justification
      → enemy_dehumanization
        → total_commitment
          → holy_war (emergent absolutism)
```

## 発動条件

1. 超越的権威の援用
2. 善悪二元論の適用
3. 敵の非人間化
4. 殉教の美化

## 失敗条件

- **世俗化**: 宗教的正当化の喪失
- **疲弊**: 双方のリソース枯渇
- **多元主義**: 絶対性の相対化
- **実利優先**: 現実的利害の再認識
