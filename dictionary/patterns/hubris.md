# hubris（傲慢）

成功や能力への過信が、限界の見落としを生み、自己破壊的な行動を引き起こす構造。

## 定義

過去の成功・知識・能力への確信が、謙虚さと外部フィードバックへの感受性を低下させ、現実との乖離を拡大させる。最終的に、見落とされた限界が破滅をもたらす。

## 構造

```
success_or_expertise
  → confidence_accumulation
    → feedback_resistance
      → reality_gap_widening
        → catastrophic_failure (from blind spots)
```

## 発動条件

1. 過去の顕著な成功
2. 成功の内部帰属（運ではなく実力）
3. 批判・警告の排除
4. 環境変化への鈍感さ

## 失敗条件

- **謙虚**: 限界認識の維持
- **多様性**: 異なる視点の取り込み
- **失敗経験**: 小さな挫折による学習
- **外部監視**: 第三者チェックの機能
