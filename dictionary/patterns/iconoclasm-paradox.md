# Iconoclasm Paradox（イコノクラスムの逆説）

## Definition

破壊者と保存者は、対象に「力がある」という同じ前提を共有している。

破壊行為自体が、対象の力を証明・強化する構造。

## Formula

```
object_exists → destroyer("it has power, must destroy") 
              → preserver("it has power, must protect")
              → both_confirm("it has power")
              → destruction_amplifies_power
```

## Core Insight

- 「壊すべきか、守るべきか」は間違った問い
- 正しい問いは「なぜ私たちはこれに力を感じるのか」
- 力がなければ、壊す理由も守る理由もない

## Historical Examples

| 時代 | 事例 | 破壊者 | 保存者 | 共有前提 |
|------|------|--------|--------|----------|
| 726年 | ビザンツ・イコノクラスム | レオ3世 | 修道士ステファノス | 聖像には神聖な力がある |
| 1517年 | 宗教改革期の聖像破壊 | プロテスタント | カトリック | 像は信仰に影響を与える |
| 2020年 | BLM銅像引き倒し | 抗議者 | 保守派 | 銅像は歴史を形作る |
| 現代 | 掛け算の順序論争 | 撤廃派 | 維持派 | 順序には教育的意味がある |

## Modern Applications

### 教育（順序権力）
- 掛け算の順序を「壊せ」と叫ぶ人 → 順序に力があると認めている
- 順序を「守れ」と主張する人 → 順序に力があると認めている
- 本当の問い：なぜ「×をつける権利」に力を感じるのか

### 組織文化
- 「古いルールを壊せ」派 → ルールに力があると認めている
- 「伝統を守れ」派 → ルールに力があると認めている
- 本当の問い：なぜ「誰かが決めた順番」に従っているのか

### SNS・キャンセルカルチャー
- 炎上させる側 → 対象の発言に影響力があると認めている
- 擁護する側 → 対象の発言に影響力があると認めている
- 無視が最大の否定だが、炎上はそれを不可能にする

## Derived Concepts

### 順序権力（Order Power）
「順番を決める者」が「正しさを決める者」になる構造。

| 領域 | 順序の例 | 順序が持つ力 |
|------|---------|-------------|
| 教育 | 掛け算の順序 | 正答の審判権 |
| ビジネス | 稟議書の押印順 | 序列の可視化 |
| 宗教 | 儀式の手順 | 聖と俗の境界 |
| 軍隊 | 敬礼の作法 | 命令系統の確認 |

## Escape Pattern

このパラドックスから抜け出す方法：

1. **力の言語化**: 「なぜこれに力を感じるのか」を問う
2. **無関心の選択**: 壊しも守りもしない（ただし社会的コストあり）
3. **構造への介入**: モグラではなく穴を叩く

## Related Patterns

- **chilling-effect**: 破壊の見せしめが萎縮を生む
- **labeling-cascade**: 破壊対象にラベルが付与される
- **purity**: 「穢れ」を破壊することで純粋性を保つ構造

## Usage Notes

記事で使う際のポイント：
- 「壊す/守る」の二項対立を俯瞰させる
- 読者を「第三の立場」（問いを立てる側）に誘導する
- 夫婦喧嘩、ダイエット、モグラ叩きなどの身近な比喩が有効
