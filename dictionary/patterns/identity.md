# identity（アイデンティティ）

自己表現と他者・文化との対話から、独自の芸術的自己が形成される構造。

## 定義

芸術家が既存の伝統・様式・影響との対話を通じて、模倣を超えた固有の声・スタイル・世界観を確立する。アイデンティティは与えられるものではなく、創造的実践の中から創発する。

## 構造

```
influences (tradition, masters, culture)
  → imitation_and_learning
    → dialogue_and_resistance
      → self_expression
        → unique_voice
          → identity (artistic_self)
```

## 発動条件

1. 伝統・師匠との深い対話
2. 模倣から逸脱への意志
3. 自己内省と実験
4. 独自の主題・技法の発見

## 失敗条件

- **埋没**: 影響に飲み込まれ独自性を失う
- **孤立**: 対話なき独善
- **固定化**: 確立したスタイルへの囚われ
- **模倣**: エピゴーネンに留まる
