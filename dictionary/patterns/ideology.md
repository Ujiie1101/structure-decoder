# ideology（イデオロギー）

芸術を通じた思想・世界観・政治的立場の表現から、集合意識の変容が出現する構造。

## 定義

芸術が特定の思想、政治的立場、世界観を体現・伝播・批判することで、観客の意識・価値観・行動に影響を与え、社会的変化の触媒となる。

## 構造

```
ideological_content (worldview/politics)
  → artistic_encoding
    → aesthetic_transmission
      → consciousness_impact
        → emergent_change (collective_shift)
```

## 発動条件

1. 明確な思想的立場
2. 芸術的技巧との統合
3. 観客の感受性
4. 社会的文脈との共鳴

## 失敗条件

- **プロパガンダ**: 芸術性の犠牲
- **説教**: 直接的すぎるメッセージ
- **無視**: 観客への不到達
- **誤読**: 意図と解釈のズレ

## 適用事例

| Layer | Entities |
|-------|----------|
| art | social-realism, political-art, agitprop |
| film | propaganda-film, documentary |
| literature | engaged-literature, dystopia |
| music | protest-song, national-anthem |
| architecture | fascist-architecture, socialist-realism |

## 関連パターン

- **emergence**: イデオロギーから意識変容が創発
- **resistance**: 体制への抵抗
- **transformation**: 社会変革
