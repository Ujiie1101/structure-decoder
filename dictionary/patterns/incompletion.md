# incompletion（未完成）

作品の未完状態・余白・不確定性から、鑑賞者の想像力が補完する意味が出現する構造。

## 定義

意図的または結果的な未完成性が、閉じた完成作品では得られない開放性・多義性・生成性を生む。欠如が創造的参加を誘発し、作品と鑑賞者の間に意味が創発する。

## 構造

```
incomplete_work (gap/absence)
  → viewer_engagement
    → imaginative_completion
      → multiple_interpretations
        → emergent_meaning (co-created)
```

## 発動条件

1. 意味ある欠如の存在
2. 鑑賞者の参加意欲
3. 想像力を誘発する手がかり
4. 解釈の自由度

## 失敗条件

- **単なる失敗**: 未完成と意図的不完全の混同
- **過剰な欠如**: 手がかり不足
- **閉鎖性**: 解釈を許さない
- **無関心**: 参加動機の欠如

## 関連パターン

- **emergence**: 不完全性から意味が創発
- **presence**: 欠如による現前効果
- **appropriation**: 鑑賞者による補完
