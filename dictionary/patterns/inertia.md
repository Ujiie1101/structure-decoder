# inertia（慣性）

既存の状態・方向性・習慣が、変化への抵抗力として作用し、現状維持を強化する構造。

## 定義

組織・個人・社会が獲得した慣習・投資・アイデンティティが、変化のコストを増大させ、環境変化にもかかわらず既存の軌道を維持し続ける力を生む。

## 構造

```
established_state
  → investment_accumulation
    → change_cost_increase
      → adaptation_resistance
        → path_persistence (despite external pressure)
```

## 発動条件

1. 長期間の安定状態
2. 既存システムへの大規模投資
3. 変化コストの可視性
4. 変化便益の不確実性

## 失敗条件

- **危機**: 現状維持の不可能性の明確化
- **リーダーシップ**: 変化を推進する強力な意志
- **競争圧力**: 適応しなければ淘汰
- **外部ショック**: 既存投資の無効化
