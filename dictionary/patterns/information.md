# information（情報）

不確実性を減少させる差異のパターンが、価値・権力・変化を生み出す構造。

## 定義

シャノンの定義「不確実性の減少」から、バイトソンの「差異を生む差異」まで。物質・エネルギーと並ぶ基本存在として、知識、意思決定、制御の基盤となる。情報の非対称性が権力を生む。

## 構造

```
uncertainty
  → signal_transmission
    → pattern_recognition
      → meaning_extraction
        → decision / action
```

## 発動条件

1. 不確実性の存在
2. 信号の伝達可能性
3. 受信者の解釈能力
4. 情報の時間価値

## 失敗条件

- **ノイズ**: 信号の劣化
- **過負荷**: 処理能力超過
- **誤解**: 解釈の失敗
- **陳腐化**: 情報価値の消失

## 適用事例

| Layer | Entities |
|-------|----------|

## 関連パターン

- **emergence**: 情報からの秩序創発
- **information-asymmetry**: 情報の非対称性
- **feedback**: 情報フィードバック
- **network-effect**: 情報ネットワークの価値
