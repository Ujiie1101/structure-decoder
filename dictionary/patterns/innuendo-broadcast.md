# innuendo-broadcast（意味の欠落投下）

第三者の前で**固有名を出し、詳細を曖昧化し、聞かれていることを認識した上で所作を運用する**装置。匂わせの三要素（投下・遮断・観察）の同時運用。

## 定義

以下の三要素が揃った状態を「匂わせの完成形」と定義する:

1. **投下**: ある固有名・形容詞・象徴を会話に出す
2. **遮断**: その詳細を語らず、慌てて遮るふりをする
3. **観察**: 第三者が居ることを認識した上で、その目線を意識して所作を作る

聞き手は意味の欠落を抱えたまま除外される。確認させないことで、聞き手を永遠に外に置く。

## 構造

```
fragment_emit（断片を出す）
  → faux_self_censorship（慌てて遮るふりをする）
    → third_party_acknowledge（聞き手の存在を意識した所作）
      → meaning_gap_in_listener（意味の欠落が聞き手の中に残る）
        → permanent_outsider（聞き手は永遠にinsider側に入れない）
```

## 観察可能な現代具体例

- SNSの「○○の件、まあ言わないけど」で止まる投稿
- 会議室で「あの人ってさ、いや何でもない」と笑う
- グループチャットで内輪のスタンプが特定の人だけ反応する
- 退室直後に発生する笑い声

## 対抗装置

- 即座の名指し要求（「で、それは誰のことですか」）
- 退室時の合言葉（「私のことなら、戻ってくる前に終わらせて」）
- 書面化の強制
