# inquiry-foreclosure（問いの先回り閉鎖）

知らないもの・未分類のものに出会ったとき、**問いが立ち上がる前に既知の箱へ分類してしまい、その分類が問いの座席を占有することで「なぜ」がはじまらなくなる**構造。無知が問題なのではなく、無知の次に来るはずの問いが、分類によって先回りで閉じられる。

## 定義

以下の三条件を満たす:

1. **未知との遭遇**: 受け手にとって分類先の定まらない対象が現れる
2. **分類の速度優位**: 分類する知性は問う知性より速く、先に反応を完了する
3. **座席の占有**: 下された分類（「意識高い」「変わってる」等）が問いの位置を埋め、以後その対象に問いが向かわなくなる

## 構造

```
未知の対象との遭遇
  ↓ 分類する知性が起動（速い）
最も近い既知の箱へ投入「これはあの箱だ」
  ↓ 分類が「問い」の座席に着席する
問う知性が起動する前に座席が埋まる（遅い）
  ↓
「なぜ」がはじまらない（無問いではなく、問いの先取り閉鎖）
```

## 発動条件

1. 分類する知性に最適化された空間（期限・役割・道具的合理性が支配的な場 = 職場など）
2. 対象が非道具的・私的（「何に効くか」で測れない）
3. 「知らない」と「問わない」が観察者から地続きに見えること（問いの閉鎖が無知に偽装される）

## 失敗が見えにくい理由

- 分類は沈黙のうちに完了する（ウィトゲンシュタイン命題7「語りえぬものについては沈黙せねばならない」の実演）
- だが沈黙の裏で判定は下りている（ポランニー「我々は語れる以上のことを知っている」＝暗黙知）
- → 当人は「問わなかった」自覚を持たない。問いが立ち上がる前に終わっているため

## 観察可能な現代具体例

- 職場で同僚の趣味・読書を「意識高いね」「変わってるね」で受け、なぜを聞かない
- 子供の突飛な質問を「そういうお年頃」で分類し、問い返さない
- 新人の異なるやり方を「うちのやり方じゃない」で閉じる
- SNSで異論を「アンチ」「信者」のラベルに投入し、内容を問わない

## 対抗装置

- 分類が出た瞬間に「いま箱に入れたか、それとも理解したか」を自問する（速度差の可視化）
- 「知らないから教えて」を会話の入り口として温存する（無知を問いの起点に戻す）
- 非道具的な対象を道具的空間に持ち込む側も、受け皿の不在を理解しておく（期待の誤射を防ぐ）

## 並走関係

- **分類する知性 / 問う知性** は同次元・優劣なしの並走。分類は生存に必須（有限の頭で膨大な世界を渡る装備）、問いは発見に必須（箱のないものに箱を作る）。本パターンは前者が後者を時間的に先取りする局面を指す。「より高い真の知性」への統合は起きない。
