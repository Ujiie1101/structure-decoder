# institutionalization（制度化）

非公式な慣行・運動・関係が、公式な制度・組織・規則として固定化される構造。

## 定義

自発的・流動的だった活動が、規則・役職・手続きを伴う恒久的な制度へと変容する。安定性と予測可能性が高まる一方、柔軟性と革新性が低下するリスクを伴う。

## 構造

```
informal_practice
  → routinization
    → formalization (rules/roles/procedures)
      → legitimization
        → institutional_stability + rigidity
```

## 発動条件

1. 継続的な活動の存在
2. 予測可能性・安定性への需要
3. 正当性付与のメカニズム
4. 資源の継続的確保

## 失敗条件

- **硬直化**: 環境変化への適応不全
- **形骸化**: 本来の目的の喪失
- **官僚化**: 手続きが目的化
- **反発**: 制度への抵抗・離脱
