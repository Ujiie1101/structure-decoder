# integration（統合）

異なる要素・分野・システムが結合し、より高次の統一体を形成する構造。

## 定義

かつて分離していた部分が、共通の枠組み・目的・技術によって一体化する。水平統合（同種結合）と垂直統合（異種結合）、さらに学際統合（知の越境）の三類型。統合後は部分の自律性が低下し、全体としての新機能が発現する。

## 構造

```
separate_elements (A, B, C)
  → interface_creation
    → mutual_adaptation
      → unified_system
        → emergent_capabilities (> A + B + C)
```

## 発動条件

1. 統合による相乗効果の期待
2. 共通言語・プロトコルの存在
3. 統合を主導するアクター
4. 部分間の補完性

## 失敗条件

- **非互換性**: 技術・文化の根本的不一致
- **抵抗**: 部分の自律性喪失への反発
- **複雑性爆発**: 統合コストが便益を超過
- **identity喪失**: 部分の独自性消滅
