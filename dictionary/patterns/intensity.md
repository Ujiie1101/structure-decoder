# intensity（強度）

エネルギーの集中から、質的に異なる体験が出現する構造。

## 定義

感覚、感情、または力の密度が臨界点を超えて集中することで、量的増加では説明できない質的変容が起こる。強度は単なる量の増大ではなく、新たな存在様態の創発である。

## 構造

```
diffuse_energy
  → concentration
    → density_increase
      → critical_threshold
        → qualitative_emergence (new mode of being)
```

## 発動条件

1. エネルギーの集中
2. 持続的な密度維持
3. 臨界点の存在
4. 変容を受け入れる器

## 失敗条件

- **拡散**: 集中の維持失敗
- **破壊**: 過剰強度による崩壊
- **鈍化**: 反復による感度低下
- **分散**: 焦点の喪失
