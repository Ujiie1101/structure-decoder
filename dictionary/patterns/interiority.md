# interiority（内面性）

外的表現から内的経験・主観的世界の探求へと重心が移行し、深層心理が出現する構造。

## 定義

芸術・思想において、外部世界の客観的描写から、内面世界・意識・感情・無意識の探求へと関心が移行する。主観的経験そのものが表現対象となり、内省が創造の源泉となる。

## 構造

```
external_representation
  → self_reflection
    → inner_exploration
      → psychological_depth
        → interiority (subjective_world_as_content)
```

## 発動条件

1. 外的表現の限界への自覚
2. 内省的態度の発達
3. 心理学・精神分析の影響
4. 個人主義の文化的基盤

## 失敗条件

- **自閉**: 内面に閉じて共有不能
- **過剰露出**: プライバシーの消失
- **ナルシシズム**: 自己陶酔への堕落
- **表面化**: 内面の商品化・定型化

## 適用事例

| Layer | Entities |
|-------|----------|
| art | expressionism, romanticism, stream-of-consciousness |
| philosophy | phenomenology, existentialism, introspection |
| psychology | psychoanalysis, depth-psychology |
| literature | confessional-poetry, bildungsroman |
| religion | mysticism, contemplation |

## 関連パターン

- **emergence**: 内面世界の創発的表現
- **mastery**: 内面探求の深化
- **transformation**: 外向から内向への転換
