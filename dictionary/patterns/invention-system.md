# invention-system（発明システム）

発明を促進・保護・普及させる制度的フレームワークの構造。

## 定義

知的財産権、研究機関、資金調達、技術移転などの制度が連携し、革新的アイデアの創出から社会実装までを体系的に支援する仕組み。

## 構造

```
individual_creativity
  → institutional_support (funding, facilities)
    → protection_mechanism (patents, IP)
      → knowledge_diffusion
        → systemic_innovation (continuous invention cycle)
```

## 発動条件

1. 知的財産保護の法的枠組み
2. 研究開発への資金供給
3. 知識共有のインフラ
4. 失敗を許容する文化

## 失敗条件

- **過剰保護**: 特許の乱用が後続革新を阻害
- **資金枯渇**: 研究開発投資の断絶
- **孤立**: 知識移転・共有の失敗
- **官僚化**: 制度の硬直化が創造性を阻害
