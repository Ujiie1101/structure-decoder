# inversion（反転）

価値・意味・関係が正反対に転換する構造。

## 定義

ある対象・概念・関係の評価や意味が、その反対のものへと変化する。善が悪に、強さが弱さに、中心が周縁に反転する。

## 構造

```
initial_valuation (+A / -B)
  → destabilizing_event
    → reframing
      → value_reversal (-A / +B)
        → new_equilibrium
```

## 発動条件

1. 既存評価体系の動揺
2. 反転を正当化するナラティブ
3. 反転を採用する主体の存在
4. 新評価の定着メカニズム

## 失敗条件

- **中途半端**: 反転が不完全で曖昧な状態に留まる
- **反動**: 旧評価への回帰
- **無関心**: 反転が認知されない

## 適用事例

| Layer | Entities |
|-------|----------|
| philosophy | nietzsche (価値の転倒), deconstruction |
| religion | martyrdom (死が勝利), kenosis (空虚が充満) |
| economics | crisis-to-opportunity |
| politics | revolution (被支配者が支配者に) |
| art | avant-garde (醜が美に), outsider-art |
| history | decolonization (植民地が独立国に) |

## 関連パターン

- **substitution**: 反転後の新価値が旧価値を置換
- **collapse**: 反転に耐えられない場合の崩壊
- **emergence**: 反転から新秩序が創発
