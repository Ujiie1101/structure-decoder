# invisible-hand（見えざる手）

自己利益を追求する個人の行動が、意図せず社会全体の利益に貢献する構造。

## 定義

分散した個人の利己的行動が、市場メカニズムを通じて、中央計画なしに効率的な資源配分を達成する。

## 構造

```
individual_self_interest
  → market_exchange
    → price_signals
      → resource_allocation
        → unintended_collective_benefit
```

## 発動条件

1. 競争的市場（独占なし）
2. 価格シグナルの機能
3. 取引コストの低さ
4. 外部性の不在（または内部化）

## 失敗条件

- **独占**: 競争圧力の欠如
- **外部性**: 価格に反映されない費用・便益
- **情報非対称**: 価格シグナルの歪み
- **道徳的基盤の欠如**: 搾取への堕落

## 注意

アダム・スミスは『国富論』で「見えざる手」を1回のみ使用。普遍的市場調和の法則として拡大解釈される傾向があるが、スミス自身は独占や政府介入の必要性も認識していた。
