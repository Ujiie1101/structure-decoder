# luxury（贅沢）

必要を超えた過剰の蓄積から、社会的差異化・美的価値・文化的意味が創発する構造。

## 定義

生存に必須でない余剰の存在と消費から、地位、趣味、洗練、芸術性といった必要性とは異なる次元の価値が出現する。

## 構造

```
surplus_resources
  → excess_consumption
    → distinction_marking
      → aesthetic_refinement
        → emergent_cultural_value (beyond utility)
```

## 発動条件

1. 余剰資源の存在
2. 社会的差異化の欲求
3. 洗練を評価する文化
4. 贅沢を可能にする技術・職人

## 失敗条件

- **欠乏**: 余剰がない状態
- **平等主義**: 差異化の抑制
- **下品**: 洗練なき過剰
- **禁欲**: 贅沢への道徳的非難
