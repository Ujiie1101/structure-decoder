# malice-threshold-lowering（悪意の敷居低下）

経済的圧力と技術民主化の組み合わせにより、善人が加害者になる敷居が下がる構造。

## 定義

以下の条件が揃うとき、通常は悪事に手を出さない人間が「試しにやってみる」段階に到達する。

1. **経済的圧力**: 物価高、給料据え置き、生活苦
2. **技術的敷居の消滅**: AIやツールが「やり方」を提供
3. **結果の不可視性**: 被害者の顔が見えない
4. **段階的エスカレーション**: 「ちょっとだけ」から始まる

## 構造

```
経済的圧力（物価高、給料据え置き）
  → 「ちょっとした副収入」への誘惑
    → AIが技術的敷居を下げる
      → 「試しにやってみるか」
        → 意図せず大きな被害を引き起こす
          → 「こんなはずじゃなかった」
```

## 発動条件

1. **経済的ストレス**: 生活費の圧迫、副業への関心
2. **技術的アクセス**: 誰でも使えるツールの存在
3. **匿名性**: 被害者との接点がない
4. **正当化の余地**: 「大企業からなら」「保険があるから」

## 失敗条件（抑止条件）

- **倫理的ブレーキ**: 「これでいいのか」と問う習慣
- **結果の可視化**: 被害者の顔が見える
- **社会的監視**: 周囲の人間が気づく
- **代替手段**: 正当な副収入の選択肢

## 関連する歴史的構造

### ミルグラム実験（1961）

権威者の指示により、65%の被験者が致死量の電気ショックを与えた。

| 要素 | ミルグラム実験 | 2026年のAI |
|------|---------------|------------|
| 権威者 | 白衣の実験者 | AIツール |
| エスカレーション | 15V刻み | 「簡単なものから」 |
| 責任の分散 | 「実験者の責任」 | 「ツールがやった」 |
| 結果の不可視 | 壁の向こうの叫び声 | 画面上の数字 |

### アーレント「悪の凡庸さ」

> 「彼の悪は悪魔的悪意ではなく、思考の完全な欠如から生じた」

アイヒマンは怪物ではなく、「仕事をしていた」だけ。AIツールを使う加害者も、「副業をしていた」だけ。

## 2×2マトリックス：意図と結果

|  | **結果：軽微** | **結果：深刻** |
|---|---|---|
| **意図：悪意あり** | 小悪党 | 犯罪者 |
| **意図：悪意なし** | ミス | **意図なき加害者** |

AIツールが増やしているのは**右下の象限**。

## 関連パターン

- **兵器転用のパラドックス**: 善意の技術が悪意の武器に
- **thoughtlessness**: 思考の欠如が道徳的破局を可能にする
- **democratization**: 技術の民主化が攻撃の民主化も生む
- **圧力革新効果（逆転）**: 経済的圧力が「悪事への革新」を引き起こす

## 構造的教訓

「悪人が悪事をする」は単純すぎる図式。

現実は「善人が、条件次第で、悪事に手を染める」。

条件とは：
1. 経済的圧力
2. 技術的敷居の消滅
3. 結果の不可視性
4. 思考の停止

これらを認識することが、予防の第一歩。
