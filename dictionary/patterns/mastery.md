# mastery（熟達）

継続的な練習と経験の蓄積から、技能の質的飛躍が出現する構造。

## 定義

長期間の意図的練習を通じて、個別スキルの統合が起こり、意識的努力なしに高度な技能を発揮できる状態に達する。量的蓄積が質的変容を生み、暗黙知が身体化される。

## 構造

```
novice (conscious_incompetence)
  → deliberate_practice
    → skill_accumulation
      → integration
        → automatization
          → mastery (unconscious_competence)
```

## 発動条件

1. 長期間の継続的練習（10,000時間説）
2. 質の高いフィードバック
3. 限界への挑戦（意図的練習）
4. 師弟関係または手本の存在

## 失敗条件

- **停滞**: コンフォートゾーンへの安住
- **過学習**: 柔軟性の喪失
- **燃え尽き**: 継続不能
- **模倣止まり**: 独自性の欠如
