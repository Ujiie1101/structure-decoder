# media-amplification（メディア増幅）

メディアが情報を伝達する過程で、意味・影響・感情が増幅される構造。

## 定義

出来事やメッセージがメディアを通じて伝播する際、選択・編集・反復・感情付加により、元の規模や意味を超えた影響力を獲得する。増幅は個々の報道の集積から創発する。

## 構造

```
original_event
  → media_selection (newsworthiness)
    → narrative_framing
      → emotional_loading
        → repetition_cascade
          → amplified_impact (emergent magnitude)
```

## 発動条件

1. ニュースバリューの存在
2. 感情的訴求力
3. 反復報道
4. ソーシャルメディアによる拡散

## 失敗条件

- **無関心**: 報道価値の欠如
- **飽和**: 過剰報道による麻痺
- **反証**: 事実による相殺
- **競合**: より大きなニュースによる埋没

## 適用事例

| Layer | Entities |
|-------|----------|
| history | yellow-journalism, propaganda, war-coverage |
| politics | moral-panic, scandal-amplification |
| economics | market-sentiment-contagion |
| technology | viral-misinformation |
| religion | televangelist-phenomenon |
| art | celebrity-culture |

## 関連パターン

- **cascade**: 増幅の連鎖
- **emergence**: 集合的関心の創発
- **reflexivity**: 報道が現実を変える
