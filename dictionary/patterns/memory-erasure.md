# memory-erasure（記憶消去）

集合的記憶から特定の出来事・人物・思想が消去される構造。

## 定義

権力による意図的削除、社会的忘却、ナラティブ再編などが相互作用し、あたかも「なかったこと」として歴史から姿を消す。消去は完全な抹消ではなく、想起の困難化として創発する。

## 構造

```
historical_event_or_figure
  → multiple_erasure_pressures
    → archival_reduction
      → narrative_exclusion
        → generational_forgetting
          → effective_nonexistence (emergent historical void)
```

## 発動条件

1. 現体制にとっての不都合
2. 記録・アーカイブへのアクセス制御
3. 教育・メディアの統制
4. 時間の経過と世代交代

## 失敗条件

- **分散記録**: 消去不能な複数のアーカイブ
- **亡命者**: 記憶を外部に持ち出す人々
- **考古学的発見**: 物理的証拠の残存
- **口承伝統**: 非文書的記憶の持続
