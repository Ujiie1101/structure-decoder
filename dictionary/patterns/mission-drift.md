# mission-drift（ミッション漂流）

組織・運動・個人の本来の目的が、時間経過とともに変質・拡散・逸脱していく構造。

## 定義

設立時の明確な使命が、成功・成長・環境適応の過程で徐々に変容し、本来の目的から乖離していく。漂流は多くの場合、意図せず漸進的に進行する。

## 構造

```
original_mission
  → success_and_growth
    → resource_acquisition
      → stakeholder_diversification
        → purpose_dilution (gradual deviation)
```

## 発動条件

1. 組織の成長と資源獲得
2. 多様なステークホルダーの参入
3. 短期的成果への圧力
4. 創設者・原点からの距離

## 失敗条件

- **ミッション再確認**: 定期的な原点回帰
- **創設者関与**: 本来の目的の守護者
- **測定システム**: ミッションとの整合性評価
- **外部監視**: 逸脱への警告機能

## 適用事例

## 関連パターン

- **institutionalization**: 制度化がミッション漂流を招く
- **capture**: 特定利害によるミッション歪曲
- **erosion**: ミッションの漸進的侵食
