# monumentality（記念碑性）

物理的スケール・永続性の追求から、集合的記憶・アイデンティティ・権威が創発する構造。

## 定義

巨大さ、永続性、荘厳さの物質的実現から、個人を超えた集合的意味、歴史的連続性、社会的統合が出現する。

## 構造

```
physical_scale
  → permanence_aspiration
    → collective_labor
      → symbolic_condensation
        → emergent_memory (transcends material)
```

## 発動条件

1. 動員可能な資源・労働力
2. 永続への意志
3. 集合的アイデンティティの核
4. 記念すべき出来事・人物・理念

## 失敗条件

- **資源不足**: 完成不能
- **権威失墜**: 記念碑の意味喪失
- **破壊**: 物理的消滅
- **忘却**: 記憶との断絶

## 適用事例

| Layer | Entities |
|-------|----------|
| art | architecture, public-sculpture, land-art |
| history | pyramids, war-memorials |
| religion | cathedral, temple, pilgrimage-site |
| politics | national-monuments, propaganda-architecture |
| philosophy | sublime, aura |
| technology | mega-projects, infrastructure |

## 関連パターン

- **emergence**: 物質から記憶が創発
- **preservation**: 記念碑の保存機能
- **authority**: 記念碑による権威表象
