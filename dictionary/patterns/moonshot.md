# moonshot（ムーンショット）

現状の延長線上にない大胆な目標を設定し、それに向けて資源を集中投下する構造。

## 定義

10%改善ではなく10倍改善を目指す野心的目標設定。不可能に見える目標が逆に制約を解除し、根本的な再設計を可能にする。失敗を許容する文化と、長期コミットメントが前提。

## 構造

```
audacious_goal
  → constraint_liberation
    → resource_concentration
      → radical_innovation
        → breakthrough / failure
```

## 発動条件

1. 既存アプローチの限界認識
2. 大胆な目標設定
3. 十分な資源・時間
4. 失敗許容の文化

## 失敗条件

- **資源枯渇**: 長期投資の継続困難
- **目標喪失**: 方向性の迷走
- **日和見**: 安全な改善への回帰
- **検証不足**: 実現不可能性の放置
