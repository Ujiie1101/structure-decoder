# mutual-checking（相互牽制）

複数の権力が相互に監視・抑制し合うことで均衡が創発する構造。

## 定義

各アクターが他者の行動を牽制することで、いずれも単独での支配に至らない安定状態が自己組織化する。意図的設計（三権分立）と創発的均衡（勢力均衡）の両方を含む。

## 構造

```
multiple_power_centers
  → mutual_observation
    → threat_of_countermeasures
      → behavioral_restraint
        → dynamic_equilibrium (emergent stability)
```

## 発動条件

1. 複数の対等な権力主体
2. 相互行動の可視性
3. 報復能力の確保
4. 全面対立のコスト認識

## 失敗条件

- **一極集中**: 一者が圧倒的優位
- **共謀**: 牽制主体間の結託
- **情報非対称**: 行動の不可視化
- **外部ショック**: 均衡破壊的危機
