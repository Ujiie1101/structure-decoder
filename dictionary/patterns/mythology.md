# mythology（神話）

集合的物語の蓄積から、共同体の意味体系が出現する構造。

## 定義

個々の物語、象徴、儀礼が相互作用し、世代を超えて伝承される過程で、個別のエピソードの総和を超えた統合的な世界観・価値体系が創発する。神話は現実を説明するだけでなく、現実を構成する。

## 構造

```
individual_stories
  → collective_transmission
    → symbolic_condensation
      → ritual_embodiment
        → worldview_emergence (transcends individual narratives)
```

## 発動条件

1. 物語の反復的伝承
2. 共同体の共有経験
3. 象徴的意味の凝縮
4. 儀礼による身体化

## 失敗条件

- **脱神話化**: 合理的批判による解体
- **断絶**: 伝承の途絶
- **形骸化**: 意味の喪失と形式のみの残存
- **私物化**: 権力による神話の専有
