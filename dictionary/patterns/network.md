# network（ネットワーク）

ノードとリンクの接続構造から生まれる効果と動態。

## 定義

要素（ノード）が相互接続（リンク）することで、個々の要素の総和を超える機能・価値・効果が生まれる。ネットワーク効果、スケールフリー構造、ハブの役割を含む。

## 構造

```
nodes (elements)
  → links (connections)
    → network_topology
      → network_effects
        → emergent_functionality
```

## 発動条件

1. 接続可能な複数ノード
2. 接続の価値/動機
3. 接続を可能にするプロトコル
4. クリティカルマスの達成

## 失敗条件

- **孤立**: 接続の不成立
- **過負荷**: ハブの崩壊
- **分断**: ネットワークの分裂
- **独占**: 中央集権化
