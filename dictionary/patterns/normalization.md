# normalization（正規化）

異常・逸脱が繰り返されることで、次第に正常・許容範囲として認識されるようになる構造。

## 定義

当初は逸脱・異常・例外として認識されていた事象が、反復・継続により心理的閾値が移動し、新たな「正常」として受容されるプロセス。

## 構造

```
initial_deviation
  → repeated_exposure
    → threshold_shift
      → cognitive_adaptation
        → normalized_state (deviance becomes baseline)
```

## 発動条件

1. 逸脱の反復的発生
2. 即時的な重大結果の不在
3. 漸進的な変化
4. 比較基準の移動

## 失敗条件

- **ショック**: 急激な変化が適応を阻害
- **外部視点**: 第三者の指摘による再認識
- **危機**: 正規化された逸脱が破局を招く
- **記憶**: 本来の基準の想起
