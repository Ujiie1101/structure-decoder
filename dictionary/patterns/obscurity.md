# obscurity（曖昧）

明確な意味の確定を拒否し、多義性・不確定性から深い解釈可能性が生まれる構造。

## 定義

芸術作品が一義的な解釈を拒み、曖昧さ・多義性・謎を内包することで、鑑賞者の能動的参加と無限の解釈を可能にする。曖昧さは欠陥ではなく、意味の豊穣さの条件である。

## 構造

```
clear_meaning (determinate)
  → ambiguity_introduction
    → multiple_readings
      → interpretive_openness
        → depth_of_meaning
          → obscurity (inexhaustible_significance)
```

## 発動条件

1. 単純な意味伝達への不満
2. 複雑な現実への誠実さ
3. 鑑賞者への信頼
4. 言語・表現の限界への自覚

## 失敗条件

- **不明瞭**: 意図なき混乱
- **難解主義**: 曖昧さの装飾的使用
- **排他性**: 理解可能な人を限定
- **空虚**: 深みのない曖昧さ
