# observation（観察）

対象への精密な注視から、従来見落とされていた現実の層が出現する構造。

## 定義

先入観を排し、対象を忠実に観察することで、慣習的な見方では捉えられなかった細部、関係性、真実が浮かび上がる。科学的態度と芸術的洞察が交差する地点。

## 構造

```
subject (observer)
  → attentive_looking
    → suspension_of_preconception
      → detail_emergence
        → new_perception (reality_layer)
```

## 発動条件

1. 観察者の意図的集中
2. 十分な時間と忍耐
3. 先入観からの解放
4. 記録・表現の手段

## 失敗条件

- **選択的知覚**: 見たいものだけを見る
- **過剰解釈**: 観察を超えた意味付け
- **表面停滞**: 深層に至らない
- **客観性の錯覚**: 観察者の影響の無視

## 適用事例

| Layer | Entities |
|-------|----------|
| art | impressionism, realism, documentary-photography |
| science | empiricism, natural-history, ethnography |
| literature | naturalism, journalism, reportage |
| philosophy | phenomenology, empirical-observation |

## 関連パターン

- **emergence**: 観察から新知覚が創発
- **interiority**: 内面への観察
- **mastery**: 観察技術の深化
