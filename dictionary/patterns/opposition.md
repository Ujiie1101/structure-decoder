# opposition（対立）

相異なる立場・価値・勢力が互いを否定し合う関係から、境界・アイデンティティ・動態が創発する構造。

## 定義

二つ以上の要素が相互に排斥し合う関係が成立することで、各要素の輪郭が明確化し、緊張・競争・対話の場が生成される。対立は破壊的にも創造的にも作用する。

## 構造

```
difference
  → mutual_negation
    → boundary_formation
      → identity_crystallization
        → dynamic_tension (emergent opposition)
```

## 発動条件

1. 異なる立場の存在
2. 相互認識と接触
3. 価値・利害の非両立性
4. 境界を維持する動機

## 失敗条件

- **無関心**: 相互認識の欠如
- **吸収**: 一方が他方に同化
- **消滅**: 一方の完全な敗北
- **融合**: 第三の立場への統合

## 適用事例

| Layer | Entities |
|-------|----------|
| religion | orthodoxy-heresy, monotheism-polytheism, sacred-profane |
| philosophy | idealism-materialism, rationalism-empiricism |
| politics | left-right, authority-liberty |
| economics | capital-labor, free-market-protectionism |
| history | reformation-counter-reformation, east-west |

## 関連パターン

- **dialectic**: 対立からの止揚
- **schism**: 対立による分裂
- **reconciliation**: 対立の解消
