# organic（有機的）

自然の成長原理に倣い、部分が全体と調和しながら有機的統一を形成する構造。

## 定義

芸術作品において、機械的・人工的な構成ではなく、生命体のように部分と全体が相互依存し、内的必然性によって形態が展開する。各要素が全体の生命に寄与し、全体が各部分に意味を与える。

## 構造

```
natural_principle
  → growth_process
    → interdependence
      → internal_necessity
        → organic_unity (living_whole)
```

## 発動条件

1. 自然・生命への観察と模倣
2. 機械論的世界観への反発
3. 内的必然性の感覚
4. 全体と部分の相互照応への意識

## 失敗条件

- **機械的模倣**: 有機性の外形だけを借用
- **無秩序**: 成長原理なき増殖
- **過剰装飾**: アール・ヌーヴォーの退廃
- **形式主義**: 有機性の様式化・硬直化
