# oscillation（振動）

対極的な状態の間を周期的に往復する動きから、動的均衡が出現する構造。

## 定義

聖と俗、拡張と収縮、革新と伝統、熱狂と沈静など、対立する極の間を振り子のように行き来する。静的な固定ではなく、振動そのものがシステムの本質となり、動的な安定を生み出す。

## 構造

```
pole_A (e.g., sacred)
  → movement_toward_B
    → pole_B (e.g., profane)
      → counter_movement
        → return_to_A
          → dynamic_equilibrium (oscillation as pattern)
```

## 発動条件

1. 対極的な状態の存在
2. 一方への過度の傾斜が反動を生む
3. 復元力・反作用の存在
4. システムの開放性

## 失敗条件

- **固定化**: 一方の極への定着
- **崩壊**: 振幅が大きすぎて破綻
- **減衰**: エネルギー不足で停止
- **無秩序**: パターンの消失

## 適用事例

| Layer | Entities |
|-------|----------|
| religion | sacred-profane, revival-decline, mysticism-rationalism |
| economics | boom-bust-cycle, inflation-deflation |
| politics | left-right-swing, centralization-decentralization |
| culture | tradition-innovation, purism-eclecticism |
| psychology | mania-depression, introversion-extraversion |

## 関連パターン

- **emergence**: 振動パターンの創発
- **balance**: 振動による均衡維持
- **transformation**: 振動を通じた変容
