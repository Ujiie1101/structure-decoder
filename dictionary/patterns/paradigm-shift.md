# paradigm-shift（パラダイムシフト）

支配的な思考枠組み・前提が根本的に転換する構造。

## 定義

既存のパラダイム（支配的な理論・方法論・世界観）が、蓄積された anomaly（異常事例）により危機に陥り、新しいパラダイムへと移行する。Thomas Kuhn の科学革命論に由来。

## 構造

```
dominant_paradigm
  → accumulating_anomalies
    → paradigm_crisis
      → competing_paradigms
        → new_paradigm (incommensurable)
```

## 発動条件

1. 既存パラダイムの anomaly 蓄積
2. 危機の認識と articulation
3. 代替パラダイムの提示
4. コミュニティの転向

## 失敗条件

- **修正吸収**: anomaly が既存パラダイム内で解決
- **孤立**: 新パラダイム支持者の不足
- **通約不能性**: 新旧間の対話不成立
- **制度障壁**: 既存権威による抑圧

## 適用事例

| Layer | Entities |
|-------|----------|
| technology | digital-revolution, ai-revolution, internet |
| philosophy | copernican-revolution, kuhn |
| science | relativity, quantum-mechanics |

## 関連パターン

- **transformation**: パラダイムシフトは変革の一形態
- **substitution**: 新パラダイムが旧を置換
- **emergence**: 新パラダイムからの創発
