# Parental Deposit（親権の供託）

## 定義

私的権威（家庭・コミュニティ・親）が機能不全に陥ったとき、その失われた権威を**国家・上位制度に外注する**構造。委託する側は「自分にはできない」と認め、委託先は「公益のため」と引き受ける。両者の暗黙の合意で成立する。

decision-delegation-desire（決定委託欲求）の**家族領域における特殊形**。

## 公式

```
家庭の権威（親が子に直接命じる）
   ↓ 機能しなくなる
親「子供がSNSをやめない」
   ↓ 認知の歪み
親「私が止めさせられない」← × → 親「子供が言うことを聞かない」← ○
   ↓ 真因の隠蔽
「誰か外部権威に止めさせてほしい」
   ↓
国家・学校・専門家・アルゴリズムに委託
   ↓ 結果
私的権威の崩壊が、公的権威の肥大で見えなくなる
```

**親権は失われたのではない。供託されたのである。**

## メカニズム

### 1. 認知の歪み：能動から受動へ

親自身がSNS規制を実行できないとき、二通りの言語化がある：

| 真の状態 | 表向きの語り |
|----------|--------------|
| 私は子供に従わせる権威を失った | 子供はSNSをやめない |
| 私は決断と継続的執行ができない | 子供がやめてくれない |
| 私は親としての機能不全である | 国家がやってくれるべき |

**主語が「私」から「子供」「国家」に転換**される。これによって自己の無能の直視を回避できる。

### 2. 国家の正当性供給

国家側にとって、親権の供託は**正当性の源泉**になる。
「親が困っている」「家庭が機能しない」と国家が言うとき、それは国家介入の許可証になる。
親の無能は、国家の存在意義に転化される。

### 3. 中間層（学校・専門家）の媒介

親→国家の直接接続は稀で、通常は中間層を経由する：
- 学校（しつけ・道徳教育・スマホ管理）
- 児童相談所・スクールカウンセラー（親の役割の補完）
- 育児書・SNS（「専門家が言うには」）
- アルゴリズム（YouTube Kids、ペアレンタルコントロール）

中間層が膨張するほど、親が直接子供と向き合う場面が減る。

### 4. 不在地主化

親は子供という「土地」を所有しているが、もう自分で耕せない。
だから国王（国家）に治安維持を頼む。これが**「不在地主化した親」**の構造。
所有権は手放さないが、実効支配は委託する。

## 発動条件

1. 私的権威の機能不全（親が子をコントロールできない）
2. 自己の無能の言語化を回避したい心理
3. 委託先となる強い公的制度の存在（国家、学校、専門家、テック）
4. 委託の文化的正当性（「子供のため」「専門家に任せよう」）

4つ揃うと親権の供託が発動する。

## 失敗条件（供託が止まる条件）

- **親自身による直接的執行**: 親がSNSアプリを物理的に削除し継続管理する
- **コミュニティの再活性化**: 近所・親戚・地域が子育てを共有
- **委託先の機能不全**: 国家が頼りにならないと露呈する（例：教師の権威崩壊）
- **「親であること」の再定義**: 自分の無能を直視し、それでも親であろうとする実存的選択

## 判別法

1. 親が「自分が」ではなく「子供が」「制度が」を主語にする
2. 規制・ルールを国家に求める頻度が、家庭内ルールを設定する頻度より高い
3. 親が自己の判断より「専門家」「アプリ」「学校方針」に従う
4. 親自身は同じ行為（SNS）を続けている

4つ揃うと親権の供託が作動している。

## 事例

| 領域 | 供託される機能 | 受け皿 |
|------|----------------|--------|
| SNS利用 | 時間管理・コンテンツ判断 | 国家規制、ペアレンタルコントロールアプリ |
| 食事 | 栄養管理 | 給食制度、栄養士、宅食サービス |
| しつけ | 規範形成 | 学校、塾、習い事 |
| 安全 | 居場所監視 | GPS、見守りアプリ、警察 |
| 性教育 | 知識提供 | 学校、ネット |
| メンタル | 心理ケア | スクールカウンセラー、児童精神科 |
| 進路 | 意思決定支援 | 進学校、コンサル、AI診断 |

**現代の親は「子供を持つ」が「子供を育てる機能の大半は外注」している。**

## 反対パターン

- **直接親権**：親が自分で判断し執行する。摩擦が高いが権威を保つ。
- **共同親権**：親族・コミュニティが分担。中世以前のモデル。
- **意図的非介入**：あえて何もしない。子供の自律を信じる（リスク高）。

## 核心洞察

> **「親が子供を止められない」ではなく「親が止めさせる権威を国家に供託している」。**
> **親権は失われたのではない。供託されたのである。所有権は親に、実効支配は国家に。**
> **これは中世封建制の不在地主と同じ構造である。**

## メタ観察

親権の供託は、家庭領域だけでなく**「自分でやるべきことを社会化する」一般的構造**の一例である：

- 食事 → 外食・宅食
- 健康管理 → 医療・保険
- 老後 → 年金・介護施設
- 葬儀 → 葬祭業
- 結婚 → マッチングアプリ
- 友人選び → SNSアルゴリズム
- 自己理解 → 性格診断・コーチング

**現代人は「自分の人生を自分で運営する」機能の大半を、外部制度に供託している。**

そして供託先が崩れたとき（国家の財政破綻、企業の倒産、アルゴリズムの暴走）、
**人間は自分が何をできなくなっていたかに、はじめて気づく。**

親権の供託は、その縮図である。
