# pedigree-leverage（門閥レバレッジ）

生まれた家柄が、その後の人生すべてに「てこ」として効いてしまう構造。

## 定義

初期条件（出生時の家柄・階層・地域）が、教育・人脈・機会・自己認識を通じて増幅され、人生全体の軌道を決定する構造。レバレッジが高い社会では、個人の努力や才能より「どこに生まれたか」が成果を支配する。

## 構造

```
birth_condition (family, class, region)
  → access_differential (education, network, opportunity)
    → compound_effect (advantage breeds advantage)
      → life_trajectory_determination
        → social_reproduction
```

## レバレッジ率

| 時代・社会 | レバレッジ率 | 説明 |
|-----------|-------------|------|
| 江戸時代（士農工商） | ~100% | 身分が人生を完全決定 |
| 明治初期（1872年〜） | ~60% | 身分制崩壊、しかし経済格差残存 |
| 高度成長期 | ~40% | 公教育普及、階層移動加速 |
| 現代日本 | ~50% | 教育格差・情報格差で再上昇 |
| **第二次ガチャリセット後** | ~30%? | AI・ネットで学習環境民主化 |

## 発動条件

1. **教育格差** — 家庭の経済力が教育機会を左右
2. **人脈の相続** — 親の人脈が子の機会を創出
3. **文化資本** — 「何が価値か」を知っている/知らない
4. **自己認識** — 「自分はここまでの人間」という内面化

## 失敗条件（レバレッジ低減）

- **知の民主化**: 印刷革命、公教育、インターネット、AI
- **身分制度崩壊**: 革命、敗戦、制度改革
- **流動性向上**: 転職市場、起業環境、国際移動
- **構造資本の獲得**: 「仕組みを見る力」は相続されない

## 福沢諭吉の洞察

> 「門閥制度は親の仇でござる」
> ──『福翁自伝』（1899年）

福沢の父・百助は学者の才がありながら、下級武士という身分のため出世できず、44歳で死去。福沢は『学問のすゝめ』で門閥レバレッジを否定し、「学ぶと学ばざるとによりてできる」と主張した。

## 関連パターン

- **exit-blockade**: 階層から「逃げられない」構造
- **causal-misattribution**: 構造問題を個人の努力不足に誤帰属
- **democratization**: レバレッジを低減させる方向
- **compound-effect**: レバレッジが時間とともに増幅

## 核心洞察

> **家柄ガチャで負けても、「構造資本」は相続されない。だから逆転できる。**
