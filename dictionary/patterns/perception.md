# perception（知覚）

感覚データの統合から、対象の意味・質・全体性が創発する構造。

## 定義

断片的な感覚入力（光、音、触覚など）の相互作用から、単なる感覚の総和ではない統一された経験・意味・美的判断が出現する。

## 構造

```
sensory_data
  → neural_integration
    → gestalt_formation
      → meaning_attribution
        → emergent_experience (irreducible to stimuli)
```

## 発動条件

1. 感覚器官の機能
2. 注意の方向づけ
3. 過去経験との照合
4. 文脈の存在

## 失敗条件

- **感覚遮断**: 入力データの欠如
- **錯覚**: 統合プロセスの誤作動
- **先入観**: 過去経験による歪曲
- **注意散漫**: 統合の不完全性

## 関連パターン

- **emergence**: 知覚は感覚からの創発
- **interiority**: 知覚は内的経験を構成
- **transformation**: 知覚が世界観を変革
