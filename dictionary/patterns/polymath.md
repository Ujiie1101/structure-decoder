# polymath（博学）

多領域の知識・技能が一人の中で統合され、分野横断的な洞察を生み出す構造。

## 定義

異なる分野の知識や技能を深く習得することで、単一分野の専門家には見えない接続点・パターン・可能性が認識され、革新的な創造や発見が可能になる。

## 構造

```
diverse_knowledge_acquisition
  → cross_domain_connections
    → pattern_recognition_across_fields
      → novel_synthesis
        → breakthrough_insight (irreducible to single domain)
```

## 発動条件

1. 複数分野への深い関与
2. 分野間の類推能力
3. 知的好奇心と学習継続
4. 統合的思考の習慣

## 失敗条件

- **散漫**: 広く浅くで深い理解に至らない
- **孤立**: 分野間を接続する視点の欠如
- **専門圧力**: 社会的に単一専門への収斂を強制される
- **時間制約**: 複数分野習得の時間不足

## 適用事例

| Layer | Entities |
|-------|----------|
| philosophy | aristotle, leibniz |
| art | leonardo-da-vinci, goethe |
| technology | benjamin-franklin, elon-musk |
| economics | herbert-simon, hayek |

## 関連パターン

- **synthesis**: 博学が統合を可能にする
- **emergence**: 博学から新しい洞察が創発
- **convergence**: 異分野の収斂
