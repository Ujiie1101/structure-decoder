# posthumous（死後評価）

生前には認められなかった価値が、死後に再発見・再評価されて出現する構造。

## 定義

芸術家・思想家が存命中には無視・拒絶・誤解されていたが、死後に時代の文脈が変化し、その作品・思想の先駆性や価値が認識される。遺作や未発表作品の発見が契機となることも多い。

## 構造

```
living_artist (unrecognized)
  → death
    → temporal_distance
      → contextual_shift
        → rediscovery
          → canonical_status (posthumous fame)
```

## 発動条件

1. 同時代の価値基準との乖離
2. 作品・記録の保存
3. 後世の解釈者・擁護者の出現
4. 時代の価値観変化

## 失敗条件

- **散逸**: 作品・記録が失われる
- **永続的無視**: 再評価の契機が訪れない
- **誤読の固定**: 誤った解釈が定着
