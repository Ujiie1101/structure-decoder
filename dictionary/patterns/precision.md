# precision（精密）

技術的正確さと細部への注意から、高度な完成度と説得力が達成される構造。

## 定義

細部に至るまでの正確な制御と、誤差を許さない技術的完璧さを追求する。精密さは冷たい技術主義ではなく、対象への深い敬意と理解の表れであり、見えないところまでの配慮が全体の質を決定する。

## 構造

```
attention_to_detail
  → technical_control
    → error_elimination
      → consistency
        → integration
          → precision (flawless_whole)
```

## 発動条件

1. 高度な技術的訓練
2. 完璧主義的態度
3. 細部への敬意
4. 品質基準の内面化

## 失敗条件

- **機械的**: 生命感の喪失
- **強迫的**: 創造性の抑圧
- **冷淡**: 感情の欠如
- **過剰**: 細部への執着で全体を見失う
