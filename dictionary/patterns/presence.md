# presence（現前）

作品・パフォーマンスの「今ここ」における直接的存在から、媒介を超えた体験が出現する構造。

## 定義

芸術作品やパフォーマンスが、再現・複製・記録ではなく、「今この瞬間」に直接経験される一回性の体験を生み出す。オーラ、ライブ性、身体性が媒介なき強度を生む。

## 構造

```
direct_encounter (here_and_now)
  → unmediated_experience
    → temporal_singularity
      → bodily_engagement
        → emergent_intensity (aura/liveness)
```

## 発動条件

1. 直接的な身体的遭遇
2. 一回性・不可逆性の自覚
3. 複製不可能な要素
4. 観客の全身的参与

## 失敗条件

- **記録依存**: 記録への還元
- **反復**: 定型化による強度喪失
- **距離**: 身体的接近の欠如
- **散漫**: 集中の不足

## 適用事例

| Layer | Entities |
|-------|----------|
| art | performance-art, happening, live-art |
| music | live-concert, improvisation |
| theatre | live-theatre, ritual |
| dance | contemporary-dance, butoh |
| religion | liturgy, pilgrimage |

## 関連パターン

- **emergence**: 直接経験から強度が創発
- **interiority**: 内面との共振
- **incompletion**: 不確定性の現前
