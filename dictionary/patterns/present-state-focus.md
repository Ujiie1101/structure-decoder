# present-state-focus（現状認識優位）

未来予測より、今何があるかを見る方が有効な状況での意思決定構造。

## 定義

不確実性が高い転換点において、未来を予測しようとするより「今、何があるか」「今、何ができるか」に集中することで、適応的な行動が可能になる現象。

## 構造

```
high_uncertainty_environment
  → prediction_failure (future is unknowable)
    → present_state_assessment
      → available_resources_inventory
        → immediate_action_with_feedback
          → emergent_adaptation
```

## 発動条件

1. 未来予測が構造的に困難な状況
2. 認知の限界の自覚（「自分が何を知らないか」を知っている）
3. 即座のフィードバックが得られる環境
4. 行動の柔軟性（固定計画への執着がない）

## 失敗条件

- **予測への執着**: 「もう少しデータが欲しい」の繰り返し
- **計画の硬直性**: 計画通りに進めることが目的化
- **過去パターンへの依存**: 前例を未来に投影
- **認知の過信**: 「自分は客観的だ」という錯覚

## 適用事例

## 関連パターン

- **emergence**: 現状からの秩序創発
- **feedback**: 即座のフィードバックによる調整
- **adaptation**: 環境変化への適応
