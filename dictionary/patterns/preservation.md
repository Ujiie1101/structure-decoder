# preservation（保存）

価値・伝統・知識・制度が時間を超えて維持される構造。

## 定義

変化の圧力に対抗して、重要なものを損なわずに次世代へ伝達する。意図的保存（アーカイブ）と無意識的保存（慣習）を含む。

## 構造

```
valued_content
  → encoding/institutionalization
    → transmission_mechanism
      → reception/interpretation
        → continuation (identity maintained)
```

## 発動条件

1. 保存に値する価値認識
2. 伝達メカニズム（教育、儀式、文書）
3. 受容する後継者
4. 保護する制度・共同体

## 失敗条件

- **断絶**: 伝達メカニズムの崩壊
- **忘却**: 価値認識の消失
- **汚染**: 本質の歪曲・変質
- **拒絶**: 後継者の不受容

## 適用事例

| Layer | Entities |
|-------|----------|
| religion | scripture, liturgy, tradition, canon |
| politics | constitution, institution, precedent |
| art | museum, restoration, canon |
| technology | archive, backup, version-control |

## 関連パターン

- **procedural**: 手続きが保存を実現
- **constraint**: 制約が変化を抑制
- **resistance**: 変化への抵抗として保存
