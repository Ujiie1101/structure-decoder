# prevention-self-erasure（予防の自己消去）

予防の成功そのものが、その成功の証拠を消去し、結果として予防の正統性を侵食する構造。便益が不可視化され、コストのみ可視のまま、世代内および世代間で記憶が劣化することで、予防拒否が「合理的選択」として立ち上がる。

## 定義

予防介入が機能している期間中、(1) 防がれた被害は反実仮想でしか語れず数えられない、(2) 副作用や手間などコストは可視化され続ける、(3) 体験者世代がまだ生存していても記憶は「他人の経験」として浮遊化する、結果として (4) 予防そのものが「過剰」「不要」と再フレーミングされ拒否が始まる。

prevention の親パターンの**失敗条件「予防のパラドックス」**を独立構造として展開したもの。

## 公式

```
prevention-self-erasure
  = prevention-invisibility    （反実仮想の構造的弱さ：便益が記録されない）
  × experiential-amnesia       （世代内記憶の他人事化：身体化記憶の劣化）
  × generational-amnesia       （長期背景：体験者世代の退場、補強因子）
```

2 atom（prevention-invisibility × experiential-amnesia）で発動。3つ目の generational-amnesia は長期的な決定要因。

## 発動条件

1. **十分な成功期間**: 予防が機能している期間が概ね20-60年継続し、被害の身体的記憶が薄まる
2. **コストの可視性持続**: 副作用、手間、費用などのコストは介入のたびに可視化される
3. **記憶の制度化への移行**: 体験者の口承 → 教科書/統計/Wikipedia への記憶質の劣化
4. **「自然」「権利」型物語の利用可能性**: 拒否を正当化する代替言説が存在する（オルテガの "natural rights" 論理）

## 失敗条件（自己消去が回避される条件）

### 個別原子への介入

- **prevention-invisibility**: 反実仮想の数値化、推計を伴う「防がれた被害」の物語化（counterfactual-quantification）
- **experiential-amnesia**: 体験者の一人称証言の制度化（オーラルヒストリー、博物館、当事者対話）
- **generational-amnesia**: 周期的な小規模再発を許容することで記憶を更新（ワクチン例外集団の限定的存在など）

### 構造的介入

- **同型のドメイン横断比較**: ビタミンK・HPV・耐震など複数ドメインで同型構造を可視化することで「これは私の選択の問題ではなく、構造の問題だ」を伝達
- **コストの不可視化**: 副作用報告システムを統計化し、個別ドラマ化を防ぐ（ただし倫理的トレードオフあり）

## 判別法

以下が観察された場合、prevention-self-erasure が作動している:

1. 拒否者が「自然」「化学物質」「自分の選択」など属性ベース言語で語る
2. 体験者世代（祖父母世代）の警戒が「過保護」「神経質」と再フレーミングされる
3. 撲滅前の被害数字（死亡率、入院率）が日常会話から消えている
4. 予防の便益を語る際に推計や反実仮想を必須としなければならない
5. 「自分の知っている誰か」が被害者リストに登場しない

- **ハンナ・アーレント『過去と未来の間』(1961)**: 伝統の断絶。本パターンは予防の伝統の断絶
- **マイケル・ポラニー『暗黙知の次元』(1966)**: 体験的知識の制度化困難性
- **ハンス・ヨナス『責任という原理』(1979)**: 予防原則の倫理的基礎。本パターンは予防原則の社会的脆弱性を示す
- **アルバート・ハーシュマン『Exit, Voice, Loyalty』(1970)**: 衰退する制度への応答。予防拒否は Exit の特殊形
- **ジョージ・サンタヤーナ『理性の生活』(1905)**: 「過去を記憶できない者は繰り返す運命にある」
- **タレブ『反脆弱性』(2012)**: スタビライザーが介入を不要に見せる構造への警告

## 核心洞察

> **成功した予防は、自分を消す。**
>
> 予防は、その本質において、自分自身の必要性の証拠を消す行為である。なぜなら、防がれた被害は記録されないからである。記録されない便益は、記憶されない。記憶されない便益は、世代をまたがずとも数十年で「他人事」化する。そのとき、コストだけが残る。
>
> ビタミンK、HPV、耐震、はしか、防災 ── 別々のドメインに見える事象は、同じ構造の異なる投影にすぎない。

## メタ観察

本パターンは **daily-writing skill 由来の第2パターン**（surface-axis-occlusion に続く）であり、**atoms 層由来の第4パターン**（algorithmic-abdication, epidemiological-amnesia, mechanization-resistance に続く）。

特徴:
- 単一ニュース（CNN ビタミンK拒否報道, 2026-05-08）への私的違和感を起点とし、atoms 層の既存原子（prevention-invisibility）と新規原子（experiential-amnesia）の合成として演繹
- 3 ドメイン（公衆衛生 / 建築 / セキュリティ）への横断適用で実証
- epidemiological-amnesia との関係性で示されるとおり、atoms-pattern 層の **階層構造（基底パターン / 領域特化パターン）** の初の明示的事例

## 反証条件

以下が観察された場合、本パターンの定義は再検討が必要:

1. prevention-invisibility と experiential-amnesia の両方が観察されている社会で予防拒否が立ち上がらない
2. どちらか一方の atom だけで予防拒否が大規模に立ち上がる（最小構成の修正が必要）
3. 体験者世代の証言の制度化（オーラルヒストリー）で experiential-amnesia が完全に阻止された事例
4. 短期（10年未満）で本パターンが完成する事例（時間スケールの修正が必要）

観察期間：2026-2046年（20年スパン）。
検証対象：
- 米国ビタミンK欠乏性出血症の発生率トレンド
- 日本のHPVワクチン接種勧奨再開後の世代別接種率
- 子宮頸がん死亡率の世代コホート分析（2030年代以降）
- 日本の耐震基準への議論動向
