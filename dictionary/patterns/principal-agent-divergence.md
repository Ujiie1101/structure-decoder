# principal-agent-divergence（本人代理人乖離）

委任者と受任者の利害・情報の不一致が、代理人の機会主義的行動を生む構造。

## 定義

本人（principal）が代理人（agent）に意思決定を委任するとき、両者の目的関数の相違と情報非対称により、代理人が本人の利益ではなく自己利益を追求する。

## 構造

```
principal_delegates_to_agent
  → information_asymmetry (agent knows more)
    → interest_divergence (agent has own goals)
      → agent_shirking_or_opportunism
        → value_destruction_for_principal
```

## 発動条件

1. 利害の不一致
2. 代理人の行動が観察困難
3. 成果の検証困難
4. 契約の不完備性

## 典型的形態

- **Shirking**: 努力の手抜き
- **Empire Building**: 規模拡大への偏向
- **Entrenchment**: 自己保身行動
- **Tunneling**: 資産の私的流用

## 適用事例

| Layer | Entities |
|-------|----------|
| economics | corporate-governance, executive-pay, fund-management |
| politics | bureaucracy, representative-democracy, regulatory-capture |
| history | east-india-company, savings-and-loan-crisis |

## 緩和策

- **インセンティブ調整**: 報酬の業績連動
- **モニタリング**: 監査、取締役会、格付け
- **シグナリング**: 代理人の自己拘束
- **評判メカニズム**: 長期的関係の構築

## 関連パターン

- **information-asymmetry**: 乖離の情報的基盤
- **incentive-distortion**: 調整策の副作用
- **fragility**: 代理連鎖による脆弱性増大

## 注意

「誰かのために」動く者は、「自分のために」動く誘惑に常にさらされる。組織のあらゆる層に潜む構造的緊張。Jensen & Mecklingが1976年に定式化。
