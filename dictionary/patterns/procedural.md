# procedural（手続き的）

定められた手順・儀式・プロセスを通じて、意味・効果・正当性が生成される構造。

## 定義

特定の手順や形式を正しく遂行することで、内容を超えた効果（聖性、法的効力、社会的承認）が発生する。形式が実質を生む。

## 構造

```
prescribed_procedure
  → correct_execution
    → symbolic_transformation
      → efficacy_generation
        → legitimated_outcome (form produces substance)
```

## 発動条件

1. 認知された手順・形式の存在
2. 有資格の執行者
3. 適切な文脈・条件
4. 参加者の信念・承認

## 失敗条件

- **逸脱**: 手順からの離反
- **無資格**: 執行者の正当性欠如
- **文脈外**: 不適切な状況での執行
- **不信**: 参加者の信念崩壊

## 適用事例

| Layer | Entities |
|-------|----------|
| religion | sacrament, ritual, ordination, prayer |
| politics | election, legislation, coronation |
| law | trial, contract-signing, oath |
| economics | audit, certification |

## 関連パターン

- **coordination**: 手続きが協調を可能にする
- **constraint**: 手続きが行動を制約
- **preservation**: 手続きが伝統を保存
