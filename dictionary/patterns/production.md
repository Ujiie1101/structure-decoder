# production（生産）

芸術の創作・流通・消費の様式変化から、新たな芸術概念が出現する構造。

## 定義

芸術作品がどのように作られ、誰によって所有され、どのように流通し消費されるかという生産様式の変化が、芸術の意味・価値・機能自体を変容させる。

## 構造

```
production_mode (creation/distribution/consumption)
  → structural_change
    → value_reconfiguration
      → new_art_concept
        → emergent_paradigm (art_definition)
```

## 発動条件

1. 生産技術・手段の変化
2. 経済システムの変容
3. 流通チャネルの再編
4. 消費者・観客の変化

## 失敗条件

- **商品化**: 芸術価値の市場価値への還元
- **疎外**: 創作者と作品の分離
- **標準化**: 個性の喪失
- **搾取**: 創作者の不当な扱い

## 適用事例

| Layer | Entities |
|-------|----------|
| art | mechanical-reproduction, nft, art-market |
| music | recording-industry, streaming |
| publishing | self-publishing, e-book |
| film | studio-system, independent-film |
| craft | mass-production, maker-movement |

## 関連パターン

- **emergence**: 生産様式から新概念が創発
- **commercialization**: 商品化との関連
- **transformation**: 生産による変容
