# prophecy-gap-synthesis（予言ギャップ統合）

過去の構造予測と現実の乖離から、新たな構造洞察を創発する方法論。

## 定義

ある思想家が「構造的未来」を予測する。時間が経過し、予測の「答え合わせ」が可能になる。予測と現実のギャップを分析すると、予測者が見落とした構造が浮かび上がる。その構造は、別の思想家がすでに指摘していることが多い。両者の統合からAufheben（止揚）が創発する。

## 構造

```
思想家A: 構造的未来を予測
    │
    ▼
時間経過 → 「答え合わせ」可能に
    │
    ├── 予測と一致した構造
    │
    └── 予測と乖離した構造 ← 思想家Bが既に指摘
                │
                ▼
        A + B の統合 → Aufheben（新たな構造洞察）
```

## このパターン自体が構造である理由

1. **思想家は構造を見る** — 各思想家は特定の構造に焦点を当てる
2. **盲点は必然** — 焦点を当てると、別の構造が見えなくなる
3. **時間が盲点を暴く** — 予測の失敗は盲点の存在証明
4. **盲点は既に発見済み** — 別の思想家が別の角度から見ている
5. **統合は創発** — A + B は A でも B でもない第三の洞察

## 発動条件

1. **具体的な予測** — 検証可能な未来像がある
2. **十分な時間経過** — 答え合わせが可能な時点にいる
3. **データの存在** — 予測と現実を比較できる
4. **対立する思想家** — 予測者の盲点を埋める別視点

## 関連パターン

- **self-undermining-success**: 予測の成功が予測を無効化する
- **reflexivity**: 予測が現実を変える
- **hubris**: 予測者の盲点の構造
- **dialectic**: 正→反→合の運動
