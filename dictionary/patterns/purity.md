# purity（純粋性）

異質な要素の排除を通じて集団的同一性が強化される構造。

## 定義

「われわれ」と「彼ら」の境界が相互作用により明確化され、内部の純粋性を維持・強化するための排除メカニズムが創発する。純粋性の基準自体も、競争的ダイナミクスから創発的に先鋭化する。

## 構造

```
group_identity
  → boundary_definition
    → purity_standard_emergence
      → purification_practices
        → reinforced_homogeneity (emergent group solidarity)
```

## 発動条件

1. 集団アイデンティティの脅威認識
2. 純粋性のシグナリング競争
3. 「汚染」への恐怖
4. 逸脱者への制裁メカニズム

## 失敗条件

- **多元主義**: 異質性の積極的価値化
- **混交**: 境界を越えた交流・通婚
- **普遍主義**: 包摂的アイデンティティ
- **外部圧力**: 分裂コストの顕在化
