# rationalization（合理化）

事後的に正当化の論理が構築される構造。

## 定義

行為や決定の後から、それを正当化するための理由・論理・物語が創発的に生成される。個々の正当化要素が相互作用し、整合的なナラティブとして自己組織化する。

## 構造

```
action_or_decision
  → cognitive_dissonance
    → justification_elements
      → narrative_formation
        → rationalized_belief (irreducible post-hoc construction)
```

## 発動条件

1. 行為と信念の不一致（認知的不協和）
2. 自己イメージ維持の欲求
3. 社会的正当性の圧力
4. 説明責任の要求

## 失敗条件

- **透明性**: 動機が明確で隠す必要がない
- **一貫性**: 行為が既存の信念と整合
- **謙虚さ**: 誤りを認める能力
- **外部検証**: 合理化が通用しない環境

## 適用事例

| Layer | Entities |

## 関連パターン

- **memory-erasure**: 合理化が記憶を書き換える
- **suppression**: 合理化できない要素の抑圧
- **emergence**: 合理化も一種の創発
