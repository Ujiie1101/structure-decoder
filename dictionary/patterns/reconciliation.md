# reconciliation（和解）

対立・断絶・罪責の状態から、関係の修復・統合が出現する構造。

## 定義

神と人間、人間と人間、あるいは自己内部の分裂した要素が、贖罪・赦し・対話を通じて和解に達する。断絶を超えた新しい関係性が出現し、より高次の統合が実現する。

## 構造

```
rupture (sin, conflict, alienation)
  → recognition_of_breach
    → mediating_act (sacrifice, forgiveness, dialogue)
      → mutual_acceptance
        → reconciled_relationship (new unity)
```

## 発動条件

1. 断絶・対立の認識
2. 和解への意志
3. 仲介者または仲介的行為
4. 相互の譲歩または赦し

## 失敗条件

- **否認**: 断絶の認識拒否
- **報復循環**: 赦しの不在
- **表面的和解**: 根本原因の未解決
- **非対称性**: 一方的な譲歩のみ
