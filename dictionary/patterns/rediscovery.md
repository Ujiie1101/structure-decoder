# rediscovery（再発見）

忘却された・過小評価された芸術の価値が新たな文脈で認識され、復活する構造。

## 定義

時代の変化により埋もれていた芸術家・作品・様式が、後世の眼差しによって再評価される。再発見は単なる復元ではなく、現代の問題意識を通じた新たな価値の創出であり、過去と現在の創造的対話である。

## 構造

```
forgotten_work (neglected, marginalized)
  → changed_context
    → new_perspective
      → revaluation
        → restoration
          → rediscovery (renewed_significance)
```

## 発動条件

1. 現在の問題意識との共鳴
2. 発見者の存在（学者、批評家、芸術家）
3. 再評価を可能にする制度・メディア
4. 時代精神の変化

## 失敗条件

- **過度の現代化**: 歴史的文脈の無視
- **ロマン化**: 神話的美化
- **利用**: 現代的目的のための歪曲
- **一時的流行**: 持続しない関心
