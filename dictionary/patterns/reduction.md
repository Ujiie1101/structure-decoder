# reduction（還元）

要素を削ぎ落とし本質へ向かうことで、逆説的に新たな表現の豊かさが出現する構造。

## 定義

装飾・複雑性・余剰を意図的に排除し、最小限の要素へ還元することで、各要素の強度が増し、純粋な表現力が顕在化する。「より少なく」が「より多く」を生む逆説。

## 構造

```
complex_expression
  → deliberate_subtraction
    → essential_elements
      → intensification
        → emergent_purity (less_is_more)
```

## 発動条件

1. 削減可能な複雑性の存在
2. 本質と装飾の識別能力
3. ミニマリズム的美学への共感
4. 残された要素の質的強化

## 失敗条件

- **貧困**: 本質まで削ってしまう
- **退屈**: 刺激の不足
- **形式主義**: 還元自体の自己目的化
- **解釈不能**: 手がかりの過剰削減
