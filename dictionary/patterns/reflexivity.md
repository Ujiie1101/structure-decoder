# reflexivity（再帰性）

認識と現実が相互に影響し合い、自己強化的な乖離を生む構造。

## 定義

参加者の認識が現実に影響を与え、その変化した現実が認識をさらに変化させる双方向的フィードバック。均衡ではなく、自己強化的な不均衡プロセスを生む。

## 構造

```
participant_beliefs
  → influence_on_reality
    → changed_reality
      → altered_beliefs
        → further_reality_change
          → boom_or_bust_dynamics
```

## 二つの機能

### 認知機能（Cognitive Function）

- 参加者が現実を理解しようとする
- 常に不完全・偏向（fallibility）

### 操作機能（Manipulative Function）

- 参加者の行動が現実を変える
- 認識が自己実現的になる

## 発動条件

1. 参加者の認識が行動を決定
2. 行動が観察対象（現実）を変化させる
3. フィードバックの存在
4. 誤謬可能性（完全な知識は不可能）

## 関連パターン

- **self-undermining-success**: 再帰的成功による条件変化
- **cascade**: 再帰的フィードバックの連鎖
- **emergence**: 再帰性から創発する秩序・無秩序

## 注意

George Sorosが定式化。主流経済学の均衡概念を批判し、市場を「永続的な不均衡プロセス」として捉える。バブルと崩壊は異常ではなく、再帰性の当然の帰結。
