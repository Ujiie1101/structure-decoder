# regulatory-arbitrage（規制裁定）

規制の境界・差異を利用して、制約を回避しながら利益を得る構造。

## 定義

異なる法域・分類・定義間の規制差異を戦略的に利用し、形式的にはルールに従いながら実質的に規制を回避する。規制の進化より速く行動することで一時的優位を獲得。

## 構造

```
regulatory_difference
  → boundary_identification
    → strategic_positioning
      → rule_circumvention
        → regulatory_response / new_arbitrage
```

## 発動条件

1. 規制間の差異・境界
2. 移動・構造変更の自由度
3. 規制対応の遅延
4. 法的グレーゾーンの存在

## 失敗条件

- **規制収斂**: 差異の解消
- **解釈変更**: グレーゾーンの黒化
- **評判リスク**: 社会的批判
- **制裁**: 遡及適用・罰則強化
