# reinvention（再発明）

既存のものを根本から作り直し、新たな文脈で価値を生み出す構造。

## 定義

過去の発明・概念・制度を、新しい技術・知識・文脈の下で再構築する。単なる改良ではなく、本質的な再設計。「車輪の再発明」は無駄とされるが、意図的再発明は破壊的イノベーションとなる。

## 構造

```
existing_solution
  → fundamental_questioning
    → new_context_application
      → radical_redesign
        → enhanced_value / obsolescence
```

## 発動条件

1. 既存解の限界認識
2. 新技術・新文脈の存在
3. 根本からの再設計意志
4. 既存資産への非依存

## 失敗条件

- **無意味な重複**: 既存解で十分
- **文脈無視**: 新文脈への不適合
- **過去の軽視**: 先人の知恵の無駄
- **移行コスト**: 採用障壁の過大

## 適用事例

| Layer | Entities |
|-------|----------|
| technology | iphone, tesla, aws, netflix |
| economics | fintech, sharing-economy |
| history | meiji-restoration, renaissance |
| philosophy | paradigm-shift, re-interpretation |
| art | neo-classicism, sampling, remix |

## 関連パターン

- **emergence**: 再発明からの新価値創発
- **disruption**: 既存秩序の破壊
- **transformation**: 根本的変革
- **substitution**: 旧から新への置換
