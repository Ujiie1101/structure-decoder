# rejection（拒絶）

既存の規範・伝統・権威を意図的に否定することで、新たな表現領域が出現する構造。

## 定義

過去の様式、技法、価値観、制度を明示的に拒否し、その否定を通じて新しい芸術的可能性を切り開く。アヴァンギャルドの根本動因であり、断絶によって創造空間を生成する。

## 構造

```
established_norm (convention/tradition)
  → conscious_negation
    → transgression
      → void_creation
        → new_possibility (emergent_space)
```

## 発動条件

1. 権威的な既存秩序の存在
2. 否定の対象の明確な認識
3. 代替ビジョンまたは破壊的エネルギー
4. 支持者・共鳴者の存在

## 失敗条件

- **単なる反抗**: 建設的ビジョンの欠如
- **無視**: 拒絶自体が無視される
- **回収**: 体制に取り込まれる
- **自己目的化**: 拒絶のための拒絶

## 適用事例

| Layer | Entities |
|-------|----------|
| art | dada, punk, anti-art, marcel-duchamp |
| music | noise-music, free-jazz, industrial |
| literature | beat-generation, theatre-of-the-absurd |
| philosophy | nihilism, deconstruction |
| politics | anarchism, counter-culture |

## 関連パターン

- **emergence**: 拒絶から新空間が創発
- **transformation**: 否定による変容
- **disruption**: 既存秩序の破壊
