# replication（複製）

反復的再生産から、オリジナルを超えた新たな意味が出現する構造。

## 定義

作品や形式の複製行為を通じて、単なるコピーを超えた新たな意味、文脈、価値が創発する。複製は劣化ではなく、変異と増殖による意味の拡張である。

## 構造

```
original
  → reproduction_process
    → multiple_copies
      → contextual_variation
        → meaning_emergence (transcends original)
```

## 発動条件

1. 再生産可能な形式
2. 複製技術の存在
3. 文脈の多様化
4. 受容の多元化

## 失敗条件

- **劣化**: 質的低下のみの複製
- **固定**: 文脈依存性の消失
- **飽和**: 過剰複製による価値喪失
- **権威依存**: オリジナルへの従属

## 適用事例

| Layer | Entities |
|-------|----------|
| art | appropriation-art, meme, remix, sampling |
| biology | dna-replication, viral-spread |
| technology | digital-copy, version-control |
| philosophy | simulacrum, baudrillard |
| media | mass-reproduction, viral-content |

## 関連パターン

- **emergence**: 複製から新意味が創発
- **amplification**: 複製による拡散
- **transformation**: 複製過程での変容
