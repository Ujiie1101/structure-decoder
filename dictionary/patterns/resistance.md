# resistance（抵抗）

変化・支配・介入に対抗し、現状維持または反転を図る構造。

## 定義

外部からの圧力や変化の力に対して、個人・集団・システムが押し返す動き。受動的抵抗（不服従）から能動的抵抗（反乱）まで含む。

## 構造

```
external_pressure
  → threat_perception
    → mobilization
      → counter_action
        → friction/opposition (change impeded or reversed)
```

## 発動条件

1. 変化・圧力の認知
2. 脅威としての解釈
3. 抵抗資源（人、金、正当性）
4. 組織化・動員能力

## 失敗条件

- **圧倒**: 抵抗力を上回る圧力
- **分断**: 抵抗勢力の内部対立
- **適応**: 抵抗対象の柔軟対応
- **疲弊**: 持続力の枯渇

## 適用事例

| Layer | Entities |
|-------|----------|
| politics | civil-disobedience, opposition, revolution |
| technology | luddism, privacy-advocacy, decentralization |
| economics | labor-movement, protectionism, strike |
| religion | martyrdom, persecution-survival |

## 関連パターン

- **preservation**: 抵抗の目的としての保存
- **constraint**: 抵抗が変化を制約
- **transformation**: 抵抗が変革を誘発
