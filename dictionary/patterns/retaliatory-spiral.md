# 報復の螺旋パターン

## 構造式

```
A_attack → B_retaliate → A_escalate → ... → mutual_exhaustion
    ↓           ↓            ↓                    ↓
  攻撃        報復      エスカレーション         両者消耗
  （合理的）  （合理的）  （合理的）           （最悪の結果）
```

## 核心

**個人として正しい選択が、全体として最悪の結果を生む。**

ゲーム理論における「囚人のジレンマ」の動的版。
各当事者にとって「報復しない」は損になる。
だが、全員が報復すると、全員が損をする。

## 構造の要素

### 1. 初期攻撃（Initial Attack）

- 「守る」意図で始まる
- 短期的には効果があるように見える
- 例：関税、禁輸令、制裁

### 2. 報復（Retaliation）

- 「防御」として正当化される
- 同等または上回る強度
- 例：報復関税、対抗制裁

### 3. エスカレーション（Escalation）

- 各ステップで強度が増す
- 撤退は「敗北」に見える
- 政治的インセンティブが撤退を阻む

### 4. 両者消耗（Mutual Exhaustion）

- 勝者なき終結
- 当初守ろうとしたものも失われる
- 第三者の台頭を許す

## なぜ繰り返されるか

### 1. 短期と長期の乖離

- 関税は短期的に「効く」ように見える
- 長期的コストは見えにくい
- 政治家の任期は長期的結果より短い

### 2. 報復の非対称的認知

- 自分の行動 = 「防御」
- 相手の報復 = 「攻撃」
- 同じことをしているのに認知が異なる

### 3. 撤退のコスト

- 撤退 = 「敗北」「弱腰」
- エスカレーション = 「強い指導者」
- 政治的インセンティブが撤退を阻む

### 4. 学習の困難

- 歴史の教訓は「別の時代の話」
- 「今回は違う」という確信
- 構造的類似性への盲目

## 警告サイン

1. **「守る」という言葉** - 守るための攻撃は螺旋の始まり
2. **「相手が先に」という主張** - 報復の正当化は螺旋の加速
3. **「勝つまで続ける」という決意** - 出口なき螺旋への突入
4. **第三者の不在** - 仲裁者なき対立は終わらない

## 脱出方法（理論的）

1. **相互依存の認識** - 「勝者なき構造」の可視化
2. **コミットメント問題の解決** - 信頼できる撤退メカニズム
3. **第三者の介入** - 中立的仲裁者の設定
4. **反復ゲーム化** - 単発ではなく長期関係として認識

## 哲学的接続

### ヘラクレイトス

> 「戦いは万物の父である」

対立から調和が生まれる可能性。
だが、対立が調和ではなく消耗を生むとき、螺旋は下降する。

### ゲーム理論

> 「個人合理性と集団合理性の乖離」

各プレイヤーの最適戦略が、全体として最悪の結果を生む。
これが囚人のジレンマの本質であり、報復の螺旋の構造でもある。

## 現代的教訓

- **関税ニュースを見たら**: 「これは囚人のジレンマ構造か？」と問う
- **報復の連鎖**: どこまで続くか、誰が最初に降りるか
- **「勝者」の存在**: 螺旋の中に勝者はいるか、全員が敗者か
- **第三者の利益**: 螺旋の外で誰が得をしているか
