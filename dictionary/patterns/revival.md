# revival（復興）

衰退・忘却された文化・技術・思想が再発見され、新たな文脈で復活する構造。

## 定義

過去に存在し一度は失われた、または周縁化された要素が、現代の必要性や関心に応じて再評価・再活性化される。単なる復古ではなく、現代的な解釈・適応を伴う。

## 構造

```
dormant_tradition
  → rediscovery (academic/artistic/popular)
    → recontextualization
      → contemporary_adaptation
        → renewed_vitality + selective_memory
```

## 発動条件

1. 過去の遺産へのアクセス
2. 現代における関連性・需要
3. 復興の担い手の存在
4. 正当化する言説の構築

## 失敗条件

- **断絶**: 継承経路の完全消失
- **無関係**: 現代ニーズとの乖離
- **批判**: 復興対象への否定的評価
- **商業化過剰**: 本質の希薄化

- **authority**: 復興が伝統の権威を借用
- **transformation**: 復興に伴う変容
