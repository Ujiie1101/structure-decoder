# scale-effects（規模効果）

規模の拡大に伴い、質的に異なる現象・効率・問題が創発する構造。

## 定義

組織・システム・市場が一定の規模を超えると、単なる量的拡大ではなく質的変化が生じる。規模の経済（コスト逓減）、規模の不経済（官僚化・調整困難）、スケールの法則（表面積対体積比）など、規模自体が新しい現象を生み出す。

## 構造

```
growth (quantitative)
  → threshold_crossing
    → qualitative_change
      → emergent_properties (efficiency or dysfunction)
        → scale_dynamics (new rules apply)
```

## 発動条件

1. 量的拡大の継続
2. 閾値・臨界点の存在
3. 非線形な関係性
4. フィードバックループ

## 失敗条件

- **スケール盲目**: 規模変化に伴う質的変化の無視
- **過大規模**: 規模の不経済を超えた拡大
- **早期の最適化**: 規模到達前の効率追求
- **ガバナンス遅延**: 組織構造の適応不全
