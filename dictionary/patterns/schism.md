# schism（分裂）

統一された信仰・組織・運動が、教義・権威・アイデンティティをめぐる対立から分裂する構造。

## 定義

共通の基盤を持つ集団が、解釈の相違・権威の争奪・アイデンティティの分化により、相互に排他的な複数の集団へと分裂する。分裂は対立の蓄積から創発する。

## 構造

```
unified_community
  → interpretive_divergence
    → factional_formation
      → authority_dispute
        → mutual_exclusion
          → formal_schism (emergent separation)
```

## 発動条件

1. 教義・理念の解釈相違
2. 権威・正統性の争奪
3. 指導者の対立
4. 外部圧力による亀裂顕在化

## 失敗条件

- **教義的柔軟性**: 解釈の幅の許容
- **権威の明確化**: 正統性の一元化
- **調停機構**: 紛争解決メカニズム
- **共通の敵**: 外敵による結束
