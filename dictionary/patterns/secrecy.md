# secrecy（秘匿）

情報の意図的な隠蔽が、権力・優位性・神秘性を生み出し、独自の動態を形成する構造。

## 定義

知識・情報・意図を選択的に隠すことで、情報非対称性が生まれ、秘匿する側に権力・優位性・正統性が付与される。秘密それ自体が価値と影響力の源泉となる。

## 構造

```
information_possession
  → selective_concealment
    → asymmetry_creation
      → power_differential
        → emergent_authority (from secrecy itself)
```

## 発動条件

1. 価値ある情報の保有
2. アクセス制限の仕組み
3. 秘密の存在認知（完全な無知との区別）
4. 漏洩への制裁メカニズム

## 失敗条件

- **漏洩**: 秘密の暴露で優位性消失
- **無関心**: 秘密への関心が失われる
- **過剰秘匿**: 正当性への疑念が増大
- **技術変化**: 秘匿手段の無効化
