# self-curation（自己キュレーション）

自己を素材として、複数の役割ステージを並列に配置・編集・運用する第三項（観察者＝舞台監督）を持つ自己関係。観察者は各ステージを管理し整合性を保つが、観察者自身を観察できないため、その盲点が "整合性" の代償として隠される構造。

## 定義

ゴッフマンの2層モデル（フロントステージ／バックステージ）に **第三項としての観察者（舞台監督）** が加わった3層構造。観察者は：

1. 各ステージのパフォーマー（家庭の自分／浮気の自分／仕事の自分など）を **管理する**
2. ステージ間の整合性を **設計し配置する**
3. 罪悪感を **カテゴリ別に分配する**
4. 自分自身を **観察できない**（再帰観察の構造的不可能性）

`identity-fork`（2層分岐）でも、`mission-drift`（漂流）でもない第三領域：**並列管理労働が個人内に内部化されたとき発生する、監査不在の自己経営**。

## 公式

```
self-curation
  = frontstage-backstage  （文脈ごとの演技の必然性）
  × third-observer        （並列を管理する第三項の登場）
  × category-management   （関係を「枠」として運用する操作）
  × selective-blindness   （観察者の自己盲点が整合性を担保）
```

4つすべて揃ったとき発動。1つでも欠ければ self-curation は成立せず、別の構造に解体される。

## 発動条件

1. **複数ステージの並走必然性**：単一役割では生活が成立しない（家庭+仕事+恋愛+SNSなど）
2. **並列管理コストの個人化**：管理を委譲する制度や共同体が機能していない
3. **明示的カテゴリ言語の流通**：「○○枠」「○○モード」など運用語彙が手元にある
4. **整合性圧力の存在**：「ブレない自分」「一貫性」を要求する文化的圧力

4つ揃わない場合の停止形態：
- frontstage-backstage 欠如 → 単一自己（前近代的、または病理的）
- third-observer 欠如 → 単純な分裂（解離・dissoziation）
- category-management 欠如 → 暗黙的二重生活（伝統的浮気）
- selective-blindness 欠如 → 統合不可能な認知不協和、神経症化

## 失敗条件（self-curation が解体する条件）

### 個別原子への介入

- **frontstage-backstage**：すべてのステージを統合する一つの場（家族・修道院・閉鎖的共同体）
- **third-observer**：観察者を観察する第四項の導入（セラピー、宗教的告白、信頼できる他者）
- **category-management**：「枠」言語を捨てる、関係を運用しない
- **selective-blindness**：観察者の盲点を意識化する（瞑想、自己分析、外部からの異議）

### 組み合わせ介入（最強）

**third-observer × selective-blindness への同時介入** = 「観察者を観察する第四項」を信頼できる他者として外部化する。これが伝統的に告解・カウンセリング・親友が果たしてきた機能。第三項を孤立させない。

## 判別法

1. 「○○枠」「○○モード」のような運用語彙が日常的に使われている
2. 「私は私、○○は○○」型の発言で各ステージを分離している
3. 整合性の保持に労働感（疲弊・スケジュール管理）が伴う
4. 観察者自身の状態（疲弊／虚無／万能感）が言語化されていない

4つ揃うと self-curation が作動している。

## 適用事例

| 領域 | 具体例 | 並走するステージ | 第三項の現れ方 |
|------|--------|----------------|--------------|
| 恋愛 | 「本命枠／セフレ枠／キープ枠」 | 家庭・浮気・観察 | 「私は私、家庭は家庭」 |
| 労働 | 副業時代のキャリア運用 | 本業・副業・キャリア管理 | 「キャリアの自分」 |
| 育児 | ワーママの並列運用 | 母・職業人・自己 | 「人生のCEO」言説 |
| SNS | 媒体別自己呈示の分裂 | Instagram・X・LinkedIn | 「中の人」「運用」 |
| 学習 | 司法書士試験＋本業＋家庭 | 学習者・職業人・家族 | 「人生設計」する自分 |

## 倫理の変質

self-curation 下では倫理の構造が変質する：
- **行為倫理（伝統）**：「○○してはいけない」（行為そのものへの罪）
- **整合性倫理（self-curation）**：「ステージ間の整合性を保たねばならない」（運用責任）

罪は **行為** から **運用ミス** に変質する。この変質が "本命枠" 言説の道徳的中性化を可能にする。

## 関連メカニズム

- 観察者の盲点：`triple-blindness`（家庭の疲弊／浮気の虚無／観察者の万能感錯覚）
- 監査不在：`auditless-self-management`（自分が経営する組織の決算書を自分で書き、自分で監査する）
- 再帰観察の失敗：`recursive-observer-failure`（観察者を観察するには第四項が要り、無限後退する）

## 並走構造としての位置づけ

self-curation は典型的な **並走（同次元・同時間軸の別世界の共存）** パターン：
- 家庭の自分・浮気の自分・観察する自分は **統合されない**
- アウフヘーベン的処方（「整合性のある自己を取り戻せ」）は **失敗する**
- 治癒は統合ではなく、**観察者の盲点を第四項で外部化すること**

これは Knowledge OS の並走原則と完全に整合する。

## 検証ポイント

- 「枠」言語の流通頻度の歴史的増加（2000年代以降の言説変化）
- マッチングアプリ・SNS時代における self-curation の加速可能性
- セラピー文化の同時拡大は「第四項の外部化需要」として読めるか
