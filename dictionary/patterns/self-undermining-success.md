# self-undermining-success（自己破壊的成功）

成功そのものが、その成功を可能にした条件を破壊する構造。

## 定義

ある戦略・制度・行動が成功することで、その成功を支えていた前提条件が変化・消滅し、やがて同じ戦略が機能しなくなる、あるいは逆効果を生む。

## 構造

```
successful_strategy
  → changes_in_environment
    → erosion_of_original_conditions
      → strategy_becomes_ineffective
        → failure_from_success
```

## 発動条件

1. 成功が環境を変化させる
2. 成功の前提条件が環境依存的
3. 成功の模倣・拡散が起こる
4. フィードバックの認識遅延

## 失敗条件

- **環境不変**: 成功が環境に影響を与えない
- **適応能力**: 条件変化に応じて戦略を修正できる
- **限定的成功**: 影響範囲が小さく環境変化が起きない

## 適用事例

| Layer | Entities |
|-------|----------|
| economics | quantitative-easing, low-interest-rate-policy |
| technology | platform-dominance, network-effects |
| history | pax-romana, british-empire |
| philosophy | performative-contradiction |

## 関連パターン

- **hubris-nemesis**: 成功による傲慢が破滅を招く
- **reflexivity**: 成功が市場を変え、その変化が成功を無効化
- **fragility**: 成功による最適化が脆弱性を生む

## 注意

「何かを得るために行動すると、まさにその行動によって、得ようとしたものを失う」という構造。勝利条件の自己消滅。
