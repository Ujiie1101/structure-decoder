# serendipity（偶然の発見）

意図せぬ出会い・失敗・逸脱から、予期しない価値ある発見が生まれる構造。

## 定義

目的とは異なる方向からの偶発的な遭遇が、準備された心（sagacity）と結びつくことで、当初の意図を超えた革新的発見・創造をもたらす。偶然は必要だが十分ではない。

## 構造

```
purposeful_activity
  → unexpected_encounter
    → anomaly_recognition
      → prepared_mind_connection
        → valuable_discovery (unintended by design)
```

## 発動条件

1. 能動的な探求・活動
2. 開放性と好奇心
3. 異常を認識する知識基盤
4. 偶然を活かす柔軟性

## 失敗条件

- **計画固執**: 予定外の発見の無視
- **無知**: 偶然の意味を認識できない
- **孤立**: 異分野との接触機会の欠如
- **効率偏重**: 寄り道・逸脱の排除

## 適用事例

| Layer | Entities |
|-------|----------|
| technology | penicillin, post-it-notes, microwave-oven |
| art | action-painting, found-art |

| economics | pivot-success, market-discovery |

## 関連パターン

- **discovery**: 偶然の発見は発見の一形態
- **emergence**: 偶然から新価値が創発
- **polymath**: 博学が偶然を活かす基盤
