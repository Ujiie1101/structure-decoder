# seriality（連続性）

反復・連続の中から、個々の要素を超えたリズム・パターン・意味が創発する構造。

## 定義

類似した要素の連続的配置・反復から、単体では存在しない時間性、リズム、累積効果、ナラティブが出現する。

## 構造

```
individual_units
  → sequential_arrangement
    → rhythm_pattern
      → cumulative_effect
        → emergent_narrative (transcends units)
```

## 発動条件

1. 反復可能な単位の存在
2. 配列・順序の設計
3. 変奏と統一のバランス
4. 受容者の時間的関与

## 失敗条件

- **単調**: 変化なき反復による退屈
- **断絶**: 連続性の破綻
- **過剰変化**: 統一性喪失
- **短絡**: 累積効果が生じる前の中断

## 適用事例

| Layer | Entities |
|-------|----------|
| art | serial-music, pop-art, episodic-narrative |
| technology | assembly-line, software-iteration |
| economics | subscription-model, compound-interest |
| philosophy | eternal-return, process-philosophy |
| religion | liturgical-calendar, ritual-repetition |
| history | periodization, cyclical-history |

## 関連パターン

- **emergence**: 連続から意味が創発
- **cyclical**: 連続性の循環形態
- **amplification**: 反復による増幅

## メカニズムID

**[E-SER] Serial Emergence** — 連続・反復から累積的意味が創発
