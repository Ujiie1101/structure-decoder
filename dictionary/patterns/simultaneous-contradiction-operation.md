# simultaneous-contradiction-operation（矛盾の同時運用）

書き手・話し手が**互いに矛盾する複数の命題を同時に主張し、状況に応じて使い分ける**装置。矛盾は「解消すべきエラー」ではなく「両方を保持するための資源」として運用される。

## 定義

以下の三条件を満たす:

1. **同時性の主張**: 矛盾する命題A・Bが同一発話・同一文書内で同時に呈示される
2. **使い分けの可動性**: 状況によってAかBか、または両方が引用可能
3. **解消の保留**: 当事者は矛盾を意識しつつ解消を意図しない

## 構造

```
発話A（命題甲）
  ＋（同時）
発話B（命題乙、甲と矛盾）
  ↓
受け手の解釈空間が二重化される
  ↓
状況変化に応じて甲か乙が引用される
  ↓
両方が「書かれていた」事実は不可逆に残存する
```

## 観察可能な現代具体例

- 政治家の演説における対立する有権者層への同時メッセージング
- 企業の広報で「成長戦略」と「持続可能性」の同時主張
- 恋愛における「自由でいたい」と「束縛してほしい」の同時表明
- 経営者の「変化を恐れない」と「伝統を守る」の同時運用
- 教育現場での「個性を伸ばす」と「規律を守る」の並列宣言

## 対抗装置

- 矛盾命題の対比的同時呈示（「ここで矛盾している」の指摘）
- 命題の優先順位の強制（「Aを取るならBを下ろせ」）
- 時間軸への矛盾の分散（「いまはA、次はB」の段階化）
