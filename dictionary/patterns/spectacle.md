# spectacle（スペクタクル）

圧倒的な視覚体験・規模・演出から、日常を超えた集合的陶酔が出現する構造。

## 定義

壮大なスケール、華麗な演出、圧倒的な視覚効果によって、観客を日常的認識から引き離し、共同的な驚嘆・興奮・没入状態を生み出す。見せ物としての芸術の極致。

## 構造

```
spectacular_display (scale/effect)
  → sensory_overwhelm
    → suspension_of_ordinary
      → collective_immersion
        → emergent_ecstasy (shared_transcendence)
```

## 発動条件

1. 圧倒的な規模または技術
2. 共同視聴の場
3. 日常との明確な断絶
4. 感覚的インパクト

## 失敗条件

- **麻痺**: 過剰刺激による無感覚
- **空虚**: 内容なきスペクタクル
- **操作**: 権力によるプロパガンダ化
- **商品化**: 消費対象への堕落
