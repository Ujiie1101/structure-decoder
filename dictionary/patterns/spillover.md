# spillover（波及）

ある領域での効果が意図せず他領域へと広がる構造。

## 定義

特定の文脈で生じた影響・技術・知識が、境界を越えて他の領域へと波及し、予期せぬ効果をもたらす。外部性、副次効果を含む。

## 構造

```
origin_domain
  → boundary_crossing
    → adjacent_domain_impact
      → secondary_spillover
        → systemic_diffusion
```

## 発動条件

1. 効果の移転可能性
2. 領域間の接続性
3. 受容側の適応能力
4. 媒介メカニズム

## 失敗条件

- **封じ込め**: 効果の領域内限定
- **非互換**: 受容側との不適合
- **減衰**: 伝播過程での希薄化
- **障壁**: 制度的・物理的遮断

## 適用事例

| Layer | Entities |
|-------|----------|
| technology | military-to-civilian, space-tech-spinoff |
| economics | knowledge-spillover, technology-transfer |
| politics | policy-diffusion |
| science | cross-disciplinary-impact |

## 関連パターン

- **cascade**: 波及が連鎖に発展
- **network**: 波及の経路構造
- **emergence**: 波及からの新秩序創発
