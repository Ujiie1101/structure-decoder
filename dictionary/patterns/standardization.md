# standardization（標準化）

多様な形式・手法・規格が、共通の基準へと収斂する構造。

## 定義

異なる主体・地域・時代で別々に発展してきた方法やフォーマットが、相互運用性・効率性・品質保証のために統一される。多様性と効率性のトレードオフを内包する。

## 構造

```
diverse_practices
  → coordination_need
    → standard_proposal
      → adoption_diffusion
        → interoperability + homogenization
```

## 発動条件

1. 相互運用性への需要
2. 標準設定主体の存在
3. 採用のインセンティブ
4. 切り替えコストの許容

## 失敗条件

- **標準化戦争**: 競合規格の並立
- **過剰標準化**: イノベーション阻害
- **不適合**: 標準が現実に合わない
- **政治化**: 標準が権力闘争の道具化

## 適用事例

| Layer | Entities |
|-------|----------|
| art | musical-notation, film-format |
| technology | protocol, file-format |
| economics | accounting-standard, currency |
| politics | legal-code, diplomatic-protocol |
| religion | liturgy, canon |

## 関連パターン

- **network-effects**: 標準化がネットワーク効果を強化
- **institutionalization**: 標準の制度化
- **commercialization**: 標準化による市場拡大
