# stasis（停滞）

システムが変化の契機を失い、自己保存のみに向かう構造。

## 定義

変革への圧力と現状維持の力が均衡し、システムが動的平衡を失って固着する。個々のアクターの合理的選択が、全体としての停滞を創発させる。

## 構造

```
initial_dynamism
  → stabilization_pressures
    → vested_interests
      → change_resistance
        → innovation_penalty
          → systemic_stasis (emergent inertia)
```

## 発動条件

1. 既得権益層の形成
2. 変革リスクの内部化
3. 成功体験への執着
4. フィードバックループの閉塞

## 失敗条件

- **外圧**: 外部からの強制的変化
- **危機**: 現状維持が不可能になる事態
- **世代交代**: 既得権益層の退場
- **技術革新**: 抵抗不可能な変化

## 適用事例

| Layer | Entities |
|-------|----------|
| history | tokugawa-isolation, byzantine-decline, late-qing |

## 関連パターン

- **emergence**: 停滞からの新秩序創発
- **disruption**: 停滞を破壊する力
- **erosion**: 停滞が長期化すると侵食へ
