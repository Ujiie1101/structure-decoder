# Structure-Emotion Spread Pattern（構造×感情拡散パターン）

## 定義
構造的可視化と普遍的感情を組み合わせることで、「なるほど」（構造のみ）や「共感」（感情のみ）を超えた「誰かに伝えたい」という拡散衝動を生むパターン。

## 公式
```
構造のみ → 「なるほど」（理解で完結）
感情のみ → 「共感」（個人で完結）
構造 × 感情 → 「誰かに伝えたい」（拡散へ）
```

## メカニズム

### 1. 構造が「言語化できなかったモヤモヤ」を可視化
- 読者は「そういうことだったのか」と発見する
- 2×2マトリックスなど視覚的フレームワークが効果的

### 2. 感情が「自分ごと」にする
- 普遍的感情（恐怖、怒り、希望、悲しみ）に触れる
- 読者は自分がその構造のどこにいるか考える

### 3. 組み合わせが「言いたいけど言えなかったこと」を代弁
- 読者は「これを言ってくれた」と感じる
- シェアすることで自分の立場を表明できる

## 構成要素

### タイトル
- 三段対比構造（A｜B｜C）
- 具体的数字（99万人、11億ドルなど）
- 物語性を示唆

### 冒頭
- クイズ形式で引き込み
- 期待を裏切る展開（「正解はC」）

### 事例積み上げ
- 5件以上の類似事例
- 「特殊事例ではない」証明
- 時系列配置で説得力強化

### 構造フレーム
- 2×2マトリックス
- 各象限に名前をつける
- 読者が自分の位置を特定できる

### 独自造語
- 新概念にラベル付け
- 英語併記で権威付け（例: output-less-critique）

### 身近な比喩
- 抽象構造を日常例に落とす
- 「違いは○○だけ」で一言結論

### 感情的着地
- 怒りや批判で終わらない
- 希望や問いかけで終わる
- 権威ある引用で補強

## 事例

### 成功事例: Day25記事（スキ11）
- **構造**: 2×2マトリックス（継続意志×制裁強度）
- **感情**: 「批判される恐怖」「沈黙の正規化」
- **着地**: 「願いは批判より長く残る」（希望）

### 類似パターン
- ウイルス性コンテンツの拡散（cascade）
- 注意経済（attention）
- 増幅効果（amplification）

## 適用条件
- 多くの人が感じているが言語化できていない問題
- 構造として可視化可能
- 読者自身がその構造内に位置づけられる

## 注意点
- 構造だけに偏ると「頭でっかち」に
- 感情だけに偏ると「お気持ち」に
- バランスが重要
