# struggle（闘争）

対立・葛藤の持続から、新たな秩序・アイデンティティ・価値が創発する構造。

## 定義

相反する力の衝突と持続的緊張から、単なる勝敗ではなく、両者を超えた新しい状態、認識、社会関係が出現する。

## 構造

```
opposing_forces
  → sustained_conflict
    → mutual_transformation
      → dialectical_tension
        → emergent_synthesis (transcends opposition)
```

## 発動条件

1. 対立する利害・価値の存在
2. 闘争を持続させる資源
3. 完全な殲滅の不可能性
4. 相互作用を通じた学習

## 失敗条件

- **殲滅**: 一方の完全消滅
- **回避**: 対立の抑圧
- **消耗**: 創造なき破壊
- **固着**: 創発なき膠着
