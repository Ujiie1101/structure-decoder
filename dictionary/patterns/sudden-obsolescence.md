# sudden-obsolescence（一撃淘汰）

段階的な衰退ではなく、ある日突然まとめて消える構造。

## 定義

技術・市場・制度の変化が閾値を超えた瞬間に、それまで価値を持っていたものが一斉に無価値化する現象。「中途半端な専門性は一撃で淘汰される」（南場智子）。

## 構造

```
gradual_accumulation_of_change
  → threshold_approach (invisible)
    → tipping_point_crossed
      → sudden_mass_obsolescence
        → no_time_for_adaptation
```

## 発動条件

1. 代替技術・手法の成熟
2. 閾値に近づいていることの不可視性
3. 「自分は大丈夫」という油断
4. 速度の非対称性（変化 >> 適応）

## 失敗条件（生存条件）

- **速度（ベロシティ）**: 変化より速く動く
- **深さ×速さの両立**: どちらか一方では不十分
- **閾値の監視**: 変化の累積を定量的に追跡
- **カニバリゼーション**: 自ら既存事業を破壊

## 適用事例

| Layer | Entities |
|-------|----------|
| economics | ai-job-displacement, retail-apocalypse |
| technology | film-to-digital, horse-to-automobile |
| history | empire-collapse（ローマ、ソ連） |
| philosophy | paradigm-shift（クーン） |

## 関連パターン

- **disruption**: 漸進的な置き換え（一撃淘汰はより急激）
- **collapse**: システム全体の崩壊
- **tipping-point**: 閾値の概念
