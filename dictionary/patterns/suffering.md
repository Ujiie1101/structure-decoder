# suffering（苦悩）

苦痛・葛藤・絶望が創造の原動力となり、深遠な芸術作品が生み出される構造。

## 定義

芸術家の精神的・肉体的苦悩が、安易な表現を拒否し、本質的な問いと対峙させることで、深い共感と普遍性を持つ作品を創発させる。苦悩は単なる素材ではなく、変容のるつぼとなる。

## 構造

```
personal_crisis (illness, loss, alienation)
  → confrontation_with_suffering
    → creative_struggle
      → sublimation
        → profound_expression
          → universal_resonance
```

## 発動条件

1. 深刻な内的・外的苦悩
2. 苦悩と対峙する勇気
3. 表現への衝動
4. 昇華の能力

## 失敗条件

- **圧倒**: 苦悩に潰される
- **感傷**: 自己憐憫への堕落
- **搾取**: 苦悩の商品化
- **反復**: トラウマの再演に留まる
