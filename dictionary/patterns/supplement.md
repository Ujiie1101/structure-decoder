# supplement（補遺）

外部からの追加が、内部の欠如を暴露し、本質と副次の階層を転覆する構造。

## 定義

「本来のもの」に後から加えられた「補足」が、実は「本来のもの」の不完全性を示している。補遺は単なる付加ではなく、欠如の代補であり、起源の純粋性を汚染する。

## 構造

```
origin_claimed_as_complete
  → supplement_added_as_extra
    → supplement_reveals_lack_in_origin
      → origin_was_never_self-sufficient
        → hierarchy_of_primary/secondary_collapses
```

## 補遺の二重論理

### 付加（Addition）

- 完全なものに追加される余剰
- 外部的、副次的、派生的

### 代補（Substitution）

- 欠如を埋める必要な要素
- 実は不可欠、起源を補完

→ 両者は矛盾するが同時に成立

## Rousseauの事例

- **自然/文化**: 文化は自然の補遺（だが自然は常に既に文化化）
- **話し言葉/書き言葉**: 書は話の補遺（だが話も既に書の構造を持つ）
- **自慰**: 性交の補遺（だが性交も既に代補的）

## 関連パターン

- **differance**: 補遺性を生む差延の運動
- **trace**: 補遺として機能する痕跡
- **substitution**: 補遺による置換

## 注意

「補足説明」「付録」「追記」と訳すと見落とす。Derridaの「supplement」は、西洋形而上学の「起源の優位」「現前の特権」を内部から崩壊させる爆薬。単純な二項対立を不可能にする。
