# suppression（抑圧）

不都合な要素が意識・公共空間から排除される構造。

## 定義

システム維持に反する情報・意見・記憶が、分散的なメカニズムによって抑え込まれる。明示的な禁止だけでなく、自己検閲・沈黙の螺旋・タブー化など、相互作用から創発する抑圧を含む。

## 構造

```
threatening_element
  → detection_mechanisms
    → distributed_suppression_actions
      → marginalization
        → invisible_absence (emergent silence)
```

## 発動条件

1. 支配的ナラティブへの脅威
2. 権力の非対称性
3. 制裁への恐怖（実際または想像）
4. 同調圧力

## 失敗条件

- **多元性**: 抑圧を逃れる複数の表現経路
- **外部観察者**: 抑圧を可視化する第三者
- **臨界点超過**: 抑圧コストが維持不能
- **技術変化**: 新たな伝播手段
