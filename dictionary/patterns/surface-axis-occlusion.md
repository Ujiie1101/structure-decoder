# surface-axis-occlusion（表層軸の覆い）

流通する比較フレームほど、本当は重要な軸を覆い隠す現象。比較記事は「並べやすさ」で軸を選ぶため、業務文脈に踏み込まないと並ばない軸は記事から消える。結果、流通している比較フレームに従うと、最も決定的な軸を欠落させた意思決定になる。

## 定義

ある領域における比較フレームが流通するとき、流通選択は「比較容易性」（数字化／感覚化が容易か）に駆動される。文脈依存的な軸（業務・契約・組織構造に踏み込まないと並ばない軸）は流通フレームから脱落する。読者は流通フレーム内で意思決定するため、最も決定的な軸を見落とす。

「コスパと質」が正しい軸だから流通しているのではない。**比較しやすい軸だから**流通している。

## 発動条件

1. **比較フレーム流通**: ある領域に、比較記事・ベンチマーク・○○選などの比較フレームが流通している
2. **比較容易性バイアス**: 流通フレームに含まれる軸は、すべて数字化／感覚化が容易な軸である
3. **文脈依存軸の存在**: 業務文脈・契約構造・組織制約に踏み込まないと並ばない軸が、領域内に存在する
4. **読者の文脈忘却**: 読者が比較フレームを採用するとき、自分の業務文脈を投入せず「どっちがいい？」と一般化する

4つ揃うと、流通フレームが意思決定を覆う。

## 適用事例

### 動画生成ツール選定（2026年）

| 軸 | 流通頻度 | 業務文脈依存度 |
|---|---|---|
| コスト | 高（数字で並ぶ） | 低 |
| 品質 | 高（感覚で並ぶ） | 中 |
| 量産性（programmability） | 低 | 高（量産業務を持っているか） |
| 独立性（ownership-vs-access） | 低 | 高（コンプライアンス要件を持っているか） |

「コスパと質」だけで HyperFrames vs Claude Design を比較すると、量産業務やコンプライアンス要件を持つ事業者の意思決定が誤る。

### 横展開候補

- **人材評価**: 「学歴」「職歴」が流通／「組織との相性」「未踏領域での意思決定特性」が業務文脈軸
- **物件評価**: 「価格」「築年数」「立地」が流通／「再販時の価格訴求性」「リフォーム余地」が業務文脈軸（買取再販事業者にしか効かない）
- **書籍評価**: 「人気」「評価点」が流通／「自分の読み筋に対する位置づけ」が文脈軸

## 介入点（Disable conditions）

1. **比較フレームの自作**: 既存の○○選を使わず、自業務軸を先に書き出してから候補を当てはめる
2. **業務文脈の言語化**: 「自分の業務はどの軸の上に乗っているか」を意思決定前に明示する
3. **流通フレームの構造批判**: 「なぜこの軸が流通しているのか／何が抜け落ちているか」を比較記事内で明示する

## 関連パターン

- **visibility-asymmetry**: 可視性の非対称。本パターンは「比較容易性」を可視性の特殊形として持つ
- **narrative-pressure**: 物語化圧力。本パターンと並行して発動しうる
- **causal-misattribution**: 因果誤帰属。本パターンが意思決定段階で発動した場合、後段の事後分析が因果誤帰属に陥る

## 哲学的親族

- **マルティン・ハイデガー『存在と時間』(1927) §15-16**: 道具の存在様式（Zuhandenheit / Vorhandenheit）。本パターンは比較フレーム自体が Vorhandenheit 化した現象として読める。流通フレームは「対象として並べる」操作であり、業務における Zuhanden な評価軸を切り捨てる
- **ハーバート・サイモン『満足化原理』**: 限定合理性下で人は「比較しやすい選択肢」を選ぶ。本パターンの認知的基盤
- **ピエール・ブルデュー『ディスタンクシオン』**: 流通する評価フレームが文化的支配を再生産する。本パターンの社会学的同型

## 反証条件

ある領域で「比較フレームが流通している／流通フレームの軸はすべて文脈非依存」が観察されたが、文脈依存軸を入れた意思決定が誤りを引き起こさなかった場合、本パターンの定義は再検討が必要。

## メタ観察

本パターンは daily-writing skill の発動から発見された atom 由来パターン候補。発見順序：

1. 動画ツール選定の私的違和感（「コスパと質では決まらなかった」）

4. パターン化
