# synthesis（統合・合成）

異なる要素が結合し、新しい統一体を形成する構造。

## 定義

異質な要素・思想・伝統が融合し、元の要素を超えた新しい統一体・体系が生まれる。弁証法的統合、シンクレティズム、クロスオーバーを含む。

## 構造

```
element_A + element_B
  → interaction/tension
    → mutual_transformation
      → synthesis (new unity)
        → emergent_properties
```

## 発動条件

1. 異質な要素の接触
2. 相互作用の場
3. 統合を担う主体/プロセス
4. 新しい統一性の articulation

## 失敗条件

- **拒絶**: 要素間の非和解
- **同化**: 一方による他方の吸収
- **混乱**: 統一なき混合
- **表層化**: 深い統合の欠如

## 適用事例

| Layer | Entities |
|-------|----------|
| art | fusion-genres, cross-cultural-art, postmodernism |
| religion | syncretism, interfaith-dialogue, new-religious-movements |
| philosophy | dialectical-synthesis, integral-philosophy |
| culture | cultural-hybridity, creolization |

## 関連パターン

- **emergence**: 統合からの創発
- **transformation**: 統合による変革
- **dialectic**: 正反合の構造
