# technique（技術）

道具・方法・素材の革新から、それまで不可能だった表現が出現する構造。

## 定義

新しい技術、道具、素材、手法の発明・導入によって、従来の表現限界が突破され、まったく新しい芸術形式・ジャンル・美学が可能になる。技術が想像力を拡張する。

## 構造

```
new_technique (tool/method/material)
  → capability_expansion
    → expressive_possibility
      → artistic_innovation
        → emergent_form (new_medium)
```

## 発動条件

1. 技術革新の発生
2. 芸術家による採用
3. 表現への応用実験
4. 新しい美学の言語化

## 失敗条件

- **技術偏重**: 内容なき技術誇示
- **道具奴隷**: 技術に使われる
- **陳腐化**: 技術の急速な旧式化
- **アクセス格差**: 技術の独占

## 適用事例

| Layer | Entities |
|-------|----------|
| art | photography, digital-art, 3d-printing |
| music | synthesizer, sampling, autotune |

| printing | movable-type, lithography |

## 関連パターン

- **emergence**: 技術から新形式が創発
- **transformation**: 技術による変容
- **disruption**: 旧技術の破壊
