# threshold（閾値）

量的変化が一定点を超えると質的変化が不可逆的に創発する構造。

## 定義

漸進的な変化が蓄積し、臨界点（閾値）に達した瞬間、システム全体が新しい状態へと非連続的に移行する。閾値を超えるまでは変化は潜在的であり、超えた後は元に戻れない。

## 構造

```
gradual_accumulation
  → latent_tension
    → critical_point
      → phase_transition
        → new_state (emergent qualitative shift)
```

## 発動条件

1. 量的変化の継続的蓄積
2. システムの臨界点の存在
3. トリガーとなる最後の一押し
4. 非可逆性のメカニズム

## 失敗条件

- **散逸**: 蓄積が閾値に達する前に消散
- **適応**: システムが閾値を動的に上昇
- **分散**: エネルギーが複数経路に分散
- **抑制**: 閾値到達を阻止する力の介入

## 適用事例

| Layer | Entities |
|-------|----------|
| religion | conversion, revelation, enlightenment, martyrdom-threshold |
| philosophy | paradigm-shift, epistemic-rupture |
| economics | hyperinflation, market-crash, tipping-point |
| history | revolution, collapse, regime-change |
| technology | critical-mass, viral-adoption |

## 関連パターン

- **cascade**: 閾値超過後の連鎖
- **transformation**: 閾値による質的変化
- **catastrophe**: 閾値超過による崩壊
