# tragedy（悲劇）

システム・個人・集団が、内在的な矛盾や構造的欠陥により不可避的な破局に至る構造。

## 定義

行為者の合理的選択や善意にもかかわらず、システムの構造的性質により破滅的な結果が生じる。個々の最適化が全体の最悪化を招く。

## 構造

```
rational_actors
  → individual_optimization
    → collective_action_problem
      → resource_depletion/conflict
        → tragic_outcome (foreseeable but unavoidable)
```

## 発動条件

1. 共有資源または相互依存関係
2. 個別最適と全体最適の乖離
3. 協調メカニズムの不在
4. 短期利益と長期損害の時間差

## 失敗条件

- **ガバナンス**: 効果的な調整メカニズムの導入
- **所有権確立**: 共有資源の私有化
- **インセンティブ整合**: 個別と全体の利益一致
- **技術革新**: 資源制約の解消

## 適用事例

| Layer | Entities |
|-------|----------|
| technology | social-media-addiction, surveillance-capitalism |
| economics | tragedy-of-the-commons, market-failure |
| politics | arms-race, climate-negotiation |
| religion | sectarian-conflict, holy-war |
| art | artist-exploitation, streaming-devaluation |

## 関連パターン

- **erosion**: 悲劇に至る漸進的劣化
- **cascade**: 悲劇の連鎖的拡大
- **collapse**: 悲劇の終着点
