# transition（移行）

ある状態から別の状態へと漸進的または段階的に移る構造。

## 定義

システム・文化・技術が、急激な断絶ではなく、中間状態を経由しながら新しい状態へと移行する。連続性を保ちながらの変化。

## 構造

```
state_A
  → transitional_phase
    → hybrid_state (A+B elements)
      → gradual_shift
        → state_B
```

## 発動条件

1. 変化への動因（内的/外的圧力）
2. 移行を可能にする媒介
3. 連続性の維持メカニズム
4. 適応の時間

## 失敗条件

- **急進化**: 移行が革命に転化
- **固着**: 中間状態での停滞
- **断絶**: 連続性の喪失
- **逆行**: 元の状態への回帰

## 適用事例

| Layer | Entities |
|-------|----------|
| technology | analog-to-digital, energy-transition |
| art | baroque-to-classical, modernism-to-postmodernism |
| history | feudalism-to-capitalism, cold-war-end |
| politics | democratization, liberalization |

## 関連パターン

- **transformation**: より急進的な変化
- **emergence**: 移行からの新秩序創発
- **erosion**: 旧状態の漸進的衰退
