# un-accusable-attack（告発できない攻撃）

被害者が**「何が起きたか」を言語化できないまま被害だけが残る**攻撃形態。攻撃の輪郭が言語の側から溶けているため、被害者が告発しようとしても告発の対象が定まらない。

## 定義

攻撃が以下の三条件を満たす構造:

1. **証拠不在の運用**: 加害者は「証拠の挙がらないように工夫する」
2. **言語による否認可能性**: 攻撃発話は曖昧化・形容詞化・断片化されており、文字通りには何も主張していない
3. **被害の身体化なし**: 被害は心理的・社会的に届くが、物理的痕跡を残さない

被害者は「やな心持ち」を抱えるが、それを第三者に説明できない。

## 構造

```
attack_emission（曖昧な発話）
  → reception_with_certainty（被害者は自分のことだと確信）
    → unspeakability（だが何を言われたか説明できない）
      → internal_circulation（被害は被害者の中だけで完結）
        → no_target_for_accusation（告発の対象が定まらない）
```

## 観察可能な現代具体例

- フェイクニュース・SLAPP訴訟
- 「そういう意図ではなかった」型のハラスメント否認
- SNSの意味深な投稿（誰を指すかは明示しない）
- 「業界内では有名な話」型の評判破壊
- 会議室での音量低下＋特定の単語だけが届く密談
