# unintended-consequences（意図せざる結果）

行為の意図と結果が乖離し、予期せぬ帰結が創発する構造。

## 定義

合理的に計画された行為が、複雑系の相互作用を通じて、当初の意図とは異なる（しばしば正反対の）結果を生み出す。個々の合理性が集合的非合理性を創発する。

## 構造

```
intended_action
  → system_interaction
    → secondary_effects
      → feedback_loops
        → emergent_outcome
          → unintended_result (opposite or perpendicular to intent)
```

## 発動条件

1. 複雑系における介入
2. 長期的時間軸
3. 多数のアクターの相互作用
4. 不完全情報下での意思決定

## 失敗条件

- **単純系**: 予測可能な線形関係
- **短期**: 二次効果が顕在化しない
- **孤立系**: 外部との相互作用なし
