# vertical-integration（垂直統合）

バリューチェーンの複数段階を単一組織が支配することで、効率性・支配力が創発する構造。

## 定義

原材料から最終消費者までのサプライチェーンにおいて、上流・下流の複数段階を内部化する。取引コスト削減、品質管理、市場支配力の獲得が動機となるが、柔軟性の喪失というトレードオフを伴う。

## 構造

```
value_chain_stages (supplier → producer → distributor → retailer)
  → internalization
    → coordination_efficiency
      → market_power
        → integrated_system (control over chain)
```

## 発動条件

1. 高い取引コスト（交渉・監視・執行）
2. 品質管理の必要性
3. 希少資源・ボトルネックの存在
4. 規模の経済の実現可能性

## 失敗条件

- **官僚化**: 内部調整コストの肥大
- **硬直化**: 市場変化への適応遅延
- **専門性喪失**: 各段階の最適化困難
- **反トラスト**: 独占規制の発動

## 関連パターン

- **emergence**: 統合から新しい組織能力が創発

- **capture**: バリューチェーンの価値捕獲
- **constraint**: 統合による市場制約
