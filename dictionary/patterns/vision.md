# vision（ビジョン）

望ましい未来像が現実を形成する力として機能する構造。

## 定義

明確な未来像（ビジョン）が、人々の行動を方向づけ、資源を動員し、自己成就的に現実化していく。理念駆動型の変革。

## 構造

```
articulated_vision
  → narrative_power
    → resource_mobilization
      → coordinated_action
        → vision_realization (or adaptation)
```

## 発動条件

1. 説得力のあるビジョンの articulation
2. ビジョンを担う主体/リーダーシップ
3. 共感と参加の獲得
4. 実行可能性の最低限の確保

## 失敗条件

- **空虚**: 実現手段との乖離
- **独善**: 共感の獲得失敗
- **固執**: 現実との不適合への無視
- **散逸**: ビジョンの希薄化

## 適用事例

| Layer | Entities |
|-------|----------|
| technology | silicon-valley-vision, space-exploration |
| politics | political-ideology, social-movements |
| business | corporate-vision, entrepreneurship |
| religion | prophetic-vision, eschatology |

## 関連パターン

- **transformation**: ビジョン駆動の変革
- **coordination**: ビジョンによる協調
- **emergence**: ビジョン追求からの創発
