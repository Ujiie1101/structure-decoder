# word-materialization（言葉の物質化）

言葉が感情に変換され、感情が行動に変換され、行動が物理的暴力に変換される構造。

## 定義

言葉それ自体は何も壊さない。しかし、言葉は人の中で**感情**に変換され、感情は**行動**に変換され、行動は**物理的結果**をもたらす。

```
言葉
  → 感情（怒り、恐怖、義憤）
    → 行動への衝動
      → 物理的暴力
```

## 構造

```
ジャーナリスト/発言者が言葉を発する
  → 言葉が拡散される（SNS、メディア）
    → 読者/聴衆が感情を抱く
      → 大多数は「ふーん」で終わる
        → しかし、ごく一部が行動に移す
          → 言葉が物理的暴力に変換される
```

## 確率的帰結

言葉の物質化は**確率的**に発生する。

| 読者数 | 行動する人数（仮定: 0.0001%） |
|--------|------------------------------|
| 1万人 | 0.01人（ほぼ0） |
| 100万人 | 1人 |
| 1億人 | 100人 |

発言者は「誰が」行動するかを予測できない。しかし、「確率的に誰かが」とは予測できる。

## 発動条件

1. **感情を喚起する言葉**: 怒り、恐怖、義憤を引き起こす内容
2. **大規模な拡散**: 読者数が多いほど確率は上がる
3. **明確なターゲット**: 怒りの対象が特定可能
4. **行動可能性**: ターゲットに接触できる手段がある

## 適用事例

| 事例 | 言葉 | 結果 |
|------|------|------|
| Sam Altman火炎瓶事件 | New Yorker「彼は信頼できるか」 | 自宅に火炎瓶 |
| 中絶医師殺害（1990s-2000s） | 「殺人者」というレトリック | 医師が射殺される |
| ルワンダ虐殺（1994） | ラジオでの「ゴキブリ」呼称 | 80万人虐殺 |
| 国会議事堂襲撃（2021） | 「選挙は盗まれた」 | 議事堂占拠 |

## 2×2マトリックス: 意図と結果

|  | **暴力なし** | **暴力あり** |
|---|---|---|
| **煽動の意図あり** | 失敗した煽動 | 扇動罪 |
| **煽動の意図なし** | 正常な言論 | **意図なき帰結** |

右下の象限が問題。意図がなくても、結果は発生する。

## ジャーナリズムのジレンマ

ジャーナリストは権力を監視する役割を持つ。批判的な言葉は、その役割の一部。

しかし、批判的な言葉は感情を喚起し、確率的に暴力を招く。

このジレンマに「正解」はない。

## 関連パターン

- **兵器転用のパラドックス**: 善意の言葉が悪意の行動を招く
- **傍観の経済学**: シェアする人は結果を見ない
- **悪意の敷居低下**: 行動の敷居が下がる
- **thoughtlessness**: 結果を想像せずに言葉を発する

## 防御策（限定的）

1. **言葉の選択**: 感情を煽りすぎない表現
2. **ターゲットの曖昧化**: 個人を特定しすぎない
3. **行動への呼びかけ回避**: 明示的な行動喚起をしない
4. **文脈の提供**: 複雑さを伝え、単純な怒りを避ける

ただし、これらは「確率を下げる」だけであり、ゼロにはできない。
