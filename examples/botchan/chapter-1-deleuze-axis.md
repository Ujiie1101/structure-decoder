## §2 修辞層（章レベル動詞分析）

| 動詞群 | 既存軸（C×D） | 新軸（ドゥルーズ） |
|---|---|---|
| 「親譲り」 | 継承・帰属 | **脱領土化**（自分から自分を切り離す） |
| 「損ばかり」 | 慢性化 | **逃走線の徴候**（コード化された場での失敗） |
| 「真っ直ぐ」 | 鏡像・承認 | **少数者の連帯**（モル的多数派 vs 分子的少数派） |
| 「便利になる」 | 場依存・反転 | **再領土化**（逃走線が新しい場で捕獲される） |
| 「小さく見えた」 | 鏡像の内在化 | **逃走線が引かれた証拠の知覚** |

「同じ語が違う動詞群で読める」── これがレンズを変える意味。

## §3 構造命題層（ドゥルーズ軸）

### 🔍 人文知レンズ
- **主**: ドゥルーズ＝ガタリ（『千のプラトー』脱領土化 / 逃走線 / 少数者の文学）
- **副1**: 西田哲学（場所の論理 — 主体ではなく場所が主体を含む）
- **副2**: ベンヤミン（『物語作者』経験の伝達と段階的反復）
- 採用理由: 章全体が**運動と場の変容**として読める。静的なハビトゥス論では捉えきれない**動詞**が章を支配している

### 🧪 構造フィルター（章レベル再判定）

| 結果 | フィルター | コメント |
|---|---|---|
| ✅ 通る | `legitimacy-source-shift` | 評価軸の転換が逃走の条件 |
| ✅ 通る | `causal-misattribution` | 自己の脱中心化＝脱領土化の冒頭装置 |
| △ 部分 | `mission-drift` | 漂流ではなく**意図的な逃走線** |
| △ 部分 | `provocation-reflex` | 即決構造は逃走運動と同型 |
| ❌ 通らない | `founders-curse` | 創業者ではなく**逃亡者** |
| ❌ 通らない | `groupthink` | 少数者の側に立つ |
| ❌ 通らない | `escalation` | 応酬ではなく離脱 |
| ❌ 通らない | `chronic-loss-narrative` | 損は逃走の徴候であって主題ではない |

**固有性の輪郭（通らないフィルターから）**:
- 創業・集団・応酬パターンが全滅 → これは**逃亡者の物語**
- 慢性損失も通らない → 損は静的状態ではなく**運動の副産物**

### 💡 章レベル抽出命題（独立性チェック後3個）

```
α  逃走線の事前宣言
   冒頭文「親譲りの無鉄砲」は、気質を自己から外部化する装置。
   「これは私のものではなく、親のもの」と名指すことで、
   主体が自分から脱領土化される。
   逃走線は物語の冒頭ですでに引かれている。
   起源: causal-misattribution × ドゥルーズ脱領土化
   反証: 終始「自分の選択で」と引き受けるなら崩壊
        → 終始「親譲り」と呼び続ける → 維持

β  少数者の連帯としての清
   清の「真っ直ぐ」は鏡像（C×D軸の読み）でもあるが、
   ドゥルーズ軸では「分子状の他者」=少数者の連帯。
   父・兄・親類は多数派（モル的）。坊っちゃんと清は少数派。
   逃走線は単独で引けない。少数者同士の認識が線を可能にする。
   起源: identification-mirror × ドゥルーズ少数者の文学
   反証: 清が多数派的価値（出世）を共有していたら命題崩壊
        → 清は出世論を共有しない → 維持

γ  逃走と再領土化の同時性
   章末「便利になる」は逃走の成功ではなく、
   逃走線が新しい場（四国の中学校）で再領土化される瞬間。
   逃げた先で再び絡め取られる。これは第二章以降の松山編で
   生徒・校長に絡め取られる伏線。
   ラスト「小さく見えた」は逃走線が引かれた証拠の知覚。
   起源: legitimacy-source-shift × ドゥルーズ再領土化
   反証: 四国で完全に自由なら命題崩壊
        → 松山でも生徒関係に絡め取られる → 維持
```

### 📐 命題間の独立性
- α vs β: 独立（個体の脱中心 vs 他者との連帯）
- α vs γ: 独立（出発の構造 vs 着地の構造）
- β vs γ: 独立（連帯の発生 vs 連帯の限界）

## §4 章接続層（書物全体）

- **第二章以降**: 再領土化（松山編）= 逃げた先で再びコード化される。生徒の悪戯・赤シャツの嘲笑＝**新しいモル的体系**
- **作品結末**: もう一度の脱出（松山→東京の街鉄技手）= 逃走線が**反復される**。第一章の構造が物語全体の最小単位
- **漱石他著作**: 『私の個人主義』(1914)『現代日本の開化』(1911)で展開される「外発的近代化からの内発的逃走」の主題と接続
- **西田哲学との関係**: 「親譲り」を場所として読むと、坊っちゃんは「無鉄砲という場所」と「東京の家族という場所」の重なりに住んでいた。場所の重なりが解けて新しい場所（四国）が開く構造
- **ベンヤミンとの関係**: 章ラスト「やっぱり立っていた。何だか大変小さく見えた」は**経験を物語ることの限界**の場面。坊っちゃんは清を物語ることに失敗し続けるが、その失敗自体が物語になる

## C×D軸（既存）vs ドゥルーズ軸（新規）の比較

| 観点 | C×D軸 (day-068) | ドゥルーズ軸 (day-069予定) |
|---|---|---|
| 主軸動詞 | 鏡像 / 承認 / 反転 | 脱領土化 / 連帯 / 再領土化 |
| 主体性質 | 受動的（場に応じて機能が変わる） | 能動的（線を引く） |
| 時間構造 | 状態の変化 | 運動の連続 |
| 中心命題 | 単独承認の鏡像 × 場依存 → 武器化 | 逃走線の事前宣言 × 連帯 → 再領土化 |
| 第一章の本質 | 呪いを武器に変える物語 | 逃走線を引く方法を描く物語 |
| 同時に成立するか | YES — 同じ章を二つの動詞群で読める |

両軸は**競合ではなく補完**。同じ第一章を別の動詞で読み直しただけ。読者は両方読んでも矛盾しない。
