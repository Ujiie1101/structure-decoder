# 『坊っちゃん』_読み直し（7）

WakaNa.U_世界を読むための思考OS
2026年5月13日

---

漆
「狭いけれ何でも分かりますぞなもし」

第七章で、第五章と第六章で運用された武器のすべてが、**たった一人の婆さんの口で裏返ります**。

マドンナとは誰か。なぜうらなりは蒼くなったか。山嵐はなぜ赤シャツに反対したか。── 舟の上でも会議室でも答えが与えられなかったすべての問いに、第七章では下宿の婆さんが「狭いけれ何でも分かりますぞなもし」と一夕の会話で答えるのです。

そして、その答えはすべて、坊っちゃんの認知地図を書き直します。

第五章で「磊落な人に注意」と暗示された山嵐は、実は赤シャツに直接抗議した正攻法の人でした。
第五章で「マドンナ……」と匂わされた相手は、実はうらなりの婚約者でした。赤シャツが「手蔓を求めて」横取りした女性でした。
第六章で「下宿のかみさんが出ろと言っている」と山嵐から告げられた追い出しは、実は計画でした。引越しの翌日、空いた部屋にはもう野だが入居していた。

すべてが、第五章で赤シャツが運用した武器（暗示・固有名・匂わせ・撒火者と消火者の同一化・伝聞経由の追い出し）によって、坊っちゃんの中に書き込まれた虚像だったのです。

匂わせは、秘密の維持を前提に成立する。**情報が透過する場所では、匂わせは初日から崩れている**。

第七章で漱石が書くのは、その崩壊の現場です。

---

1.

　驚いたのは、おれがいか銀の座敷を引き払うと、翌日から入れ違いに野だが平気な顔をして、おれの居た部屋を占領した事だ。さすがのおれもこれにはあきれた。世の中はいかさま師ばかりで、お互に乗せっこをしているのかも知れない。※1

※1.第六章で坊っちゃんは「下宿のかみさんが出てくれと言っている」と山嵐から聞きました。
第七章で坊っちゃんは下宿を引き払います。
翌日、その同じ部屋に野だが入る。

これは決定的な証拠です。

通常、下宿の客が出てから次の客が入るには、宣伝や仲介の時間がかかります。「翌日から入れ違いに」入居が起きるためには、**前の客が出る前から次の客が決まっていた**必要があります。

つまり、坊っちゃんを追い出す計画は、下宿のかみさんが「出てくれ」と言い始める前に、もう完成していたということです。

第五章で赤シャツは「磊落・淡泊・親切に下宿の世話」と暗示しました。
第六章で坊っちゃんは「裏切り者の山嵐から追い出された」と思って引っ越しました。
第七章で野だが部屋に入る。

これで、地図が完成します。**追い出しを仕掛けたのは山嵐ではない。赤シャツ・野だの側だ**。

漱石は、この事実を語りません。坊っちゃんも語らない。だが、野だが翌日に部屋を占領したという一行で、読者は前章までの全暗示が逆方向に書き直されていることに気づきます。

「世の中はいかさま師ばかりで、お互に乗せっこをしているのかも知れない」── 坊っちゃんはこの一行で、認知地図が崩れたことを告げるのです。

---

2.

　「そのマドンナさんがなもし、あなた。そらあの、あなたをここへ世話をしておくれた古賀先生なもし——あの方の所へお嫁に行く約束が出来ていたのじゃがなもし——」※1

※1.萩野家のお婆さんが、何気ない会話の中でマドンナの正体を全部明かします。

第一の事実── マドンナは遠山のお嬢さん。土地の別嬪。
第二の事実── マドンナはうらなり（古賀先生）の婚約者だった。
第三の事実── 去年うらなりの父親が死に、家計が傾いた。
第四の事実── 教頭（赤シャツ）が「手蔓を求めて」遠山家に出入りし、お嬢さんを手なづけた。
第五の事実── 山嵐が教頭の所へ抗議に行った。赤シャツは「破約になれば貰うかも知れんが、今のところは交際をしているばかり」と答えた。
第六の事実── 山嵐と赤シャツはそれ以来折合がわるい。

すべてが、一人の婆さんの一夕の会話で出てきます。

これは奇跡ではありません。「狭いけれ何でも分かる」── これが漱石が書いている装置です。

田舎には情報の透過性がある。秘密は守れない。誰が誰を訪ねたか、誰が誰と歩いたか、誰の家計が傾いたか── すべてが共有される。

そして、ここで第五章の装置が裏返ります。

第五章で赤シャツは「ターナー」「マドンナ」「磊落な人」と暗示で攻撃しました。これらの暗示は、**坊っちゃんが田舎の情報網にアクセスできない**ことを前提に成立していた。新参者で、土地の事情を知らない者にだけ、暗示は効くのです。

だが、坊っちゃんが下宿を移って、土地の情報網（萩野婆さん）にアクセスした瞬間、暗示はすべて解凍される。

漱石はここで、**情報非対称性に依存する攻撃の脆弱性**を書いています。匂わせは情報の非対称を前提とする。情報の非対称が崩れると、匂わせは効力を失う。

そして2026年。SNSと内部告発が一般化した現代では、組織内の匂わせ・暗示・派閥は、以前より早く崩れる。「狭いけれ何でも分かる」── 田舎の婆さんが運んでいた装置を、いまはタイムラインが運んでいるのではないでしょうか。

3.

　淡泊だと思った山嵐は生徒を煽動したと云うし。…厭味で練りかためたような赤シャツが存外親切で、おれに余所ながら注意をしてくれるかと思うと、マドンナを胡魔化したり…どう考えてもあてにならない。※1

※1.第七章の終盤で、坊っちゃんは自分の認知地図が書き直されていることを書きます。

| 人物 | 第五章までの坊っちゃんの認知 | 第七章で判明した実相 |
|---|---|---|
| 山嵐 | 親切で下宿の世話をしてくれた友 | 赤シャツの不正に抗議した正攻法の人 |
| 赤シャツ | 親切に注意してくれる教頭 | マドンナを横取りし、坊っちゃんを下宿から追い出させた策謀者 |
| 野だ | 単なる追従者 | 計画的に坊っちゃんの部屋に移った共謀者 |
| マドンナ | 赤シャツの馴染の芸者 | うらなりの元婚約者を奪われた被害者 |
| うらなり | 大人しい蒼い顔の同僚 | 婚約者を奪われ、家計も傾いた二重被害者 |

すべてが、第五章で赤シャツが運用した武器によって、坊っちゃんの中に書き込まれた虚像でした。

そして第六章で坊っちゃんは、その虚像に基づいて行動しました。山嵐に一銭五厘を返す。下宿を引き払う。会議で山嵐の正論を疑いつつ聞く。── すべて、操作された認知に基づく行動でした。

漱石はここで、**情報操作の射程**を書いています。

操作された情報は、被害者を「自分の判断で動いている」と思わせるのです。坊っちゃんは「自分で考えて山嵐と縁を切った」と思っていた。だが、その「自分の判断」は、舟の上で赤シャツが撒いた種の発芽でしかなかった。

そして、その種が異物だと気づくのは、種を撒いた人物の外で、別の情報源（萩野婆さん）に接した時だけです。

2026年の私たちは、これを別の名で知っています。SNSのアルゴリズムが選んだ情報で形成される「自分の意見」、上司の機嫌で書き直される「自分の評価」、同僚の噂で更新される「自分のキャリア観」── すべて、操作された情報に基づく「自分の判断」ではないでしょうか。

漱石は1906年に、これを下宿の婆さんの一夕の会話で書きました。

4.

　[清の手紙：]坊っちゃんは竹を割ったような気性だが、ただ肝癪が強過ぎてそれが心配になる。…ほかの人に無暗に渾名なんか、つけるのは人に恨まれるもとになるから、やたらに使っちゃいけない…田舎者は人がわるいそうだから、気をつけてひどい目に遭わないようにしろ。…お小遣がなくて困るかも知れないから、為替で十円あげる。※1

※1.清からの手紙が届きます。

長い。読みにくい。だが、内容は四つに整理できます。性格への忠告（あだ名を付けるな、肝癪を抑えろ）。田舎者への警戒（人がわるい、ひどい目に遭うな）。健康への配慮（寝冷えするな）。経済支援（為替で十円）。

清の手紙の中身は、第五章と第六章で坊っちゃんが浴びた装置の**ちょうど対極**にあります。

| 第五・六章の赤シャツ | 清の手紙 |
|---|---|
| 暗示で操作 | 直接の忠告 |
| 固有名で振り分け | 「坊っちゃんは……」と指す |
| 匂わせで除外 | 「人がわるい」と名指す |
| 「秘密にしてくれ」 | 「清だけに手紙で知らせろ」 |

清は、暗示も匂わせも使わない。すべて直接、固有名で名指す。

そして、為替十円。これは経済的な援助ですが、第六章の山嵐への一銭五厘の返済と対称的です。山嵐への一銭五厘は関係を切るための支払い。清からの十円は関係を続けるための贈与。

第二章で漱石が書いた「届かないはずの希望（笹飴）の、本来食べないはずの部分（笹）を、清は『薬』と呼ぶ」── この清の論理が、第七章ではもう一段進みます。世間の判定と逆向きに名付ける人が、世間の判定と逆向きに**情報を運ぶ**。

清の手紙は、坊っちゃんが赤シャツの操作から脱出する**第二の情報軸**として機能するのです。

5.

　[電車場面：]おれはこの時何となく気の毒でたまらなかったから、うらなり君のあとから、すぐ同じ車室へ乗り込んだ。上等の切符で下等へ乗るに不都合はなかろう。※1

※1.第七章の中盤、坊っちゃんは温泉に向かう電車を待っている間に、マドンナを初めて目撃します。

色白、ハイカラ頭、背が高い、美人── マドンナはこれまで坊っちゃんの中で、赤シャツの匂わせから組み立てられた抽象でした。「芸者の渾名か何か」と推測されていた。第七章でその抽象が、実在の女性として姿を現すのです。

そして直後に赤シャツが現れる。「乗り後れやしないかと思って心配して急いで来た」と言う。
そしてマドンナと赤シャツは同じ上等車両に乗る。
そしてうらなりは下等車両に一人で乗る。

これは一枚の構図です。
- 上等：赤シャツ＋マドンナ＋マドンナの母
- 下等：うらなり

電車の等級が、人間関係の現状を可視化している。

そして坊っちゃんは、上等の切符を持っているのに、下等にうらなりと一緒に乗ります。

「上等の切符で下等へ乗るに不都合はなかろう」── これは、坊っちゃんの選択です。経済的には損失だ。だが、坊っちゃんは敗者の側に座る。

第二章で坊っちゃんは「嘘をつくよりまし」で経済を超えた。
第六章で坊っちゃんは「ご免だ」で関係を超えた。
第七章で坊っちゃんは「気の毒でたまらない」で席を超える。

漱石は、坊っちゃんの倫理を経済の上に置き続けているのです。

6.

　その時おれは男の様子を見て、はてなと思った。…月は正面からおれの五分刈の頭から顋の辺りまで、会釈もなく照らす。男はあっと小声に云ったが、急に横を向いて、もう帰ろうと女を促がすが早いか、温泉の町の方へ引き返した。※1

※1.温泉から出た坊っちゃんが、月夜の野芹川の堤を散歩します。

二つの影が見える。男と女。坊っちゃんは追い付く。男の顔を覗き込む。

──赤シャツ。

これが第七章の決定的なショットです。

第五章で赤シャツが舟の上で「マドンナ……」と匂わせた時、それはただの匂わせでした。第六章で坊っちゃんが「マドンナに逢うのも精神的娯楽ですか」と逆に投げ返した時も、確証はなかった。だが第七章の月夜、坊っちゃんは赤シャツとマドンナの密会を**現認**するのです。

匂わせは、それ自体では否認可能な攻撃です。「そんなつもりじゃなかった」「ただの一般論だ」と逃げられる。
だが、匂わせている内容に**実体**があると分かった瞬間、匂わせは完全な命題に変わる。

坊っちゃんが手にしたのは、**匂わせの実弾化**です。

「赤シャツは図太くて胡魔化すつもりか、気が弱くて名乗り損なったのかしら」── 坊っちゃんはここで、赤シャツの心理を測ります。だが、答えは出さない。

漱石はここで、勝負の決着を書きません。月夜の堤で起きたのは、目撃と逃走だけです。だが、これが第七章の最後のショットになります。

---

第七章は閉じます。

漱石が第七章で書いたのは、**情報非対称性の崩壊**です。

第五章と第六章で運用された武器は、すべて被害者が情報源にアクセスできないことを前提にしていました。第七章で被害者は、別の情報源（萩野婆さん、清、月夜の現認）にアクセスする。すると、武器は次々と無効化されます。

| 第五章の武器 | 第七章の無効化 |
|---|---|
| マドンナ匂わせ | 婆さんの全証言で正体判明 |
| 「磊落な人」暗示 | 山嵐は赤シャツに抗議した正攻法の人と判明 |
| 「親切に下宿の世話」 | 翌日に野だが入居し、計画的追い出しと判明 |
| 「ホホホホ」笑い | 月夜の堤で逃走する赤シャツ |
| 「証拠のない事だから」 | 月夜の現認で証拠あり |

漱石は、武器の無効化を一章で書きました。

ただし、無効化は勝利ではありません。第七章の最後で、赤シャツは「温泉の町の方へ引き返した」── 戦線は移っただけです。

漱石はこの章を、勝利の章として書きません。**地図が書き直された章**として書く。誰が敵で誰が味方かが分かったが、戦闘はこれからだ。

第七章の最後の一行は印象的です。「ところが狭くて困ってるのは、おればかりではなかった」── 坊っちゃんは、自分を縛っている「狭さ」が、敵をも縛っていることに気づきます。

「狭い」は、被害者にとっては監視と窮屈の感覚です。だが、加害者にとっても「狭い」は監視である。秘密が守れない場所では、加害も守れない。

そして、漱石はこの反転を、たった一行で書く。**自分を縛っている条件は、敵をも縛っている**── これは、坊っちゃんが第七章で獲得した認識です。

2026年。SNSと監視カメラと内部告発の時代では、「狭い」は被害者だけでなく加害者をも縛ります。組織のパワハラ、芸能人の不倫、政治家の答弁── 加害者は以前のように匂わせと暗示で逃げられない。情報の透過性が、加害の輪郭を露出させる。

漱石は1906年に、この反転を月夜の川堤の一行で書きました。

物語は続いています。坊っちゃんは、まだ松山にいる。私たちも、まだここにいる。

──そして、いま誰かが、誰かのタイムラインで、「私は何も明言していない」と書いた直後に、別の誰かが「証拠を持っている」と返している。

あなたは、いま「あなたの意見」だと思っていることを、どこから得ましたか。誰かの撒いた種が、別の情報源と照合される前に、あなたの中で発芽していませんか。

直近の人物評価を一つ思い出してください。
それを最初にあなたに伝えたのは誰でしたか。
その人は、その評価を「自分が見た」と言いましたか、それとも「誰かから聞いた」と言いましたか。

そのすべてが、第七章の続きです。
