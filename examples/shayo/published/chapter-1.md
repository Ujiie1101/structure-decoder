# 『斜陽』_読み直し（1）

WakaNa.U_世界を読むための思考OS
2026年5月13日

---

壱
礼法どおりに飲んでいる方が、貴族ではない

第一章の朝の食堂で、母と娘がスウプを飲んでいます。

母はスプウンを口と直角に立て、尖端から流し込む。礼法どおりでは**ない**。
娘はスプウンを横向きにし、お皿に少しうつむき、礼法どおりに飲む。

**ほんものの貴族は、礼法を破っている母の方です。**
**礼法どおりに飲んでいる娘は、もう貴族ではない。**

太宰治がこの章の冒頭に置いたのは、宣言ではなく**スプウンの軌道の精密記述**でした。「我が家はもう貴族ではない」と誰も言わない。だが、母と娘のスプウンの動きが違うことで、もう娘の代では貴族の身体が継承されていないことが分かるのです。

弟・直治はこの構造を一行で言葉にします。「気取るという事は、上品という事と、ぜんぜん無関係なあさましい虚勢だ」と。

そして同じ第一章の終わりに、お母さまは引越し前夜に「死んでしまいたいのよ」と泣きます。お父さまの死、自分の流産、直治の不始末、そのどれにも泣かなかった母が、**お金が無くなった夜に初めて泣く**。

**金銭的崩壊が、感情の最後の堤防を切る瞬間**です。

第一章は、**没落は宣言の前に身体に届く**ことを書いた章です。

漱石が1906年に組織における言語的攻撃の装置を書いたのに対して、太宰は1947年に**家族における徴候の累積**を書きました。今回は装置の話ではありません。**喪失の現象学**── 何が壊れたかを誰も口にしないまま、身体・所作・徴候の側に確実に届いている、その届き方の話です。

1.

　お母さまは左手のお指を軽くテーブルの縁にかけて、上体をかがめる事も無く、お顔をしゃんと挙げて、お皿をろくに見もせずスプウンを横にしてさっと掬って、それから、燕のように、とでも形容したいくらいに軽く鮮やかにスプウンをお口と直角になるように持ち運んで、スプウンの尖端から、スウプをお唇のあいだに流し込むのである。※1

※1.ヨーロッパの正式礼法では、スウプはスプウンの側面から飲みます。お皿の上に少しうつむき、横向きのスプウンで運ぶ。これは「礼法」として成文化されています。

だが、お母さまは違う。スプウンを口と直角にして、尖端から流し込む。これは「礼法どおり」では**ない**。

ここに、太宰が書いている逆転があります。**ほんものの貴族は、礼法を守る人ではなく、礼法から外れる権利を持つ人**です。

これを「**紋章としての所作**」と呼んでおきます。

直治がこのあと言葉で説明します。「気取るという事は、上品という事と、ぜんぜん無関係なあさましい虚勢だ」と。だが、直治の説明より先に、太宰はお母さまの所作それ自体で同じ命題を提示しています。**所作は宣言を必要としない。所作それ自体が、所有者の地位を表示する**。

そして、紋章は継承できない。かず子は同じ所作ができません。「私は直治の言うような高等御乞食なのだから、お母さまのようにあんなに軽く無雑作にスプウンをあやつる事が出来ず、仕方なく、あきらめて、お皿の上にうつむき、所謂正式礼法どおりの陰気ないただき方をしているのである」と書きます。

ここに、第一章の核心の一つがあります。**所作は身体の一回限りの帰属であって、教えても、真似ても、伝わらない**のです。

そして2026年。テーブルマナー、敬語の運用、楽器の奏法、職人の手つき── これらすべてが「教育では伝わらない部分」を持っています。マニュアル化できる部分の下に、教えられない身体所作の層がある。**階級は、その教えられない層に最後まで残る**のではないでしょうか。

近い例で言えば、お箸の持ち方、お辞儀の角度、エレベーターでの立ち位置、メールの「お疲れさまです」の重みのつけ方── これらを自然にできる人とできない人の差は、本でも研修でも埋まりません。生家の食卓で、毎日少しずつ刷り込まれていく。だから、別の生家から来た者は、努力で埋めようとしても**埋めようとしている所作それ自体**が異物として浮き上がる。

これが「**継承の片道性**」です。所作は親から子へ降りるが、横からは入れない。

2.

　岩島なんてのは…まったく、新宿の遊廓の客引き番頭よりも、もっとげびてる感じじゃねえか。…テーブルスピーチの時に、あの野郎、ゴザイマスルという不可思議な言葉をつかったのには、げっとなった。気取るという事は、上品という事と、ぜんぜん無関係なあさましい虚勢だ。※1

※1.弟・直治の貴族論です。

直治は「岩島」という伯爵を批判します。タキシイドを着、「ゴザイマスル」という変な言葉を使う。**気取りは下品**である、と直治は言うのです。

ここで太宰が書いているのは、社会的地位の**逆向きの証明**ではないでしょうか。これを「**気取りの逆走**」と呼びます。

通常の社会では、地位を示す行為は「上品さ」を演じることです。しかし、ほんものの地位を持つ者にとって、演じることは必要ない。むしろ演じる必要が出てきた瞬間に、地位は失われている。**地位の証明は、地位の喪失を露わにする**のです。

直治のロジックを整理します。

| 行動 | 解釈 |
|---|---|
| 礼法どおりにスウプを飲む | 礼法を覚える必要があった者の行為 |
| スプウンを直角に流し込む | 礼法を覚える必要がなかった者の行為 |
| タキシイドを着る | タキシイドが正装だと知った者の行為 |
| ゴザイマスル | 上品な言葉を覚えた者の行為 |
| 普通の言葉 | 上品な言葉を意識する必要がない者の行為 |

直治の論理は単純です。**意識的に行為する瞬間、その行為は所属の証明ではなく非所属の証明になる**。

そしてこれは、現代の組織でも観察されるのではないでしょうか。

| 行為 | 解釈 |
|---|---|
| 新人がドレスコードを正確に守る | 新人である証明 |
| 創業者が短パンで会議に出る | 創業者である証明 |
| 中途採用者が前職の専門用語を多用する | 中の人ではない証明 |
| 古参が業界用語を崩して言う | 中の人である証明 |
| コンサルが「お疲れさまでございます」 | 顧客の外側にいる証明 |
| 役員が「おつかれ」 | 内側にいる証明 |

**意識的な礼節は、内側に居ないことの証明**です。そして恐ろしいことに、これは演じても乗り越えられない。意識して「短パンで会議に出よう」とすれば、その意識自体が既に外側にいる証拠になる。

太宰は1947年に、敗戦直後の貴族の食卓でこれを書きました。

太宰は1947年に、敗戦直後の貴族の食卓でこれを書きました。

3.

　お母さまは、平気でひょいと指先で骨のところをつまんで持ち上げ、お口で骨と肉をはなして澄ましていらっしゃる。そんな野蛮な仕草も、お母さまがなさると、可愛らしいばかりか、へんにエロチックにさえ見えるのだから、さすがにほんものは違ったものである。※1

※1.お母さまは骨つきのチキンを指で食べる。庭でおしっこをする。おむすびを手で握る。

これらはすべて、礼法から見れば**無作法**です。

だが、太宰はこれを「可愛らしいばかりか、へんにエロチックにさえ見える」と書く。「ルイ王朝の頃の貴婦人たちは、宮殿のお庭や、それから廊下の隅などで、平気でおしっこをしていた」と続ける。

ここに、貴族の所作の第二の構造があります。**礼法を超える特権としての無作法**。

ほんものの貴族は、礼法を守る必要がない。礼法を守らないことが、礼法を超えた地位にいることの証明になる。

| | 礼法を守る | 礼法を破る |
|---|---|---|
| 知らずに | 下層 | 下層（粗野） |
| 知って | 中層（覚えた者） | 上層（特権） |

「礼法を守る／破る」だけでは判定できません。「知って／知らずに」の軸が加わって初めて、貴族と粗野が区別される。

そして、この「知って破る」を演じることは**不可能**です。なぜなら、演じることは「知って守る」側の意識を含むからです。**演じた瞬間に、礼法を守る側の意識が侵入する**。

太宰は1947年に、骨つきチキンの所作でこの三層構造を書きました。

4.

　顔を見合せ、何か、すっかりわかり合ったものを感じて、うふふと私が笑うと、お母さまも、にっこりお笑いになった。何か、たまらない恥ずかしい思いに襲われた時に、あの奇妙な、あ、という幽かな叫び声が出るものなのだ。※1

※1.朝の食卓で、お母さまが「あ」と言います。
かず子も「あ」と言う。
二人は顔を見合わせて、「すっかりわかり合ったものを感じて」笑う。

この「あ」は、何でしょうか。

太宰は、これを**不随意な内面の漏出**として書いています。意識的な発話ではない。何かが内面に侵入してきた瞬間に、声帯が反応した結果です。

お母さまの「あ」は、直治を思った瞬間に出た。
かず子の「あ」は、六年前の離婚を思い出した瞬間に出た。

二つの「あ」は、別の記憶から生じた。だが、二人は「すっかりわかり合った」と感じる。**漏出の声は内容を共有しないが、漏出という形式は共有する**のです。

ここに、太宰の手つきの精密さがあるのではないでしょうか。母娘の親密性は、内容の共有ではない。**漏出の形式の同型**で成立している。

そして2026年。深夜のLINEで「あ」とだけ送ってくる友人。家族の前で突然「ふっ」と笑う祖母。── 内容を交換しない親密性は、いまも続いているのではないでしょうか。

5.

　そうして私の胸の中に住む蝮みたいにごろごろして醜い蛇が、この悲しみが深くて美しい美しい母蛇をいつか、食い殺してしまうのではなかろうかと、なぜだか、なぜだか、そんな気がした。※1

※1.かず子は近所の子供たちと10個の蛇の卵を焼こうとします。だが、生卵は燃えない。30分焼いても燃えないので、梅の木の下に埋めて石で墓標を作る。

そして、お母さまに見られる。お母さまは10年前にお父上の臨終で蛇を見ている。以来、お母さまは蛇を畏怖している。

その日の午後、母蛇が現れる。卵を探しに来たのです。

太宰はこの一連を、**不吉の手書き登記**として書いています。

ここで起きていることを整理します。

1. かず子が蛇の卵を焼こうとした（10個）
2. 卵は燃えなかった（焼き切れない）
3. お母さまに見られた
4. 母蛇が探しに来た
5. 母蛇の悲しげな姿が「お母さまの顔」と重ねられた
6. かず子の胸の中に「蝮みたいにごろごろして醜い蛇」が住み着く
7. その蛇が、母蛇（＝お母さま）を食い殺すのではないかと、かず子は予感する

これは、合理的な因果ではありません。**徴候の重層化**です。蛇の卵→母蛇→お母さま→かず子の中の蛇──この連鎖は論理的に閉じない。だが、現象学的には閉じている。**かず子は、自分が母を殺すかもしれない、と予感する**。

太宰は、後段の母の死を、この第一章で予告しているのです。だが、予告は陳述されない。蛇という象徴の重なりが、不吉として記録される。**論理的因果ではなく、徴候の連続が、未来を予告する**。

そして2026年。占星術・タロット・夢日記・「虫の知らせ」── 合理的に説明できない徴候の連鎖が、私たちの判断の基底に残っているのではないでしょうか。理性主義はこれを「迷信」と呼ぶ。だが、太宰はこれを「現象」として記録しています。

6.

　私たちが、東京の西片町のお家を捨て、伊豆のこの、ちょっと支那ふうの山荘に引越して来たのは、日本が無条件降伏をしたとしの、十二月のはじめであった。※1

※1.西片町は東京・本郷の一画です。明治・大正期に多くの華族・知識人が住んだ町。漱石の『こころ』の「先生」も西片町に住んでいます。

その西片町を捨てて、伊豆の山荘へ移る。

太宰はこの引越しを、**没落の地理学**として書きます。

| 西片町（東京） | 伊豆（山荘） |
|---|---|
| 本邸 | 別荘の小屋 |
| 西洋ふう | 支那ふう |
| 洋式食堂 | 簡素な部屋 |
| 複数の使用人 | お母さまとかず子の二人だけ |

すべての軸が、上から下へ移動します。

そして、この移動を決めたのは、和田の叔父さまです。お母さまは「お金の事は子供よりも、もっと何もわからないお方」だから、叔父が決めたとおりに頷くしかない。

ここで太宰が書いているのは、**金銭リテラシーと階級の関係**です。

ほんものの貴族は、金銭の運用を自分でしない。誰かが運用する。だが、戦争で経済構造が変わると、その「誰か」が機能しなくなる。お母さまは抵抗しない。「だって、お願いしていたのだもの」と微笑む。**抵抗できる文法を、お母さまは持っていない**のです。

これも、所作と同じ構造です。お母さまは「お金を管理する」所作を持っていない。所作を持たない者は、所作を持つ者の決定に従うしかない。

そして2026年。金融リテラシー、契約書を読む技術、行政手続き── これらの所作を持たない人は、持つ人の決定に従うことになる。**階級は、いまも所作の有無で確定する**のではないでしょうか。

7.

　「死んだほうがよいのです。お父さまの亡くなったこの家で、お母さまも、死んでしまいたいのよ」と、とぎれとぎれにおっしゃって、いよいよはげしくお泣きになった。※1

※1.引越し前夜、お母さまが初めて泣きます。

「お母さまは、今まで私に向って一度だってこんな弱音をおっしゃった事が無かった」── かず子は驚く。お父上の死、自分の結婚、流産、離婚、肺病、直治の不始末── そのどれにも、お母さまは弱音を吐かなかった。

だが、引越し前夜に、初めて崩れる。

太宰はここで、**金銭的崩壊が感情の最後の堤防を切る**ことを書いています。

なぜ、家族の死や子供の病気では崩れず、引越しで崩れるのでしょうか。

それは、家族の死や病気は**個人的な悲しみ**ですが、引越しは**所属の喪失**だからです。お母さまは「お父さまの亡くなったこの家で、お母さまも、死んでしまいたい」と言う。**場所＝アイデンティティ**なのです。場所を失うことが、所属を失うことになる。

そして、所属を失うことは、所作を失うことです。スプウンを「ヒラリ」と運ぶ所作は、西片町の食堂で成立していた。伊豆の支那ふう山荘では、同じ所作は意味を持たない。

太宰は、所作と場所の連動を、引越しという出来事で書きました。

8.

　ああ、お金が無くなるという事は、なんというおそろしい、みじめな、救いの無い地獄だろう、と生れてはじめて気がついた思いで、胸が一ぱいになり、あまり苦しくて泣きたくても泣けず、人生の厳粛とは、こんな時の感じを言うのであろうか、身動き一つ出来ない気持で、仰向に寝たまま、私は石のように凝っとしていた。※1

※1.母の崩壊を受けて、かず子は「人生の厳粛」を初めて知ります。

「お金が無くなるという事は、なんというおそろしい、みじめな、救いの無い地獄だろう」── かず子は29歳まで、お金の意味を知らなかった。離婚を経験し、肺病を病んでも、知らなかった。

なぜか。

それは、29年間、お金は誰か（父・叔父）が管理してくれていたからです。**お金は所作の対象ではなかった**。それは「あること」になっていた。

「あること」になっていた所作の前提が、無くなる。**前提を失った所作は、所作にならない**。スプウンを「ヒラリ」と運ぶ所作は、伊豆の山荘の食卓では、ただの食事になります。

太宰はここで、**所作はインフラを必要とする**ことを書いています。所作は身体に宿るが、身体だけでは成立しない。場所、人、経済、政治── これらのインフラが整って初めて、所作は紋章として機能する。

インフラが消えると、所作は宙づりになる。お母さまの「ヒラリ」も、これから先、誰のために、何のために、どこで運ばれるのか。

その問いの上に、かず子は仰向けで凝っとしているのです。

第一章は閉じます。

太宰が第一章で書いたのは、**没落は宣言の前に身体に届く**という構造でした。

| 没落の届き方 | 章内の場面 |
|---|---|
| 食事所作の継承不可能性 | お母さまの「ヒラリ」とかず子の「うつむき」 |
| 気取りと下品の同一性 | 直治の岩島批判 |
| 無作法の特権 | 指でチキン、庭でおしっこ |
| 「あ」の漏出 | 母娘の沈黙の共鳴 |
| 不吉の手書き登記 | 蛇の卵と母蛇 |
| 没落の地理学 | 西片町から伊豆へ |
| 金銭リテラシーの欠落 | 「だって、お願いしていたのだもの」 |
| 「人生の厳粛」の発見 | 引越し前夜のかず子の凝固 |

これらすべてが、**陳述ではなく徴候として届く**。

そして、徴候の連続が、後段（母の死・直治の死・かず子の革命）を予告します。第一章は、序章であると同時に、全章の予告書なのです。

漱石が1906年に組織における「装置の図録」を書いたのに対し、太宰は1947年に**家族における「徴候の累積」**を書いた。同じ社会の異なる断面です。

母蛇は、卵を探しに来た。
卵は焼かれていた。
母蛇は、何も見つけられず、かきつばたのほうに這入って行った。

太宰は、この場面を第一章の中央に置きました。母蛇が探していたものは、もう失われている。だが、母蛇はそれを知らない。お母さまが守ろうとしている階級は、もう失われている。だが、お母さまはそれを完全には知らない。かず子は、母蛇とお母さまを重ねて見る。そして、自分の胸の中の蛇が、母蛇を食い殺すかもしれないと予感する。

太宰はこれを、結核療養中の混乱の中で書きました。1947年に。
そして、私たちは80年後にこれを読んでいる。

物語は始まりました。かず子は、まだ伊豆にいる。お母さまは、まだ生きている。直治は、まだ南方の島から戻っていません。

──そして、いま誰かが、家のどこかで、誰かに継承できない所作で、スプウンを動かしている。

あなたが「自分の癖」だと思っている所作を一つ思い出してください。
お箸の持ち方、笑い方、電話の出方、何でもいい。
それは、誰の身体から渡されたものでしたか。

そして、その所作を支えていた場所（生家・職場・関係）が、もし消えたら── あなたの所作は、どこに残るのでしょうか。
それでも残るのか。それとも、所作は場所と一緒に消えるのか。

その問いの上に、かず子は仰向けで凝っとしています。
そして、あなたも今晩、同じ問いの上で眠るかもしれません。

そのすべてが、第一章の続きです。
